# Interact with the public REST API on a ServiceNow instance

The Mobile SDK provides functionality that enables iOS applications to call the public REST API on a ServiceNow instance.

Using the [NowAPIService](../../../../../app-store/dev_portal/API_reference/MobileSDKiOS/NowAPIService/concept/NowAPIServiceiOSAPI.md#) API you can interact with any of the base ServiceNow public [REST APIs](../../../../../build/applications/concept/api-rest.md), or create custom REST APIs within the instance and call those from your iOS application.
