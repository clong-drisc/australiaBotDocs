# Client mobile API reference

Use Cabrillo JS to build custom mobile experiences inside of ServiceNow AI Platform native mobile apps.

-   **[Cabrillo JS constants](../../../app-store/dev_portal/API_reference/namespace-topics/concept/cabrillo-constants.md#)**  
These are the constants used by Cabrillo JS.
-   **[Cabrillo JS data types](../../../app-store/dev_portal/API_reference/namespace-topics/concept/cabrillo-datatypes.md#)**  
These are common data types used by Cabrillo JS APIs.
-   **[cabrillo - Client](../../../app-store/dev_portal/API_reference/Cabrillo-client/concept/cabrillo-namespace.md#)**  
Cabrillo JS is a client side JavaScript API for accessing capabilities inside ServiceNow native mobile applications.
-   **[cabrillo.attachments - Client](../../../app-store/dev_portal/API_reference/CabrilloAttachments-client/concept/cabrillo-attachments-namespace.md#)**  
The name space for Cabrillo JS attachment functions. This enables adding and viewing attachments.
-   **[cabrillo.camera - Client](../../../app-store/dev_portal/API_reference/CabrilloCamera-client/concept/cabrillo-camera-namespace.md#)**  
Cabrillo JS camera functions to provide camera access for bar code scanning.
-   **[cabrillo.flowControl - Client](../../../app-store/dev_portal/API_reference/CabrilloFlowControl-client/concept/cabrillo-FC-namespace.md#)**  
Cabrillo JS functions for managing the flow of a web view.
-   **[cabrillo.message - Client](../../../app-store/dev_portal/API_reference/CabrilloMessage-client/concept/cabrillo-messgage-namespace.md#)**  
Cabrillo JS functions to display messages in the native UI.
-   **[cabrillo.modal - Client](../../../app-store/dev_portal/API_reference/CabrilloModal-client/concept/cabrillo-modal-namespace.md#)**  
Cabrillo JS functions for presenting web content inside of native modals.
-   **[cabrillo.navigation - Client](../../../app-store/dev_portal/API_reference/CabrilloNavigation-client/concept/cabrillo-navigation-namespace.md#)**  
Cabrillo JS functions for forward and backward navigation.
-   **[cabrillo.nowapp - Client](../../../app-store/dev_portal/API_reference/CabrilloNowApp-client/concept/cabrillo-nowapp-namespace.md#)**  
The name space for Cabrillo Now App functions. This enables you to interact with screens inside of ServiceNow AI Platform® native mobile apps.
-   **[cabrillo.viewLayout - Client](../../../app-store/dev_portal/API_reference/CabrilloViewLayout-client/concept/cabrillo-viewLayout-namespace.md#)**  
Cabrillo JS functions to provide access to native UI elements like buttons and spinners.
-   **[m\_form - Client](../../../app-store/dev_portal/API_reference/m_form/concept/m_formClientAPI.md#)**  
Provides methods to get and set input values on input form screens, as well as display messages.
-   **[MobileScriptIncludeCaller - Client](../../../app-store/dev_portal/API_reference/MobileScriptIncludeCaller/concept/MobileScriptInclCallerClientAPI.md#)**  
Provides methods to call script includes from Mobile UI Rule Actions.
