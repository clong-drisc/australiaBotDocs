# Enable WhatsApp channel notifications

Turn on notifications for the WhatsApp channel to send system-generated messages and automated communications.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **sys\_cs\_channel.list**.

2.  Select the WhatsApp channel record.

3.  Select **Enable notifications** check box.

4.  Select **Update**.
