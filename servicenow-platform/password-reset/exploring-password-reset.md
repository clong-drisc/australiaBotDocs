# Exploring Password Reset

Get an overview of Password Reset, Password Reset Windows Application, and Password Reset for Virtual Agent.

Understand how to set up, configure, and use Password Reset and Password Reset Windows Application for your organization.

[Password Reset](password-reset.md) The Password Reset application enforces strong and secure passwords by enabling end users to reset or change their passwords either by using the self-service process or by requesting the assistance of a service desk agent.

[Password Reset for Virtual Agent](password-reset-for-virtual-agent.md) Augment an end-user experience with Password Reset for Virtual Agent that handles common support issues, using pre-built conversations.

[Password Reset Windows Application](win-password-reset-overview.md) The Password Reset Windows Application enables a user, who forgets the password or is locked out of a Windows computer, to reset the password directly from the Windows login screen.

[Request to reset the default admin password](reset-default-admin-password.md) ServiceNow® has implemented an automated workflow to streamline the default instance admin password reset process. When a customer needs to reset their default instance admin password, they can initiate the request through a specific Service Catalog item. This request automatically creates a change record, which is then processed end-to-end using automation. This ensures that the password reset is handled efficiently and securely, without the need for manual intervention.
