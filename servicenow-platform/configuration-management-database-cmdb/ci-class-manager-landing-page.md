# CI Class Manager

Use the CI Class Manager to centrally view, create, or edit basic class definitions, class settings for Identification and Reconciliation Engine \(IRE\), CMDB Health, and manage some Service Mapping definitions.

**Note:** CI Class Manager doesn't support working with non-CMDB tables. For more details, see [IRE support for non-CMDB tables](../concept/ire-support-non-cmdb-tables.md).

To access the CI Class Manager, navigate to **All** &gt; **Configuration** &gt; **CI Class Manager**.
