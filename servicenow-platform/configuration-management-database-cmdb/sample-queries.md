# Example queries

Use the example queries as models for building your custom CMDB queries and Service Mapping queries.

-   **[CMDB — Servers connected to a database](../task/cmdb-query-builder-tutorial.md)**

    Use this example to build a CMDB query to find all servers with a connection to a database.

-   **[CMDB — Critical application services](../task/cmdb-query-builder-sample1-app-ser.md)**

    Use this example to build a CMDB query to find all critical application services, and their owner.

-   **[CMDB — Application services with incident or change](../task/cmdb-query-builder-sample2-app-ser.md)**

    Use this example to build a CMDB query to find all application services for which there is an incident or a change request. Results are returned for either the application service itself or any CI within the service.

-   **[CMDB — Hardware with Windows installed](../task/cmdb-query-builder-sample3-app-ser.md)**

    Use this example to build a CMDB query to find all hardware in my service offering that has Microsoft Windows installed.

-   **[Service Mapping — Linux servers in services](../task/service-mapping-query-tutorial.md)**

    Use this example to build a Service Mapping query to find all Linux servers in services.
