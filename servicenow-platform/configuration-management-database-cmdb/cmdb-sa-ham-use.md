# Using CMDB success advisor for HAM

As a CMDB administrator, you can use the Hardware Asset Management \(HAM\) dashboard in CMDB success advisor to keep hardware asset data accurate and aligned with data quality guidelines.

1.  [Set up CMDB success advisor for HAM](cmdb-sa-ham-config-settings.md).

    Set up the CMDB success advisor for HAM.

2.  [Monitor CMDB data quality for HAM](cmdb-sa-ham-dashboard.md).

    Identify and address data quality issues specific to HAM in the CMDB.

3.  [Improve CMDB data quality for HAM](cmdb-sa-ham-remediation.md).

    Use the Remediation actions panel available for a data card in the advisor dashboard for HAM to improve the overall quality of your CMDB.

4.  [Analyze data integrations for HAM](cmdb-sa-ham-data-integrations.md).

    Improve HAM data coverage by analyzing existing and suggested data integrations, including Discovery patterns and Service Graph Connectors.

5.  [Analyze CMDB settings for HAM](cmdb-sa-ham-analyze-settings.md#).

    Monitor and optimize CMDB settings that support CI-to-asset synchronization, life cycle alignment, and HAM data quality.
