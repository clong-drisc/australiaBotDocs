# Requires attention

The Requires Attention section helps you manage articles and feedback tasks that need your action.

## Features

The **Requires attention** section displays items that need review or updates. It includes:

-   **My Approvals**: Displays articles assigned to logged-in user for their approval.
-   **My Pending Group Requests**: Displays if a request is opened by a logged-in user and not closed or completed.
-   **My feedback task**: Tasks assigned to a logged-in user or if the logged-in user is in an ownership group or where the logged-in user is manager and the task isn’t completed. You can do the following feedback tasks:
    -   Modify the task details.
    -   Delete tasks that are no longer needed.
-   **My Article Checklist**: Displays the article checklist. Here the logged-in user should be a reviewer and state shouldn’t be already submitted.
-   **My Flagged**: Displays the Articles, if they’re flagged, and if the logged-in user is the author, the reviser, or part of the ownership group.
-   **Articles expiring in 90 days**: Views the list of articles that are assigned to the logged-in user expires soon in 90 days.
-   **My Translation Requests**: Shows that any translation requests assigned to the logged-in user and are open

You can use **Requires attention** section in  to keep your content accurate and up to date.
