# Create a document record

Create document records and include default versions for the documents.

## Before you begin

Role required: none

## Procedure

1.  Navigate to **All** &gt; **Documents** &gt; **Documents**.

2.  Select **New**.

3.  On the form, fill in the fields.

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the document.

</td></tr><tr><td>

Owner

</td><td>

Name of the document owner.

</td></tr><tr><td>

Reviewers

</td><td>

Name of the reviewers if any.

</td></tr><tr><td>

Template

</td><td>

Option to determine whether the document is a template.

</td></tr><tr><td>

Department

</td><td>

Name of the department that created the document.

</td></tr><tr><td>

Type

</td><td>

The type of document based on the content: guideline, policy, procedure, or contract.

</td></tr><tr><td>

Classification

</td><td>

Classification of the document: public, restricted, or confidential.

</td></tr><tr><td>

Audience

</td><td>

Audience of the document, either internal or external.**Note:** The **Audience** field is hidden in the default view.

</td></tr><tr><td>

State

</td><td>

The state the document is in: Draft, Submit, Review, or Complete.

</td></tr><tr><td>

Description

</td><td>

A brief description about the document.

</td></tr><tr><td>

Default Version

</td><td>

The document version that is used for document download.

 By default, the latest version is the default version.

</td></tr></tbody>
</table>4.  Select the access setting.

5.  Select **Submit**.
