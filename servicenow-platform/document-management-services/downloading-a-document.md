# Download a document

Download an attached document version based on the selected default version.

## Before you begin

Role required: none

## About this task

**Note:** To download a document, the version file type must be an attachment and not a URL.

## Procedure

1.  Navigate to **All** &gt; **Documents** &gt; **Documents**.

2.  Select the record.

3.  Select the **Documents** tab.

4.  Select the documents you want to download.

5.  In the **Actions on selected rows** drop-down menu, select **Download Document**.
