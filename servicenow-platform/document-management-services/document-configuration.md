# Configure document permissions

Set sharing permissions on external cloud files using document configuration.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Documents** &gt; **Administration** &gt; **Document Configurations**.

2.  Select **New**.

3.  Enter the **Name**, and the **Order**.

4.  Select the provider and the table.

5.  Set the conditions to be met.

6.  Select **Submit**.

    A new configuration is created under Document Configurations.

7.  To set permission rules, select the configuration by clicking on the name.

8.  Select **New**.

9.  Set the conditions to be met for document access.

10. From **User fields**, select and add fields.

11. Select internal permissions from the Internal Permissions drop down.

12. Select external permissions from the External Permissions drop down.

13. Select **Submit**.
