# Modify a shared link access

Modify the document shared link access to make the link inactive or change the expiration date.

## Before you begin

Ensure that the Permalinks table is added in the related list.

Role required: owner or writer

## Procedure

1.  Navigate to **All** &gt; **Documents** &gt; **Documents**.

2.  Scroll down to the **Documents Permalinks** tab.

3.  For the document:

    -   To make the link inactive, in the Active column, select **false**.
    -   To add or modify the expiry date, in the Expiration column, select the date.
    **Important:** The permalink generated can be accessed publicly irrespective of the document classification.
