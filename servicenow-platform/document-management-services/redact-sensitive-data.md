# Redact data from documents

Redact sensitive data from PDF documents by either searching keywords or selecting text.

## Before you begin

Documents related list should be configured for the workspace.

Role required: none

## Procedure

1.  From the workspace, navigate to the required record \(for example, Incident\).

2.  Select the **Documents** tab.

3.  Select the document and version.

4.  Select and open the attached PDF document.

5.  Select the redact icon ![icon redact](../image/redact-icon.png) to enable data redaction.

6.  Enter a keyword under **Search a keyword to redact** and select the search icon ![Search icon](../../../administer/virtual-agent/images/icon-search.png).

    **Note:**

    Searching and redacting data manually is applicable only for text and not for text within images or any other line art.

    If the redaction is unsuccessful, then check if there’s an issue with the PDF format, manually redact by highlighting the text.

7.  To select and draw over an area to redact data, select **Draw and redact**.

8.  To undo, select the undo ![icon undo](../image/redact-undo-icon.png).

9.  Select **Apply redaction**.

10. Select **Save changes**.

11. To discard, select the close icon ![icon close](../image/redact-close-icon.png) in the Redact document pane.
