## Administrator \[platform\_document\_management\_admin\]

Provides Create, Read, Update, and Delete access to the document. Admin users have access to List and List Entries modules.

### Contains Roles

List of roles contained within the role.

-   dm\_sharing\_levels\_user
-   dm\_user\_role\_read
-   dm\_user\_group\_read
-   dm\_user\_criteria\_read

### Groups

List of groups this role is assigned to by default.

None.

### Special considerations

**Note:** Avoid granting an admin role when more specialized roles are available.
