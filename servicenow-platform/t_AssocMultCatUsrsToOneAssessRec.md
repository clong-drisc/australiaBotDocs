# Associate multiple category users to one assessable record

Create many stakeholders for a single assessable record from the Assessable Record form.

## Before you begin

Role required: assessment\_admin or admin

## About this task

You can also edit or delete stakeholders from the form.

## Procedure

1.  Navigate to **All** &gt; **Assessments** &gt; **Assessable Records**.

2.  Open a record from the list.

3.  In the **Category users** related list, click **Edit**.

    These category users are the stakeholders for this assessable record.

4.  Add category users to create stakeholders or remove category users to delete stakeholders.

    The system shows only category users for categories associated to the assessable record. For example, if an assessable record is associated only to the Compliance Score and Support Rating categories, User Satisfaction category users will not be listed.


-   **[Delete a stakeholder for one assessable record](t_DelAStakeholderForOneAssessRec.md)**  
You can delete stakeholders for one assessable record.
