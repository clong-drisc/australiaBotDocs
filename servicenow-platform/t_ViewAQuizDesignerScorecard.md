# View a quiz designer scorecard

The quiz scorecard opens in the Category Results view.

## Before you begin

Role required: none

## About this task

For detailed information about scorecards, see [Quiz Scorecards](../concept/c_QuizScorecards.md).

## Procedure

1.  Navigate to **All** &gt; **Quizzes** &gt; **Quizzes**.

2.  Open the quiz whose results you want to view.

3.  Under **Related Links**, select **View Scorecard**.

    The scorecard for the selected quiz opens in the Category Results view.

4.  Select a category to view from the choice list above the chart.

    ![Quiz scorecard category results](../images/QuizScorecardCategoryResults.png)
