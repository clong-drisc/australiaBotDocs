# Save a draft of a catalog item

Save a draft of a catalog item while requesting it on the Employee Center, Service Portal, and Now Mobile.

## Before you begin

Role required: requester

## About this task

If enabled, save the draft if you don't want to discard the form and edit it later. For example, this procedure explains how you can save a draft in Employee Center.

For information about enabling **Save as Draft**, see [Catalog Item form](../reference/catalog-item-form.md).

## Procedure

1.  Navigate to **All** &gt; **Employee Center** &gt; **IT**.

2.  Select the item that you want to request and start filling the form.

3.  If you want more information to fill the form and want to save it, select **Save as Draft**.

    Your draft is saved. You can access the saved drafts from the My Request widget in the **Drafts** tab.

    If you open a new but the same item to request, and the draft of the item is already saved, you get a message stating that the draft of this item exists. If you want to save the draft of this new item as well, you must give a unique name to this item in the **Save as draft** field, and select **Save**.

    The **Drafts** tab shows all the draft items related to the same item name.

4.  Open a draft item that you want to edit, and if you still don't want to submit it, select **Update draft** or if you want to submit it, select **Submit**.
