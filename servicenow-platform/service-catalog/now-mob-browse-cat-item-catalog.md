# Browse catalog items using the catalog hierarchy

In addition to the category hierarchy, browse catalog items using the catalog hierarchy for proper context.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Mobile** &gt; **Now Mobile app** &gt; **Applet Launchers**.

2.  Select the **Services** applet launcher.

3.  Replace the **Browse Services** applet launcher section with the **Browse Catalog** applet launcher section.

    1.  On the Applet launcher form, navigate to the **Body** section.

    2.  Under the **Section** column, click **Browse Services**.

    3.  On the Item Section form, set **Destination screen** as `Browse Catalog`.

    4.  Click **Update**.

        After the **Browse Catalog** applet launcher section is enabled, the catalog appears as the first level of browsing and then the categories.
