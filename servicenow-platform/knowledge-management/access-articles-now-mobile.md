# View the most popular or recently viewed articles in the Now Mobile Information applet

The Information applet on the Now Mobile app enables you to access knowledge articles from a mobile device.

## Before you begin

You can specify user criteria for an article to control which users can read the article. If no user criteria is defined for an article, all users can read the article.

## Procedure

1.  Access your instance using the mobile application.

2.  On the Now Mobile app, tap **Information**.

3.  Access your recently viewed or the most popular knowledge articles
