# Customize fields in the approval message for request

Customize the body message for a request and request item sent in Microsoft Outlook.

## Before you begin

Role required: oam\_admin

## Procedure

1.  Navigate to **All** &gt; **Actionable Messages** &gt; **OAM Definition**.

2.  Choose one of the following:

    |Option|Action|
    |------|------|
    |Request|Click **OAM Request Approval**|
    |Request Item|Click **OAM RITM Approval**|

3.  In the OAM Container section, customize the following fields as required:

    1.  **Title**: Title of the Outlook approval notification.
    2.  **Requester info**: User who created the request.
    3.  **Request Details**: This field can be customized. You can also add additional fields as per the requirement.
4.  Click **Update**.

    The approval message is displayed in Microsoft Outlook as shown below. Click **View Details** button to view the request in ServiceNow instance.
