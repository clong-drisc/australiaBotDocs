# Address items that requires attention

Learn how to review and manage items that need your attention, such as expiring articles, flagged content, and feedback tasks.

## About this task

The **Requires Attention** section helps you quickly identify and act on items that need review. This section includes articles nearing expiration, flagged content, and feedback tasks assigned to you.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Knowledge** &gt; **Requires Attention**.

2.  Review the sections under **Requires Attention**, and select the feature that suits your interest from the following list:

    -   Select **My Approvals** to view articles that are assigned to you for your approval.
    -   Select **My Pending Group Requests** to view if a request is opened by a logged-in user is not completed or closed.
    -   Select **My Feedback Tasks** to view feedback tasks assigned to you.
    -   Select **My Article Checklist** to view the article checklist. Here the logged-in user should be a reviewer, and state shouldn’t be already submitted.
    -   Select **My Flagged** to view the articles that you flagged for review.
    -   Select **Articles expiring in 90 days** to view the articles that are assigned to the logged-in user that expire soon.
    -   Select **My Translation Requests** to view that any translation requests assigned to the logged-in user and are open
3.  Select **View all** to see the complete list of items.

4.  Open an item link, like from the list of **My Feedback Tasks**, to view details and select desired action on the items.


## Result

You can track and resolve to pending tasks efficiently from the **Requires Attention** section.
