# Associate a number to a number group

Associate a Notify number to a number group to use inbound and outbound workflows.

## Before you begin

Role required: notify\_admin

## Procedure

1.  Navigate to **All** &gt; **Notify** &gt; **Numbers**.

    ![Notify Numbers module, which you can use to associate a Notify number to a number group](../image/Numbers.png)

2.  Click the listed phone number and assign it to a **Number Group**, as appropriate.

    **Note:** Each phone number can only be associated with one number group.

    |Field|Description|
    |-----|-----------|
    |Number|Number for the entry.|
    |Phone number|E.164 phone number, if applicable.|
    |Notify group|\(Optional\) Group to which this number is assigned.|
    |Number owner|Service provider of the number.|
    |Short Code|Short code number, if applicable.|
    |Active|Check box that indicates whether the number is active or not.|

3.  Click **Update**.
