# Call a user from the WebRTC UI

Make outbound calls and receive a voice call in the browser through the WebRTC interface from On-Call Schedules landing page, Roster and Escalation Details dialog box and on-call calendar.

## Before you begin

Role required: notify\_view, or itil

## Procedure

1.  Navigate to **All** &gt; **On Call scheduling** &gt; **On Call calendars**.

2.  In the calendar that opens, click the name of the user you want to call.

3.  Choose **Call** to call the user.
