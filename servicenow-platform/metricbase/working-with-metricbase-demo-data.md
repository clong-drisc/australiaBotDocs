# Working with MetricBase demo data

Familiarize yourself with MetricBase by exploring the MetricBase demo.

## Overview of the MetricBase demo

With the MetricBase Demo plugin, you can experiment with MetricBase without configuring it or knowing much about it. The demo plugin is separate from the MetricBase plugin. Make sure to [request both](../task/request-metricbase.md).

Use the MetricBase demo to explore MetricBase before you set it up for production purposes.

-   Visualize sample time-series data.
-   Access sample metrics, triggers, and a scheduled job that collects and stores data.
-   Test the functionality of included triggers. The plugin includes scheduled time-series data updates, as if they had been received from an instance.
-   Experiment with the MetricBase JavaScript APIs using the Data Explorer.

In the demo, a fictional fleet of drones has been outfitted with sensors that broadcast their metrics, including their speed, altitude, and battery life over time.

-   **[View demo data in MetricBase](../reference/getting-familiar-with-metricbase.md)**  
After ServiceNow personnel install the MetricBase and MetricBase Demo plugins, you can use the demo to get familiar with MetricBase.
-   **[Get familiar with MetricBase APIs](../task/metricbase-data-explorer.md)**  
Experiment with MetricBase APIs using Data Explorer that is part of the MetricBase Demo application. Data Explorer uses the data installed with the MetricBase Demo application.
