# Connect administration

Administrators can configure various performance settings and features that impact both Connect Chat and Connect Support.

**Note:** There are also administrative options specifically for Connect Chat or Connect Support. For more information, see [Connect Chat administration](c_CollaborationAdministration.md#) and [Connect Support administration](c_ConnectSupportAdministration.md).
