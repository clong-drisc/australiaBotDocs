# Exploring Now Assist for ITOM Visibility setup

Now Assist for Setup guides administrators through installing ITOM Visibility applications from a centralized hub.

## Now Assist for ITOM Visibility overview

Now Assist for Setup provides a centralized, guided installation experience for ITOM Visibility administrators. After installing the Implementation Agent, administrators access the IT Operations Management Product Hub from Admin Home. The hub provides a single install option to deploy all required applications and plugins, according to your license and subscriptions.

The following applications are included in the ITOM Visibility installation:

-   Discovery Admin Workspace
-   CMDB Workspace
-   Kubernetes Visibility Agent \(KVA\)
-   Tag Governance
-   ITOM Content Service
-   Certificate Inventory and Management
-   Firewall Audits and Reporting

## Now Assist for ITOM Visibility users
