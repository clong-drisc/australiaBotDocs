# Cloud Action Library reference

Reference topics provide additional information about the Cloud Action Library application.

-   **[AWS Enable EC2 Instance Monitoring action](cal-aws-enable-ec2-instance-monitoring.md)**  
Action that turns on monitoring for the selected Amazon Web Services \(AWS\) Virtual Machine \(VM\) instance.
-   **[AWS Get IAM Login Profile action](cal-aws-get-iam-login-profile.md)**  
Action that retrieves the user name for the specified Amazon Web Services \(AWS\) Identity Access Management \(IAM\) user.
-   **[AWS List Images action](cal-aws-list-images.md)**  
Action that describes the specified images \(AMIs, AKIs, and ARIs\) available to you or all of the images available to you.
-   **[AWS Put S3 Bucket Encryption action](cal-aws-put-s3-bucket-encryption.md)**  
Action that configures the default encryption and Amazon Web Services \(AWS\) S3 Bucket Key for an existing bucket.
-   **[AWS Turn Off EC2 Instance action](cal-aws-turn-off-ec2-instance.md)**  
Action that stops the specified Amazon Web Services \(AWS\) Virtual Machine \(VM\).
-   **[Azure Turn Off Virtual Machine action](cal-azure-turn-off-virtual-machine.md)**  
Action that stops the specified Microsoft Azure Virtual Machine \(VM\).
-   **[CAL - AWS List S3 Buckets action](cal-aws-list-s3-buckets.md)**  
Action that returns a list of all the Amazon Web Services \(AWS\) buckets owned by the authenticated sender of the request.
-   **[CAL - AWS List VMs action](cal-aws-list-vms.md)**  
Action that returns a list of all Amazon Web Services \(AWS\) Virtual Machines \(VM\) that belong to the specified logical datacenter.
-   **[CAL - Azure List VMs action](cal-azure-list-vms.md)**  
Action that returns the list of Microsoft Azure Virtual Machines \(VM\) that belong to the specified logical datacenter.
-   **[CAL - Azure VM IP Metric action](cal-azure-vm-ip-metric.md)**  
Action that returns the IP details of the specified Microsoft Azure Virtual Machine \(VM\).
-   **[CAL - Azure VM Monitoring Metric Data action](cal-azure-vm-monitoring-metric-data.md)**  
Action that returns monitoring information for the specified Microsoft Azure Virtual Machine \(VM\).
-   **[CAL - Integer Stream action](cal-integer-stream.md)**  
Action that returns an array of integers in the specified range.
-   **[Disable AWS IAM User action](cal-disable-aws-iam-user.md)**  
Action that deletes the password for the specified IAM user, which terminates the user's ability to access Amazon Web Services \(AWS\) services through the AWS Management Console.
-   **[CAL - AWS Generate Credential Report subflow](cal-aws-generate-credential-report.md)**  
Subflow that lists all users in your account and the status of their various credentials, including passwords, access keys, and MFA devices.
-   **[CAL - AWS S3 Get Encryption subflow](cal-aws-s3-get-encryption.md)**  
Subflow that returns the default encryption configuration for an Amazon Web Services \(AWS\) S3 bucket.
-   **[CAL - Resolve Credential Alias subflow](cal-resolve-credential-alias.md)**  
Subflow that returns the credentials alias for the mentioned credential.
-   **[CAL - Resolve MID subflow](cal-resolve-mid.md)**  
Subflow that returns a suitable MID Server based on the specified MID Server selection criteria.
-   **[CAL - AWS S3 Get ACL subflow](cal-aws-s3-get-acl.md)**  
Subflow that retrieves the Access Control List \(ACL\) details for the specified Amazon Web Services \(AWS\) S3 bucket.
-   **[Cloud profiles with minimal permissions for Cloud Action Library actions and subflows](cloud-profiles-with-minimal-permissions-for-cal.md)**  
You need appropriate cloud permissions to execute the Cloud Action Library actions and subflows. Edit the cloud permissions profile JSON to suit the needs of your organization.
