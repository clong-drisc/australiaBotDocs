# Access Event Management configuration console

Access the IT Operations Management configuration console to set up Event Management and configure AIOps capabilities.

## Before you begin

Verify that the ITOM AIOps and Now Assist for IT Operations Management plugins are installed.

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace**.

    The Admin Home page opens.

    ![Admin home page showing the product families.](../image/aiops-admin-conf-home-page1.png)

2.  In the IT Operations Management tile, select **View product overview**.

3.  In the IT Operations Management page, select **Configure**.

    ![IT Operations Management page with installed plugins and Configure button to access the Configure IT Operations Management page.](../image/aiops-admin-conf-button.png)

    The Configure IT Operations Management page opens.

    ![The Configure IT Operations Management page where you can see the Discovery and the Event Management modules.](../image/aiops-admin-conf-itom-page.png)
