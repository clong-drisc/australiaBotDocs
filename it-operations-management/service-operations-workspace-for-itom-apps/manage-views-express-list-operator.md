# Identify an assigned Express List view

Identify Express List views assigned to you by your Event Management admin to make sure that you focus on specific services, priorities, or alerts.

## Before you begin

Role required: evt\_mgmt\_operator, evt\_mgmt\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace**.

2.  From the navigation bar, select the Express list icon ![](../../event-management/image/express-list1.png).

3.  In the Filters pane, turn on the **Use advanced filters** option to access advanced filters assigned to you.

4.  From the drop-down list in the Filters pane, select an Express List view assigned to you.

    The default view is marked as Default.

    ![Assigned views in the Filters drop-down in Express List with an arrow pointing to the default view.](../image/assigned-views-express-list.png)

    **Note:** You can’t override the filters in views assigned to you. However, you can modify, save, and share these views locally.

    The Filters pane lists the predefined filters. The Express List table displays the columns configured for the selected view.
