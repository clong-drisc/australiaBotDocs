# Tracked Configuration file form

The Tracked Configuration file form displays details about a specific configuration file.

|Field|Description|
|-----|-----------|
|File Name|Name and file path of the file.|
|File Size \(B\)|File size in bytes.|
|Last Modified|Date the file was last modified.|
|File Content|Contents of the file, if you selected the option to save the contents on the file tracking definition.|
