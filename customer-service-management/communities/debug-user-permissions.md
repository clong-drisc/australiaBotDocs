# Debug user permissions

Investigate and diagnose problems with user access to forums.

## Before you begin

Role required: sn\_communities.forum\_admin

## Procedure

1.  Navigate to **All** &gt; **Community** &gt; **Administration** &gt; **Forums**.

2.  Click the name of the forum to investigate.

3.  Click the **Debug Forum Permissions** related link.

4.  Enter the name of the user you want to check the permissions for.

    The forum user permissions information indicates which content and access types the user has for a particular forum.
