# Video Configurations form fields

Set fields on the form to configure video sources.

<table id="table_czj_2v3_pfc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Host

</td><td>

A video source website, for example, youtube.com.

</td></tr><tr><td>

Domain

</td><td>

The domain is set to **global** by default.

</td></tr><tr><td>

Application

</td><td>

The application for the video source. It is recommended that you enter `Community`.

</td></tr><tr><td>

Widget

</td><td>

A widget to render the videos from the source. Enter `Communities render video`.
