# Component properties for the Data visualization

Component properties enable you to configure the behavior of the Data visualization component instance on the third-party website.

<table id="table_csm_base_entities"><thead><tr><th>

Component property

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Data visualization type

</td><td>

Display type of data visualization chart for the component.

</td></tr><tr><td>

Show header

</td><td>

Option to display the visualization header, including title and icons.

</td></tr><tr><td>

Show header separator

</td><td>

Option to display a line separating the header from the rest of the component.

</td></tr><tr><td>

Chart title

</td><td>

Title for the visualization.

</td></tr><tr><td>

Description

</td><td>

Description for your component.

</td></tr><tr><td>

Wrap title

</td><td>

Option to wrap long titles onto a second line. If cleared, an ellipsis is displayed to truncate the long title.

</td></tr><tr><td>

Show border

</td><td>

Option to display a line around the component.

</td></tr><tr><td>

Sort by

</td><td>

Sorts groups or categories in the specified order. Currently, you can sort by value or name.

</td></tr><tr><td>

Sort by order

</td><td>

Sorts groups or categories in ascending or descending order.

</td></tr><tr><td>

Show total value

</td><td>

Option to show the total value.This field doesn’t apply to pie chart.

</td></tr><tr><td>

Show metric label

</td><td>

Option to show the metric label. This field doesn’t apply to pie chart.

</td></tr><tr><td>

Drill down label

</td><td>

Text displayed in the tooltip to help users understand what will happen upon selecting.

</td></tr><tr><td>

Show selected data point

</td><td>

Option to show the data point that has been selected. It’s commonly used when the chart functions as a filter.

</td></tr><tr><td>

Refresh requested

</td><td>

A value that triggers a data refresh. The refresh occurs only if the new value of this property is different from the previous one.

</td></tr><tr><td>

Show legend

</td><td>

Option to display the legend.

</td></tr><tr><td>

Show legend value

</td><td>

Option to display the value corresponding to the area of the chart you hover on. The value is displayed as a tooltip and in the chart legend.

</td></tr><tr><td>

Legend position

</td><td>

Position of the legend relative to the chart. Currently, you can set the location to above, below, right, or left.

</td></tr><tr><td>

Legend orientation

</td><td>

The orientation of the legend. Currently, you set it either horizontal or vertical.

</td></tr><tr><td>

Horizontal alignment

</td><td>

The horizontal alignment of elements in the legend. Currently, you can set it to start, center, or end.

</td></tr><tr><td>

Max legend item width, px

</td><td>

Defines number of pixels after which the item of the legend will be truncated.

</td></tr><tr><td>

Expand legend vertically

</td><td>

Option that enables the legend to expand as needed to fit the content without scrolling, instead of being limited to two lines.

</td></tr><tr><td>

Show data labels

</td><td>

Option to display the score of each data point.

</td></tr><tr><td>

Set color type

</td><td>

Determines the color for the chart. You can set color type to default, color palette, or fixed element color.

</td></tr></tbody>
