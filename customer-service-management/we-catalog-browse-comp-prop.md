# Component properties for the Catalog browse

Component property enables you to configure the behavior of the Catalog browse component on the third-party website.

<table id="table_csm_base_entities"><thead><tr><th>

Component property

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Selection type

</td><td>

Display catalog item by either selecting catalogs or categories. Based on your selection,​ you can enter sys\_ids of either catalogs or categories. ​If you want to display specific categories, select type= categories.

 When catalog is selected all its categories is displayed.

 The default selection is **Catalog**.

</td></tr><tr><td>

Select catalogs

</td><td>

Enter the sys\_id of the catalogs whose catalog items you want to display. The default sys\_ids for Select catalogs are: e0d08b13c3330100c8b837659bba8fb4,742ce428d7211100f2d224837e61036d.

These sys\_ids are part of service catalog and technical catalog.

</td></tr><tr><td>

Select categories

</td><td>

Enter the sys\_id of the categories whose categories items you want to display. This field appears when you select **Category** in the **Selection type** field.

The default sys\_ids are: 22555319db0150104327198d1396195c,​109f0438c6112276003ae8ac13e7009d,2809952237b1300054b6a3549dbe5dd4.

These sys\_ids are part of application and account access services software.

</td></tr><tr><td>

Hide price

</td><td>

Option to show or hide the item prices if available.

</td></tr><tr><td>

Hide navigation panel

</td><td>

Option to show or hide the navigation panel from users.

</td></tr><tr><td>

Hide popular requests

</td><td>

Option to show or hide the **Popular requests** quick action.

</td></tr><tr><td>

Hide recent requests

</td><td>

Option to show or hide the **Recently requested by you** quick action.
