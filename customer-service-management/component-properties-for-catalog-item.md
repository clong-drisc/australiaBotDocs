# Component properties for the Catalog item

Component property enables you to configure the behavior of the Catalog item component instance on the third-party website.

<table id="table_csm_base_entities"><thead><tr><th>

Component property

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Catalog item

</td><td>

The system identifier \(sys\_id\) of the catalog item from the Record Producer \[sc\_cat\_item\_producer\] table. The default label is `Create Contact` and sys\_id is **56c62944c30102003d3b7bfaa2d3ae36**.

</td></tr><tr><td>

Hide additional details

</td><td>

Hides the delivery address and special instructions fields.

</td></tr><tr><td>

Prefill fields

</td><td>

Query that automatically populates fields on the form.

</td></tr><tr><td>

Hide record created confirmation

</td><td>

Option to show or hide a confirmation message when a record is submitted.By default, it shows record confirmation message.

</td></tr><tr><td>

Custom illustration size

</td><td>

Size of the illustration asset to use. If you select auto, the component chooses the appropriate illustration based on the current size of the now-illustration component.

</td></tr><tr><td>

Confirmation text

</td><td>

Text appears when the case is submitted successfully on a third-party website.

</td></tr><tr><td>

Confirmation sub-text

</td><td>

Subtext appears when the case is submitted successfully on a third-party website.

</td></tr><tr><td>

Hide reference number

</td><td>

Option to show or hide the reference number of the record on the confirmation message.By default, it shows reference number.

</td></tr><tr><td>

Reference number label

</td><td>

Display label of the reference number on the confirmation message.

</td></tr><tr><td>

Hide primary button

</td><td>

Option to show or hide a primary button on the confirmation message.By default, it shows primary button.

</td></tr><tr><td>

Primary button label

</td><td>

Display label of a primary button on the confirmation message. The default text is `View details`.

</td></tr><tr><td>

Hide secondary button

</td><td>

Option to show or hide a secondary button on the confirmation message.By default, it shows secondary button.

</td></tr><tr><td>

Secondary button label

</td><td>

Display label of a secondary button on the confirmation message.

</td></tr><tr><td>

Hide errors

</td><td>

Option to show or hide error messages for the end user on the third-party website. By default, it shows errors.

</td></tr><tr><td>

Hide notification

</td><td>

Option to show or hide component notifications. By default, it shows notification.

</td></tr></tbody>
