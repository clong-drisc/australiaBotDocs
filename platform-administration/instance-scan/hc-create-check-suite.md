# Create a check suite

Create a check suite to bundle a group of checks into a suite and execute it.

## Before you begin

Role required: admin.

## Procedure

1.  Navigate to **All** &gt; **Instance Scan** &gt; **Suites**.

    A list of suites shows up.

2.  Select **New** to create a new suite.

3.  Fill in the details on the new suite record.

    |Fields|Description|
    |------|-----------|
    |Name|Name of the suite|
    |Application|Scope of the suite|
    |Active|Option to activate this suite|
    |Description|Information about the suite|

4.  Select **Submit** to create a new check suite.

    **Note:** The suites that you create are completely modifiable. You can modify the checks in the suite and can also add child and parent suites.
