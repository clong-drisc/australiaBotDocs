# View Now Support users

As a Now Support admin, you can view active and inactive users and their roles.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Now Support**

2.  Select the drop-down profile menu.

3.  Select **Manage users and accounts**.

4.  Select **View all Users** or search for a specific user.
