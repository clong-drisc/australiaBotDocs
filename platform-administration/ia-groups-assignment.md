# Groups assignment

Organize users and assign roles using groups.

|Field|Description|
|-----|-----------|
|Name|Name of the group|
|Description|Detail description of the group|
|Active|State of the group if its active or not|
|Manager|Name of the assigned manager of the group|
|Parent|Name of the parent group|
|Updated|Date when the group information was last updated|

Select **Add a group** if you want to add a group to configure. You can also export the list of groups into an excel sheet or an available file type by selecting **Export**.
