# Time configuration

A standard feature of the ServiceNow AI Platform, ServiceNow's time configuration system handles scheduling across your entire enterprise, enabling you to manage projects efficiently, track and schedule events, and more.

## Get started

Time configuration includes the scheduling of events and tracking of time. There are several functions available which enable tracking of time and using that information across applications.
