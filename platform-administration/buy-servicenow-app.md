# Buy a ServiceNow application

Log in to the ServiceNow Store and purchase an application that lists ServiceNow as the provider.

## Before you begin

You must be logged in to the ServiceNow Store with your Now Support account credentials.

Role required: none

## Procedure

1.  Log in to the ServiceNow Store using your Now Support credentials.

2.  Find and select the application to view its listing details.

3.  Select **Request license**

    A sales agent or your account executive contacts you by email to process the purchase. If the application uses the standard terms and conditions, it's available to install from the Application Manager within 24 hours of confirmation.

4.  If the application displays the "App Terms Not Accepted" state indicator in the Application Manager, accept the custom terms and conditions from the ServiceNow Store.

    For more information about application state indicators, see [Application state indicators in Application Manager](../../application-manager/reference/app-mgr-state-indicators.md).

    1.  From the application listing details in the ServiceNow Store, select **Accept Terms**.

    2.  Download and review the custom terms and conditions.

    3.  Select the option to agree to the terms and conditions.

    4.  Select **Accept**.


## Result

The application is available for installation from the Application Manager within 24 hours of license confirmation. For more information about using the Application Manager, see [Application Manager](../../application-manager/concept/application-manager.md).

Confirmation emails for the application and any dependencies that were procured are sent to the email address associated with your ServiceNow Store account.

If the application isn't available to install from the Application Manager within the expected amount of time, try the following actions.

-   Verify that all necessary dependencies have been procured. For more information, see [Evaluating version requirements and dependencies](../concept/versions-dependencies.md).
-   Review application state indicators from application's details in the Application Manager. For more information about what each indicator means, see [Application state indicators in Application Manager](../../application-manager/reference/app-mgr-state-indicators.md).
-   If the previous options don't work, contact Now Support for assistance.
