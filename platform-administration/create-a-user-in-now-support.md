# Create a user in Now Support

You can create users in Now Support and assign them access roles.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Now Support**

2.  Select the drop-down profile menu.

3.  Select **Manage users and accounts**.

4.  Select **View all Users**.

5.  Select **Add new user**.

6.  Fill in the required fields.

7.  Select the appropriate Now Support access roles.

8.  Select **Submit**.
