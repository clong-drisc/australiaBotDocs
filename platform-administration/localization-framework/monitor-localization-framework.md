# Monitor the Localization Framework

Track localization performance and request translations with Localization Framework Dashboards.

-   **[Localization Insights dashboard](../../../use/dashboards/application-content-packs/localization-insights-dashboard.md)**  
The Localization Insights dashboard provides reports on translations, grouped into tabs by artifact type. The reports are generated by scheduled jobs based on Localization Insights records.
-   **[Request translations from Insights Dashboard](../task/request-translations-insights-dashboard.md)**  
Request translations for Service Catalog items or Virtual Agent topics from the insights dashboard to localize them into the selected language based on their translation status.
