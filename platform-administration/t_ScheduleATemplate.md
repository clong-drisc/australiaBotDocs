# Schedule record creation from a template

You can create and schedule a scheduled job to create records based on a template. For example, you can regularly create a populated task record to perform a weekly backup.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Templates**.

2.  Select a template record.

3.  Click **Schedule**.

    The [Scheduled Entity Generation](../../reference-pages/task/t_ScheduleTheGenerationOfAnEntity.md) form appears.

4.  In the **Run** choice list, select how frequently to create a record.

5.  Complete the schedule information.

6.  Click **Submit**.
