# Manage user access in Now Support

You can manage users in Now Support and change their access roles.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Now Support**

2.  Select the drop-down profile menu.

3.  Select **Manage users and accounts**.

4.  Select **View all Users** or search for a specific user.

    You can also select **Locked users** or **Inactive users** to find the user you are looking for.

5.  After you select the user you want to change access for, select **Edit Role\(s\)** from the **Actions** pull-down field.

6.  Add or remove Now Support access roles.

7.  Select **Confirm**.
