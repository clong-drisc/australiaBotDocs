# Legacy: Open Changes Reports dashboard

Use this dashboard to get an overview of the current status of open reports.

**Important:**

Starting in Xanadu release, the Change dashboard is deprecated. Users can use [Change dashboard](performance-analytics/change.md) to view, and track the open changes.

![Open Changes List filtered on change category, priority, state, and type](../image/change-open-changes-reports-list.png "Open Changes List tab")

![Open Changes Pivot table, unfiltered](../image/change-open-changes-reports-pivot.png "Open Changes Pivot tab")

![Open Changes Heatmap, unfiltered](../image/change-open-changes-reports-heatmap.png "Open Changes Heatmap tab")

## Data visualizations

|Title|Type|Description|
|-----|----|-----------|
|PA.Source: Open changes - List|List ![List icon](../../reporting/image/icon-list-report.png)|A list report of the unclosed changes opened on or before the current date. Columns include Number, Short description, Category, Priority, State, Assignment group and Assigned to.|
|PA.Source: Open changes - MPivot - Priority/State/Category|Pivot ![Pivot icon](../../performance-analytics/image/pivot-scorecard-icon.png)|Report that shows unclosed changes opened on or before the current date. The visualization shows changes by Priority, State, and Category.|
|PA.Source: Open changes - Heatmap - Priority/Category|Heatmap ![Heatmap icon](../../performance-analytics/image/heatmap.png)|Heatmap report that shows unclosed changes opened on or before the current date. Larger values are highlighted in darker colors.|
