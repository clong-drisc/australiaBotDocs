# License management with the legacy Software Asset Management plugin

The legacy Software Asset Management \(com.snc.software\_asset\_management\) plugin allows asset managers to track and organize the number of licenses available to the organization.

Software licenses are based on [models](../../product-catalog/concept/c_Models.md) that you create to organize software licenses in meaningful ways. Different [license calculation types](../reference/r_LicenseCalculationTypes.md) determine how software is counted.

Software licenses can also be associated with a contract which enables a company to cover software licenses for multiple assets or users. You can manage and track software contracts using the [Contract Management](../../contract-management/concept/c_ContractManagement.md) application.
