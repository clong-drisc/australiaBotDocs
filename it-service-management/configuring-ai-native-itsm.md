# Configuring Simplified IT Service Management

Configure your ITSM workflows using an AI-first, centralized, and intuitive admin experience.

## Configuration overview

1.  Ensure that the Now Assist for Setup application is installed on ServiceNow instance. See [Set up Now Assist with Now Assist for Setup](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/ia-setup-now-assist.md).
2.  Request for the entitlement of the IT Service Management application from ServiceNow Store and install it. See [Install IT Service Management](../task/install-native-ai-itsm.md).
3.  Configure IT Service Management on your ServiceNow instance using a guided configuration experience or conversational AI agents \(wherever available\).

    For configuration information, see [Configure integrations and ITSM experiences in Simplified IT Service Management](../task/configure-ai-native-itsm.md).

    For information about available AI agents, see [AI agents and agentic workflows in Simplified IT Service Management](../reference/agents-ai-native-it-service-desk.md).
