# View Desktop Assistant notifications

View notifications to stay up to date with important information and events in real time.

## Before you begin

Confirm that the DEX Desktop Assistant \[sn\_dex\_desktop\] application has been installed. For more information, see [Install Application and Device Health](install-app-device-health.md) and [Download and install Desktop Assistant](download-desktop-experience.md).

Role required: sn\_dex\_desktop.user or sn\_dex\_desktop.admin

## Procedure

1.  Log in to Desktop Assistant.

    For more information, see [Open and log in to Desktop Assistant](open-desktop-exp.md).

2.  On the Desktop Assistant home page, select the notifications bell icon ![](../image/icon-notification-bell.png).

    The notifications are displayed in a drop-down list.

    **Note:** With the DEX Washington DC release, if a user logs in to Desktop Assistant on a new device, any existing unexpired notifications don't appear in the notifications list.
