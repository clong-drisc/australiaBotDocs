# Enable or disable notifications

Enable notifications to stay informed and engaged receiving timely updates and notifications about important events and activities within the system.

## Before you begin

Role required: sn\_dex\_desktop.admin

## Procedure

1.  Navigate to **Desktop Assistant** &gt; **Configuration** &gt; **Application**.

2.  On the Desktop Assistant page, enable or deactivate notifications.

    -   Enable: Select the **Notifications Enabled** check box.
    -   Deactivate: Clear the **Notifications Enabled** check box.
3.  Select **Update**.


## Result

A bell icon ![](../image/icon-notification-bell.png) appears on the Desktop Assistant header if you enable notifications. If you disable notifications, the bell icon disappears from the header.

When enabled, push notifications are sent to employees using Desktop Assistant. For more information, see [Desktop Assistant notifications](../concept/da-push-notifications.md) and [View Desktop Assistant notifications](view-notifications.md).
