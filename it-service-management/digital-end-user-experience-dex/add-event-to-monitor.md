# Add an event to monitor

Add a custom event to extend monitoring beyond the base events using the Event Log Monitoring Config table.

## Before you begin

Role required: sn\_dex.admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Service Operations Workspace**.

2.  In the primary navigation pane, select the DEX Administration icon \(![](../image/icon-administration.png)\).

3.  On the **Event log monitoring** card, select **Configure**.

4.  In the Event Log Monitoring Configs table, select **New**.

5.  Complete the New record form to add an event to monitor.

    For details about the form fields, see [New DEX event form](../reference/new-dex-event-form.md).

6.  Select **Submit**.
