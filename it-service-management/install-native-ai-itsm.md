# Install IT Service Management

Set up and get started with IT Service Management \(ITSM\) on your instance.

## Before you begin

Role required: admin

-   Request for the entitlement of the Now Assist for Setup application \(sn\_ia\) from [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home) and install it. See [Set up Now Assist with Now Assist for Setup](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/ia-setup-now-assist.md) and [Understand the Configuration page flow in Now Assist for Setup](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/ia-configure-il.md).
-   Request for entitlement of the IT Service Management application \(sn\_ai\_itsm\_cont\) from [ServiceNow Store](https://store.servicenow.com/sn_appstore_store.do#!/store/home).

## About this task

The following items are installed with IT Service Management.

-   AI agents and agentic workflows simplifying ITSM workflows, supporting a simplified Employee Center portal, and a simplified administrator, employee, and fulfiller experience. For information on AI agents, see [AI agents and agentic workflows in Simplified IT Service Management](../reference/agents-ai-native-it-service-desk.md).
-   Commonly requested catalog items. For information on catalog items, see [Catalog items installed with Simplified IT Service Management](../reference/catalog-items-base-system.md).

## Procedure

1.  From the **All** menu, navigate to **Admin Center** &gt; **Application Manager** &gt; **Available for you**.

2.  Search for the IT Service Management application \(sn\_ai\_itsm\_cont\).

3.  Review the application listing for information on dependencies, licensing or subscription requirements, and release compatibility.

4.  Select **Install**.

5.  Apply default configurations post installation of the IT Service Management application \(sn\_ai\_itsm\_cont\).

    1.  From the header of your ServiceNow instance, navigate to **Admin** &gt; **Admin Home**.

    2.  From the **Manage your products** section, select **View product overview** for IT Service Management.

        The Product Hub page for IT Service Management is displayed.

        ![Set up Simplified ITSM application](../image/getStartedInstall2.png)

    3.  In the **Installation progress** section, from the **Not Installed** tab, select **Apply default configurations** for IT Service Management.

        **Important:** Do not change the current scope while presets are being configured.

    4.  In the **Apply default configurations** pop-up window, select **Confirm**.

6.  Start configuring IT Service Management.

    1.  In the **Now you're ready to configure** pop-up window, select **Configure**.

        ![Start configuring Simplified IT Service Management](../image/ai-native-itsm-configure.png)

        The Configure IT Service Management page is displayed.

    2.  In the **Get started with configuration** pop-up window, review the configuration instructions and select **Got it**.
