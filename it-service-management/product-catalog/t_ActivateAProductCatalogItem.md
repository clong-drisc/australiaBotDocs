# Activate a product catalog item

Activate a product catalog item to make it available in the product catalog and the service catalog.

## Before you begin

Role required: catalog\_admin

## About this task

You can activate the item from either the list view or the record.

## Procedure

1.  Navigate to **All** &gt; **Product Catalog** &gt; **Catalog Definition** &gt; **Hardware &amp; Software Items**.

2.  Complete one of the following actions.

<table id="choicetable_wlj_fwj_k5"><tbody><tr><td id="d301305e92">

**Activate one or more items from the list view**

</td><td>

Select the check box next to one or more items in the record list and select **Activate** below the list.

</td></tr><tr><td id="d301305e109">

**Activate from the record**

</td><td>

Select **Activate** under **Related Links**.

</td></tr></tbody>
</table>
