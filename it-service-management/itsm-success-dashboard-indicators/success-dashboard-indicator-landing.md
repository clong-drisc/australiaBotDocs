# ITSM Success Dashboard indicators

ITSM Success Dashboard indicators provide insights to the IT leadership team and process owners so that they can measure the performance of their ITSM implementation using the KPIs defined by ServiceNow.

Success Dashboard is available for IT Service Management and HR Service Delivery and is delivered via the following applications:

-   ITSM Success Dashboard indicators

    The IT leaders can view the performance of the ITSM KPIs implemented.

-   HR Service Delivery Success Dashboard indicators

    The HR leaders can view the performance of the HR KPIs implemented. For more information on HR Success Dashboard indicators, visit [HR Success Dashboard indicators](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/employee-service-management/hr-service-delivery/success-dashboard-landing.md).
