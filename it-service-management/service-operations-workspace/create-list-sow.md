# Create a list in Service Operations Workspace

Create a list or use an existing list to create a list.

## Before you begin

Role required: itil

## Procedure

1.  From the primary navigation in Service Operations Workspace, select the **List** menu and select the **My Lists** tab.

2.  Select **Add new list**.

3.  To create a different version of the existing list, perform the following steps.

    1.  In the New List dialog box, select the **Start from existing** tab.

    2.  Specify the required details.

4.  To create your own list, perform the following steps.

    1.  In the New List dialog box, select the **Create your own** tab.

    2.  Specify the required details.

5.  Select **Create**.
