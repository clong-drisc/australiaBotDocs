# Open an IT ticket in Employee Center

Resolve a user's issue by opening an IT ticket.

## Before you begin

Role required: none

## Procedure

1.  Navigate to the Employee Center portal.

2.  Select **Open IT Ticket**.

    The AI-enabled record producer, Create incident with Now Assist, is displayed.

3.  In the **Please describe your issue below** field, enter the issue information.

    While you’re specifying the issue, Now Assist generated solutions are displayed.

4.  In the **Add attachments** field, select a file related to the issue.

    ![Create incident in employee portal](../image/AINativeCreateIncident.png)

5.  Select **Save as Draft** to save the ticket as a draft request to update and submit later.

6.  Perform one of the following actions.
