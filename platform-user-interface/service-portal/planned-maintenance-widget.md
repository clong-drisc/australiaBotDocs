# Planned Maintenance widget

Describes any planned system maintenance. You can use this base system widget as-is in your portal or clone it to suit your own business needs.

The widget gathers information from the cmdb\_ci\_outage table. Any planned maintenance within the following five days appears in the Planned Maintenance widget.

![Planned Maintenance widget](../image/WidgetPlannedMaintenance.png "Planned Maintenance widget")

## Instance options

The Planned Maintenance widget does not have any included instance options.
