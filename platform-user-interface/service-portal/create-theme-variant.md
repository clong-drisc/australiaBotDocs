# Create a theme variant

Create a theme variant from the Coral theme. A theme variant uses customizable styling CSS variables and branding options to tailor the visual experience for your users.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Service Portal** &gt; **Themes** &gt; **Coral**.

2.  Under the **Theme Variant Mappings** tab, select **New**.

3.  Add a name for the theme variant, such as Coral Blue.

4.  In Theme variant, choose a theme on which to base the new theme, such as **Coral Dark**.

5.  In a Theme variant, select the Information icon to change the CSS variables, Description, Logo, and Logo Alt Text.

6.  Match the new theme variant to an existing Next Experience theme, such as Dark.

7.  In Virtual Agent branding, select the associated Virtual Agent branding theme that displays when the theme variant is selected in Service Portal.

8.  Select **Submit**.
