# HTML widget

Use the HTML widget to directly inject HTML, text, lists, or content in general into a page. You can use this base system widget as-is in your portal or clone it to suit your own business needs.

Because the HTML widget can contain content of any kind, any example may just appear as regular text. For example, the HTML widget appears as a box of text on the 404 page.

![HMTL widget from the 404 page with text about the 404 page](../image/HTMLWidget.png "HTML widget")

## Instance options

|Field|Description|
|-----|-----------|
|Title|Name for the HTML text. Users do not see this information.|
|HTML|Text box for the HTML. Use the toolbar like most word processing tool bars. For more information on using the toolbar, see [Formatting icons for the HTML field editors](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/r_Formatting.md) .|
