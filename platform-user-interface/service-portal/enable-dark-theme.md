# Enable dark theme

Use a Coral dark theme on a portal to improve focus, readability, and accessibility.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Service Portal** &gt; **Home**, then select **User Profile**.

    **Note:** As a guest user, in the top navigation menu, select **Theme** and then select **Dark mode**.

2.  In Theme Variants, choose **Dark mode**.

    For more information about theme variants, see [Create a theme variant](create-theme-variant.md).

3.  Refresh the portal.
