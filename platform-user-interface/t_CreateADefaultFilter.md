# Create a default filter for a related list

Create a default filter for the records that load when your related list displays.

## Before you begin

Role required: admin

## About this task

**Note:** This functionality is not supported in List v3 related lists.

## Procedure

1.  Navigate to the related list \(example, **Incidents** on a problem record\).

2.  Create the desired filter using the condition builder \(example, **\[Active\] \[is\] \[true\]**\).

    ![Filter in a related list with the Set as Default Filter button highlighted](../image/DefaultFilter.png)

3.  Click **Set as Default Filter**.
