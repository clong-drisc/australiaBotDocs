# Common UI elements

This page lists common UI elements for the standard user interface.

-   **[Log in to an instance](../../navigation/task/t_LoggingIn.md)**  
Each ServiceNow instance has a unique, secure web address. The base URL for each instance has the default format: `https://<instancename>.service-now.com`.
-   **[Navigate directly to a table](../../../administer/navigation-and-ui/task/t_NavigateDirectlyToATable.md)**  
You can use commands in the navigation filter to navigate directly to the list, form, or configuration view of a table.
-   **[Tree picker](../concept/c_TreePicker.md)**  
The tree picker is a special reference lookup that you can add as an attribute to a form. Add the tree picker to the classic environment or to workspaces.
-   **[Dot-walking to data in related tables](../../navigation/concept/c_DotWalking.md)**  
Dot-walking provides access to fields on related tables from a form, list, or script.
-   **[Configure items on forms or in lists using a list collector](../task/t_UseSlushbuckets.md)**  
Use a list collector to add, remove, or move items in a form or list. List collectors allow you to select multiple items from a list of available items and to remove items from a list of selected items. Some list collectors provide filter and search controls for available items, such as adding items to related lists.
-   **[Activity streams](../../navigation/concept/c_activity-streams.md)**  
An activity stream is a list of entries in records and conversations, such as journal fields, comments, and work notes that display in task records and Sidebar conversations. Activity streams are available Core UI.
-   **[Create a condition statement using the condition builder](../task/create-cond-state-using-cond-build.md)**  
A condition builder constructs a condition statement with a series of contextually generated fields. Condition builders are used in many operations, such as creating filters, administering surveys, and administering access control.
-   **[Enable or disable an application menu or module](../../../administer/navigation-and-ui/task/t_EnDisableAppMenuOrMod.md)**  
You can enable \(show\) or disable \(hide\) an application menu or module in the application navigator.
-   **[Tags](../concept/c_Tags.md)**  
Tags are text labels that you can associate with items like records and pages. Tags enable you to group and organize the items. Tags can be visible to any user \(global\), visible only to specific groups or users \(shared\), or visible to a single user \(private\).
-   **[User presence](../../navigation/concept/c_UserPresence.md)**  
User presence is a Core UI feature that lets you see who is online when you're working in an instance.
