# Put a gauge on a CMS page

You have control over the gauges on a CMS page.

## Before you begin

Role required: content\_admin or admin

## Procedure

1.  Bring up the CMS page in edit mode.

2.  Click **Add Content**.

3.  Select the gauge you want to add.

4.  Place the gauge on the page in the desired location.
