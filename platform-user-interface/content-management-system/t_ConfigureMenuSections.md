# Configure menu sections

Menu sections define groups of links displayed within the navigation menu block.

## Before you begin

Role required: content\_admin or admin

## Procedure

1.  Navigate to **All** &gt; **Content Management** &gt; **Blocks** &gt; **Navigation Menus** and select a block.

2.  In the Menu Sections related list, click **New**.

3.  Complete the Menu Section form.
