# Customize a design theme

A theme is a collection of one or more style sheets \(CSS files\) that define a consistent look for a set of pages.

## Before you begin

Role required: content\_admin or admin

## About this task

Themes can be invoked in any of the following ways.

-   Directly by content pages.
-   As the default theme of a content site.
-   On the configuration page as a global default.

## Procedure

1.  Navigate to **All** &gt; **Content Management** &gt; **Design** &gt; **Themes**.

2.  Click **New**.

3.  Type a name for the theme and mark it as Active.

4.  Right-click the form header and click **Save**

    The **Style Sheet** related list appears.

5.  Use the **Style Sheet** related list to add style sheets to the theme.
