# Menu types

By changing the **Type** field on the navigation menu block, you can format the same menu in different ways.

|Type|Image|Description|
|----|-----|-----------|
|Super Menu for Headers|![CMS super menu](../image/CMSMenuSuper.png)|The super menu is a hybrid between the drop-down menu and the tabbed system. The user can use a super menu to create a menu from any number of menus.|
|Drop-Down Menu for Headers|![CMS menu dropdown](../image/CMSMenuDropdown.png)|The drop-down menu renders the menu sections as drop-down list. Use the mouse to point to the menu name and view the menu items.|
|Tab Menu for Headers|![CMS menu header tabs](../image/CMSMenuHeadertabs.png)|The tab menu renders the menu sections as tabs. Use the mouse to click the tab and view the menu items.|
|Horizontal Blocks|![CMS menu horizontal blocks](../image/CMSMenuHorizontalblocks.png)|The horizontal blocks menu renders the menu sections as block headings with menu items as links within the blocks. The blocks are arranged horizontally.|
|Tab Content Block|![CMS menu horizontal tabs](../image/CMSMenuHorizontaltabs.png)|The tab content block \(horizontal\) menu renders the menu sections as tabs with menu items as links within the block.|
|Vertical List|![CMS menu vertical list](../image/CMSMenuVerticallist.png)|The vertical list menu renders the menu sections as headings with menu items as links below them.|
|Vertical Blocks|![CMS menu vertical blocks](../image/CMSMenuVerticalblocks.png)|The vertical blocks menu renders the menu sections as block headings with menu items as links within the block. The blocks are arranged vertically.|

-   **[Menu style customization](../concept/c_CustomizeTheMenuStyle.md)**  
You can customize menu styles in the style sheets your site uses.
