# Disable feedback

You can disable the feedback providing mechanism by creating a system property and disabling it.

## Before you begin

Role required: admin

## Procedure

1.  In the unified navigation, type sys\_properties.list and press **Enter**.

2.  Select **New**.

3.  In the **Name** field, type **glide.ui.product\_feedback\_enabled**.

4.  In the **Type** field, select ![true-false](../image/true-false.png).

5.  In the **Value** field, select false.

6.  Save the system property.
