# Embedded lists or Related lists

Some forms may show related lists as embedded. Changes to embedded lists are saved when the form is saved.

Use these controls to work with an embedded list. For more information, see [Edit a form](../task/t_EditingInForms.md#).

|Task|Icon|Action|
|----|----|------|
|Expand an embedded list|![Expand embedded list icon.](../image/UI11FormExpand.png)|Click the expand icon in the list header.|
|Collapse an embedded list|![Collapse embedded list icon.](../image/UI11FormCollapse.png)|Click the collapse icon in the list header.|
|Insert a new row|![Insert a row.](../image/InsertRow.png)|Select **Add** and identify the record you want to add to the embedded list.|
|Edit a row| |Double-click in an empty area of that field. See [Use the list editor](../../using-lists/task/t_UseTheListEditor.md).|
|Delete a row|![Delete row icon.](../image/RowDeleteIcon.png)|Click the delete icon beside the row. New rows are removed immediately. Existing rows are designated for deletion when the record is saved. To clear this designation, click the delete icon again.|
