# Customize the banner logo in Core UI

Use the **glide.product.image** and **glide.product.description** properties to change the banner logo and description in Core UI.

## Before you begin

Role required: admin

## About this task

To update the banner image in the **System Properties** &gt; **System** page, you must first upload the image.

**Note:** Setting a banner image in the company profile overrides the image selected in **System Properties** &gt; **System**.

## Procedure

1.  To upload the image, complete the following steps.

    1.  Navigate to **System UI** &gt; **Images** and click **New**.

    2.  Enter a **Name** for your image.

        Valid names must end in `.gif`, .`png`, `.jpg`, or `.bmp`.

    3.  Click the **Click to add** link in the **Image** field, and select and upload the image.

    4.  Click **Update**.

2.  To set the uploaded image to the banner image, complete the following steps.

    1.  Navigate to **System Properties** &gt; **System**.

    2.  In the **Banner image** field, enter the file name of the uploaded image.

    3.  Click **Save**.

    **Note:** The system limits banner images to 50px.
