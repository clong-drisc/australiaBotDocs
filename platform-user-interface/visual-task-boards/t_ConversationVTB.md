# Create a Connect conversation from a Visual Task Board

Create a Connect Chat conversation from a Visual Task Board to collaborate with board members and keep track of activity as it happens.

## Before you begin

Role required: none

## About this task

When you create a conversation from a task board, all the board members become members of the conversation. The members are synchronized between the task board and the conversation. For example, if you remove a user from the conversation, the system automatically removes the user from the board as well.

## Procedure

1.  Navigate to **All** &gt; **Self-Service** &gt; **Visual Task Boards**.

2.  Open a task board.

3.  In the board header, click the connect with board members icon \(![Connect with board members icon](../image/IconVTBConnectwithBoard.png)\).

    The system automatically opens a record conversation for the board. Each board member becomes a member of the conversation.
