# Add a task to a freeform Visual Task Board from a form

You can add a task card to a freeform board from any table that extends Task. These cards represent records on whichever table you add them from.

## Before you begin

Role required: none

## About this task

You can add cards from multiple task tables, such as Incident or Problem, to the same freeform board.

## Procedure

1.  Open a task record, such as an incident.

2.  Click the menu icon \(![Menu icon](../image/additionalIcon.png)\) or right-click the form header to show the form context menu.

3.  Select **Add to Visual Task Board**.

4.  Click the name of the board you want to add the card to.

5.  To view the board, click the board name in the confirmation message.

    The task appears in the first lane by default.
