# Enable keyboard shortcuts

You can enable or disable keyboard shortcuts from the system settings menu.

## Before you begin

Role required: none

For a list of available keyboard shortcuts, see [Core UI keyboard shortcuts](../reference/r_KeyboardShortcuts.md).

## Procedure

1.  Select the user menu, and then select **Preferences**.

2.  Select **Accessibility**.

3.  Select or clear the **Enable keyboard shortcuts** option.

4.  Close the Preferences window.
