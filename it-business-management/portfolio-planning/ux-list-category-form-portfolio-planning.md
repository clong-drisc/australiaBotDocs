# UX List Category form

Learn about the fields of the UX List Category form. Use this form to create additional categories for lists in the Lists menu of Portfolio Planning workspaces.

<table id="table_r1k_rbt_2wb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Title

</td><td>

Display name of the category in the workspace.

</td></tr><tr><td>

Application

</td><td>

Application scope. Ensure that this is set to **Portfolio Planning**.

</td></tr><tr><td>

Order

</td><td>

Order in which this category is displayed in the workspace.

</td></tr><tr><td>

Active

</td><td>

Option to enable the display of this category in the workspace.

</td></tr><tr><td>

Description

</td><td>

Brief description of the category.

</td></tr><tr><td>

Configuration

</td><td>

Configuration that the category is referencing.

 By default, this field is set to **APW List Menu Configuration**.

</td></tr></tbody>
