# Create a test set in Test Management 2.0

Group related tests into a test set. Test sets help you find tests by category or tag.

## Before you begin

Role required: sn\_test\_management.tester or sn\_test\_management.test\_manager

## Procedure

1.  Navigate to **All** &gt; **Test Management 2.0** &gt; **Test Sets**.

2.  Click **New**.

3.  Enter a name for the test set and click **Submit**.


## What to do next

You must add tests to the test set.

1.  Navigate to **Test Management 2.0** &gt; **Tests**.
2.  Select the required tests.
3.  Click **Add to Test Set**.
4.  In the Select Test Set window, search for the required test set and click **Add**.

To reorder the tests in the test set, open the test set record and use the drag-and-drop functionality to move it to the required position.

**Tip:**

To change the order of steps within an individual test, see [Create a test in Test Management 2.0](create-test2.md).
