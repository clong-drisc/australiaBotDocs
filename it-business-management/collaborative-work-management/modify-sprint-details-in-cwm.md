# Update sprint details in CWM

Modify the duration, capacity, and dates for a sprint in Collaborative Work Management.

## Before you begin

Role required: sn\_cwm.cwm\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **Collaborative Work Management**.

2.  From a Space, select a Board that you're performing sprint planning for.

3.  Select the Sprint planning tab.

4.  From the sprint header, select the More options icon \(![More options icon.](../images/cwm-icon-more-options.png)\) and select **Edit**.

    ![Edit Sprint details in CWM.](../images/cwm-edit-sprint.png)

5.  Make the necessary edits to the sprint details.

6.  Select **Save**.
