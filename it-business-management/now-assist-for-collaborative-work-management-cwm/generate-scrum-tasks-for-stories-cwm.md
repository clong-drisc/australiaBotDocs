# Generate scrum tasks for a story with Now Assist for CWM

Use Now Assist to automatically generate scrum tasks for a story in Collaborative Work Management \(CWM\) based on the story's description and context, saving time on manual task breakdown.

## Before you begin

Role required: sn\_cwm\_ai.cwm\_ai\_user

## About this task

Generate scrum tasks in CWM using Now Assist 

Now Assist generates scrum tasks by analyzing the story's description and context. You can generate scrum tasks in two ways: with review, where you evaluate and approve the suggestions before they are added to the story, or without review, where suggestions are generated and added to the story immediately using the inline Now Assist icon from the Board.

## Procedure

1.  Navigate to **Workspaces** &gt; **Collaborative Work Management**.

2.  From a Space, select a Board.

3.  Open the story you want to generate scrum tasks for.

    You can open a story from any board view: List, Gantt, Kanban, or Sprint planning.

4.  On the story form, select the **Scrum Tasks** tab.

5.  Select **Create with Now Assist**.

    Now Assist analyzes the story description and context and generates scrum task recommendations.

    ![Now Assist scrum task recommendations displayed on the Scrum tasks tab of a story form in CWM.](../images/na-cwm-scrum-tasks-recommendations.png)

6.  Review the generated recommendations.

    If the suggestions don't match the story, select **Refine** &gt; **Regenerate** to produce a new set of recommendations.

7.  Select **Insert** to add the scrum tasks to the story.

    ![Scrum tasks generated by Now Assist added to a story in CWM.](../images/na-cwm-scrum-tasks-generated.png)


## Result

The generated scrum tasks are added to the story and appear in the **Scrum Tasks** tab. You can edit any of the generated tasks after they're added.

## What to do next

**Tip:**

For a quicker path that skips the review step, hover your cursor over the story's short description in the List, Gantt, or Sprint planning views to reveal the Now Assist scrum tasks icon \(![](../images/cwm-na-icon-scrum-tasks-inline.png)\) and select it. The scrum tasks are generated and added to the story immediately without a review step.

To add scrum tasks manually instead, see [Add scrum tasks to a story in CWM](../../collab-work-mgmt/task/add-scrum-tasks-to-a-story-cwm.md).
