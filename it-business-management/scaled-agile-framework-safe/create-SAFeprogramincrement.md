# Define a program increment in SAFe

With SAFe, define a program increment \(PI\). A program increment is typically 8–12 weeks long, during which an ART delivers incremental value in the form of working, tested software and systems.

## Before you begin

Role required: safe\_admin

## Procedure

1.  Navigate to the program increment form using either of the following options.

1.  Navigate to **Scaled Agile Framework \(SAFe\)** &gt; **SAFe Board**.
2.  From the list, select **ART**.
3.  From the adjacent list, select the required ART value.
4.  Click the **Backlog** tab.
5.  Click **Create Program Increment**.
