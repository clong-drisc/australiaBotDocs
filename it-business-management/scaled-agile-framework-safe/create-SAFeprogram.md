# Define an Agile Release Train

From SAFe, define an Agile Release Train \(ART\) which is a group of agile teams working towards a single solution.

## Before you begin

Role required: safe\_admin

## Procedure

1.  Create an agile release train using either of the following options.

<table id="choicetable_u3p_dtb_kfb"><thead><tr><th align="left" id="d63968e63">

Option

</th><th align="left" id="d63968e66">

Steps

</th></tr></thead><tbody><tr><td id="d63968e72">

**From the Agile Release Train module**

</td><td>

1.  Navigate to **Scaled Agile Framework \(SAFe\)** &gt; **Agile Release Trains**.
2.  Click **New**.


</td></tr><tr><td id="d63968e102">

**From the Portfolio form**

</td><td>

1.  Navigate to **Scaled Agile Framework \(SAFe\)** &gt; **Portfolios**.
2.  In the **Agile release trains** related list, click **New**.


</td></tr></tbody>
</table>2.  Enter a suitable name for the agile release train and click **Submit**.
