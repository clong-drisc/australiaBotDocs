# Apply or change a task constraint

Apply a constraint to control when an automatically scheduled task starts or finishes. Set the constraint type and the associated constraint date if required.

## Before you begin

Role required: it\_project\_manager

Verify that the task is set to automatic scheduling. Constraints apply only to automatically scheduled tasks.

## Procedure

1.  Navigate to **All** &gt; **Project** &gt; **Projects** &gt; **All**.

2.  Open a project and select Gantt Chart view.

3.  Select the task for which you want to set a constraint.
