# Modify a project template configuration item

Modify a project template configuration item included in a template.

## Before you begin

Role required: it\_pps\_admin

## Procedure

1.  Navigate to **All** &gt; **Project Administration** &gt; **Settings** &gt; **Template Config**.

2.  Select a configuration item from the list.

3.  Make any desired changes to the fields on the Project Template Configuration form.

4.  Select **Update**.

    To delete a project template, select a configuration item from the list and select **Delete**. Do not delete any of the default template configuration items: project, project task, or project subtask. Doing so affects the ability to create project templates.
