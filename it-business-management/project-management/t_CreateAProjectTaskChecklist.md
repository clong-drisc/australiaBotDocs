# Use a project task checklist

You can create a checklist in a project task to track items that must be completed in the task.

## Before you begin

Role required: it\_project\_user, it\_project\_manager

## Procedure

1.  Navigate to a project task.

2.  To create a checklist for the project task, select **Create new** in the **Checklist** section and specify a name for the checklist.

3.  To remove a checklist from the project task, select the down arrow next to **Checklist** and select **Remove Checklist**.

4.  **Note:** This option appears when there is at least one checklist in the **Checklist** section.

    To save a checklist as a template, select the down arrow next to **Checklist** and select **Save as Template**.


-   **[Save a checklist as a template](t_SaveAChecklistAsATemplate.md)**  
After you create a checklist on a project task, you can save it as a template and reuse it on other tasks.
