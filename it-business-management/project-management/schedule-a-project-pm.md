# Schedule a project from a start date

Schedule a project from a start date and plan tasks and milestones based on your project requirements.

## Before you begin

Role required: it\_project\_manager

## Procedure

1.  Navigate to **All** &gt; **Project** &gt; **Projects** &gt; **Create New**.

2.  On the Project form, enter the project name.

3.  From Dates related list, select **Schedule** and select **Approved start date**.

    All tasks without dependencies or constraints are set to start on the project start date.

4.  Select **Submit**.
