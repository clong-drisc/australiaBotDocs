# View a user calendar

View the calendar of a user to check availability.

## Before you begin

Role required: resource\_manager

## About this task

You can view a calendar for a specific user, and not for all users in a group.

## Procedure

1.  Navigate to **All** &gt; **Resource** &gt; **Resources** &gt; **Users**.

2.  Do one of the following options.

<table id="choicetable_g3t_jz2_zs"><tbody><tr><td id="d167074e86">

**From the User list**

</td><td>

Right-click a user name in the Users list and select **View Calendar**.

</td></tr><tr><td id="d167074e98">

**From the User form**

</td><td>

Select a User record to open the User form, and then select the **View Calendar** related link.

</td></tr></tbody>
</table>
