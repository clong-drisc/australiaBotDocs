# Create and compare financial baselines for your planning items

Create and compare financial baselines to get a snapshot of the project expenses and compare the planned costs against the actual expenses.

## Before you begin

Role required: sn\_align\_ws.spw\_financial\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **Strategic Planning Workspace** and select portfolio plan.

2.  Select a planning item from the Planning module.

3.  Select the **Financials** tab.

4.  Create or compare baselines.
