# Planning item milestone form

Learn about the fields of the planning item milestone form. Use this form to create milestones for the planning items in your roadmap in Strategic Planning workspace.

<table id="table_planning_milestone"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

</td><td>

Preferred name for your milestone.

</td></tr><tr><td>

Type

</td><td>

Milestone type. Select one from the following:-   Deadline
-   Key Milestone
-   Key Event
-   Launch Date
-   Important Date

</td></tr><tr><td>

Due date

</td><td>

The date of the milestone.

</td></tr><tr><td>

State

</td><td>

Status of the milestone. Select one from the following:-   Upcoming
-   Missed
-   Achieved

</td></tr><tr><td>

Assigned to

</td><td>

User who is assigned to the milestone.

</td></tr><tr><td>

Description

</td><td>

Brief description about the milestone.
