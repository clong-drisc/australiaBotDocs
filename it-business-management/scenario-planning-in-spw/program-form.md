# Strategic program form

Learn about the fields of the Strategic program form. Use this form to create or edit the details of a Strategic Program in Strategic Planning.

These fields are displayed on the full details page of the strategic program.

<table id="table_l4d_3vm_3tb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Brief description of the strategic program.

</td></tr><tr><td>

Description

</td><td>

Detailed information of the strategic program.You can refine strategic program description using the Refine records skill, if the Now Assist for SPM application is installed.

</td></tr><tr><td>

State

</td><td>

Current working state of the strategic program.

</td></tr><tr><td>

Initiative

</td><td>

Initiative that this strategic program is associated to.

</td></tr><tr><td>

Owner

</td><td>

User assigned to this strategic program.

</td></tr><tr><td>

Percent complete

</td><td>

Percentage of work completed for this strategic program.

</td></tr><tr><td>

MoSCoW

</td><td>

Priority of this strategic program based on its importance. The available options are:-   **1 - Must have**
-   **2 - Should have**
-   **3 - Could have**
-   **4 - Won’t have**

</td></tr><tr><td>

Work notes

</td><td>

Notes to capture the updates and discussions on the strategic program as the work progresses.
