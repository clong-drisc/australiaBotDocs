# Update the actual value of a target breakdown

Update the actual value of a target breakdown when its status is changed.

## Before you begin

Role required: sn\_apw\_advanced.spw\_goal\_user and \(sn\_align\_core.apw\_user or sn\_gf.goal\_admin\)

## About this task

Updating the actual value of a target breakdown automatically updates the actual value of its target. After the **Actual to date** field of the target is updated, the progress value for the target and its goal are auto-updated.

**Note:** Only the owner or contributors of the goal can update the target breakdowns.

## Procedure

1.  Open the target record of the goal using one of the following options.

## Result

After the **Actual** field is updated, the progress value for the target and its goal are auto-updated. For information on how the progress value is calculated, see [progress value calculation](../../alignment-planner-workspace/concept/progress-value-calculation-egm.md). For information on how the actual value of the target breakdown is rolled up to its target, see [Target breakdowns in Strategic Planning](../concept/target-breakdowns.md).
