# Create or modify a lens in Strategic Planning

Start defining your company's planning perspectives in Strategic Planning by creating a lens.

## Before you begin

Role required: admin

## About this task

Create a lens record and add the type of items \(Project, Demand, and others\) that can be planned using this lens. The type of planning items that you associate with the lens are the only type of work that the planning managers can use, in their portfolio plans.

## Procedure

1.  Navigate to **All** &gt; **Strategic Planning** &gt; **Lenses**.

2.  Select **New**.

3.  On the form, fill in the fields.

    For field information, see [Lens form](../reference/lens-form-alignment-planner-workspace.md).

4.  Select **Submit**.
