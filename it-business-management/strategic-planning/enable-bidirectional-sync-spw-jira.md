# Enable bidirectional sync between SPW and Jira

Enable import and export for a Jira project to ensure data is consistent across for all stakeholders using Strategic Planning Workspace and Jira.

## Before you begin

Role required: sn\_jira\_int.user

## Procedure

1.  Navigate to **All** &gt; **Strategic Planning** &gt; **Jira Integration** &gt; **Jira Projects**.

2.  Open your Jira project record.

3.  Select both the **Enable Import** and **Enable Export** options.

4.  Select **Update**.
