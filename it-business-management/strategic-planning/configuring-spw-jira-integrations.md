# Configuring SPW Jira Integrations

Configure the set up and implementation of SPW Jira Integrations. You can either follow the tasks listed in the configuration or use Guided Setup.

## Configuration overview

1.  [Install SPW Jira Integrations](../task/install-spw-jira-integrations.md).
2.  Assign the Jira integration admin role. See [Components installed with SPW Jira Integrations](../reference/components-installed-with-spw-jira-integrations.md).
3.  [Set up a Jira instance](setting-up-jira-instance-spw.md).
4.  [Connect SPW to Jira](../task/connect-spw-to-jira.md).
5.  [Import Jira projects to SPW](../task/imports-jira-projects-to-spw.md).
6.  Configure Jira Project settings.
    1.  [Associate Jira projects with EAP teams](../task/create-team-integration-settings-for-jira-project-in-spw.md).
    2.  [Review default mapping configuration for Jira projects in SPW](../task/review-default-map-config-jira-projects-spw.md).
    3.  \(Optional\) [Create custom mapping configuration](custom-mapping-configuration-spw-jira.md#).
    4.  [Enable bidirectional sync between SPW and Jira](../task/enable-bidirectional-sync-spw-jira.md).
7.  [Import issues from Jira to SPW](../task/import-issues-from-jira-to-spw.md).
8.  Repeat steps 6 and 7 for all imported Jira projects.
9.  [Activate Jira configuration in Enterprise Agile Planning](../task/activate-jira-configuration-in-eap.md).
10. [Define team structure in Enterprise Agile Planning](../task/define-agile-structure-eap-teams-spw-jira-integrations.md).

## Using guided setup to implement SPW Jira Integrations

Guided setup provides a sequence of tasks that help you configure SPW Jira Integrations on your ServiceNow instance. To open guided setup for SPW Jira Integrations, navigate to **All** &gt; **Strategic Planning** &gt; **Jira integration** &gt; **Guided Setup**.
