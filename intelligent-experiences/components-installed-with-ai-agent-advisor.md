# Components installed with AI Agent Advisor

Several components are installed with the AI Agent Advisor application.

## Roles installed

The following roles are installed with AI Agent Advisor.

-   AI Agent Advisor admin \[sn\_agent\_miner.app\_admin\]
-   AI Agent Advisor UI \[sn\_agent\_miner.app\_ui\]
-   AI Agent Advisor read only \[sn\_agent\_miner.app\_read\_only\]

For information on the roles installed with AI Agent Advisor, see [AI Agent Advisor roles](ai-agent-advisor-roles.md#).

## Tables installed

The following tables are installed with AI Agent Advisor.

-   AI Agent Advisor Agent Match \[sn\_agent\_miner\_agent\_match\]
-   AI Agent Advisor Agent Rank \[sn\_agent\_miner\_agent\_rank\]
-   AI Agent Advisor Artifact Dataset \[sn\_agent\_miner\_artifact\_dataset\]
-   AI Agent Advisor Clusters \[sn\_agent\_miner\_clusters\]
-   AI Agent Advisor Configs \[sn\_agent\_miner\_configs\]
-   AI Agent Advisor Created Automation \[sn\_agent\_miner\_created\_automation\]
-   AI Agent Advisor Dataset \[sn\_agent\_miner\_dataset\]
-   AI Agent Advisor Executions \[sn\_agent\_miner\_executions\]
-   AI Agent Advisor Existing Automation \[sn\_agent\_miner\_existing\_automation\]
-   AI Agent Advisor Resolution Steps \[sn\_agent\_miner\_resolution\_steps\]
-   AI Agent Advisor Scheduled Jobs \[sn\_agent\_miner\_sysauto\_script\]
-   AI Agent Advisor Tool Match \[sn\_agent\_miner\_tool\_match\]
