# Activate Now Assist panel assistants

You must activate Now Assist panel assistants before using them.

## Before you begin

Role required: admin

**Note:** Voice input is automatically activated when you activate the Now Assist panel.

## Procedure

1.  Navigate to **All** &gt; **Conversational Interfaces** &gt; **Assistants**.

2.  On the Assistants screen, select **Now Assist Panel - Platform \(default\).**![Assistants screen that displays Now Assist panel assistants.](../images/now-assist-panel-assistants-screen.png)

    The Now Assist Panel - Platform \(default\) page displays.

3.  If the Review tab does not display on the Now Assist Panel - Platform \(default\) page, select **Review**.![image.now-assist-panel-review-tab]

4.  On the Review tab, activate the Now Assist panel assistant.
