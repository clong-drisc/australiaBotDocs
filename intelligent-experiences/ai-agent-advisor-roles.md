## AI Agent Advisor admin \[sn\_agent\_miner.app\_admin\]

This role has full access to the AI Agent Advisor application and can access AI Agent Advisor tables.

### Contains Roles

List of roles contained within the role.

-   sn\_aia.admin
-   sn\_customerservice.case\_write\_granular

None.

None.
