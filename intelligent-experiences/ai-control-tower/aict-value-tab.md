# Value tab in AI Control Tower

Use the Value dashboard page to gain insights into the value realized from multiple types of AI in your organization.

The Value dashboard page contains visualizations that help you assess the value of your AI assets. Creator skills are included in the Value dashboard page. Value metrics for creator skills are aggregated across non-production and production instances.

**Note:** Depending on your license, you will have access to certain application features, generative AI skills, agentic workflows, and AI agents. For more information, see [ServiceNow product tiers](../../ai-implementation/concept/ai-native-sku-overview.md).

![Value tab in AI Control Tower.](../image/value-tab-aict.png "Value tab in AI Control Tower")

The indicators on the dashboard are based on the value templates associated with each AI asset. See [Using value templates](using-value-templates.md) for more information.

-   **Top 5 AI systems by value**

    This area of the dashboard shows the five AI systems that generated most value \(in hrs\).

    ![Top 5 AI systems by value.](../image/aict-value-top-5-assets-by-value.png "Top 5 AI systems by value")


-   **Productivity**

    This area of the dashboard shows the estimated productivity gains \(in hours\) in the selected period as a result of using various AI assets.

    ![Productivity indicator.](../image/aict-value-productivity.png "Productivity")

-   **Average AI users**

    This area of the dashboard shows the average unique users who used an AI asset in the selected date range.

    ![Average AI users.](../image/aict-value-avg-ai-users.png "Average AI users")

-   **Top 5 AI assets by value \(in hrs\)**

    This area of the dashboard shows the top 5 AI assets that delivered the most value to the users in the selected date range.

    ![Top 5 AI assets by value in hours.](../image/aict-value-top-5-assets-by-value.png "Top 5 AI assets by value (in hrs)")


## Daily updates

-   **Daily AI actions**

    This area of the dashboard shows the number of AI actions per each AI asset.

    ![Daily AI actions.](../image/aict-value-daily-ai-actions.png "Daily AI actions")

-   **Daily unique users**

    This area of the dashboard shows the number of daily unique users per each AI asset.

    ![Daily unique users.](../image/aict-value-daily-unique-users.png "Daily unique users")

-   **Top 5 AI systems by value**

    This area of the dashboard shows the daily value delivered \(in Mins\) by the top 5 AI systems.

    ![Top 5 AI systems by value in hours.](../image/aict-value-top-5-ai-systems-by-value.png "Top 5 AI systems by value (in Mins)")


## All systems

-   **System summary**

    This area of the dashboard lists all the AI systems along with details of users and productivity gained.

    ![AI system summary.](../image/aict-value-system-summary.png "System summary")
