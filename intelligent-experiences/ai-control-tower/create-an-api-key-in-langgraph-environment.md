# Create an API Key for LangGraph

Create an API Key for the AI Connector to authenticate with LangGraph.

## Before you begin

Role required: Workspace user \(Organization user or Organization editor\).

## Procedure

1.  Navigate to **Settings** &gt; **API Keys in LangSmith**.

2.  Select**+API Key**.

    API Key is created.

3.  Save the API Key.

    **Note:** Verify that the API Key grants access to the workspace where you intend to explore AI assets and usage details. Store the API key in a secure and private location.


## Result

An API Key is created.
