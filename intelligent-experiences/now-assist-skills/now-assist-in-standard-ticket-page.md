# Now Assist in Standard Ticket Page

The Now Assist in Standard Ticket Page skill allows you to display summaries generated by Now Assist, providing overviews of recent activities and ticket details.

When this skill is active, the standard ticket page on the Service Portal displays a **Summarize** button. If requesters wish to see the ticket activity and a summary of information generated by Now Assist, they can select the **Summarize** button.
