# Data Catalog

The Data Catalog module within the Workflow Data Fabric application helps you discover, understand, and manage your organization's data assets. Use the Data Catalog to search for data across multiple systems, view comprehensive metadata and relationships, track data lineage, and connect to data sources for analysis.
