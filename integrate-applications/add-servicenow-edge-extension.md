# Add the ServiceNow RPA Edge extension

Add the ServiceNow RPA Edge extension to your Edge browser to enable the Chromium connector to interact with the elements on the web applications that are opened in this browser.

## Before you begin

Role required: none

## Procedure

1.  Navigate to [https://microsoftedge.microsoft.com/addons/detail/servicenow-rpa-edge-exten/cdbeoliogepmegmacldfbklgboifhkhg](https://microsoftedge.microsoft.com/addons/detail/servicenow-rpa-edge-exten/cdbeoliogepmegmacldfbklgboifhkhg).

2.  Click **Get**.

3.  In the Add " ServiceNow, Inc. RPA Edge Extension" to Microsoft Edge dialog box, click **Add extension**.

4.  To verify if the extension is added, do the following actions:

    1.  Open the Edge browser.

    2.  On the top-right corner, click the three dots icon \(![Three-dots-icon.](../image/three-dots.png)\).

    3.  Click **Extensions**.

    4.  In the Extensions pop-up window, click **Manage extensions**.

    5.  Make sure that the ServiceNow, Inc. RPA Edge extension slider is enabled.![Edge-extension-slider.](../image/edge-slider.png)
