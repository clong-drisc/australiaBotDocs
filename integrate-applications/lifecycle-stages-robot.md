# Life-cycle stage statuses \(LLCS\) of a robot in RPA Hub

The state transition of a robot guides you through the stages of its life cycle.

<table id="table_t51_21c_b5"><thead><tr><th>

Life-cycle stage status

</th><th>

Description

</th></tr></thead><tbody><tr><td>

In Use

</td><td>

Default value when a new robot is created or saved.

</td></tr><tr><td>

In Maintenance

</td><td>

Robot or a robot machine that is under maintenance.

</td></tr><tr><td>

Retired

</td><td>

Robot is inactive.After you retire a robot, you cannot change the stage.

 You cannot retire a robot which is assigned to a bot process.
