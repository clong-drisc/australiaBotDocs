# Reflection

The Reflection components in the RPA Desktop Design Studio perform standard Reflection operations on objects and classes. For example, retrieve object properties.

-   **[Use the GetListItem component](../task/use-reflection-getlistitem.md)**  
Return items from a list by using the GetListItem component in RPA Desktop Design Studio.
-   **[Use the GetProperty component](../task/use-reflection-getproperty.md)**  
Access single or multiple properties of a given object at a time by using the GetProperty component in RPA Desktop Design Studio.
-   **[Use the SetListItem component](../task/use-reflection-setlistitem.md)**  
Return the specific defined st items by using the SetListItem component in RPA Desktop Design Studio.
-   **[Use the SetProperty component](../task/use-utility-reflection-setproperty.md)**  
Set the properties of an object instance by using the SetProperty component in RPA Desktop Design Studio.
-   **[Properties of the Reflection components](../reference/util-reflection-prop.md)**  
Learn about the properties of the Reflection components in RPA Desktop Design Studio.
