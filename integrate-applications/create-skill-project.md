# Create a skill project in RPA Desktop Design Studio

Create a skill automation project in RPA Desktop Design Studio that can be reused in other automations.

## Before you begin

Set up the RPA Desktop Design Studio application and add the ServiceNow instance details. For more information, see [Set up RPA Desktop Design Studio](set-up-rpa-studio.md).

Role required: none

## About this task

A skill automation is a reusable component. You can build a skill so that you can provide prebuilt capabilities for other automation processes. For example, if a login functionality is used in four different automations, that login functionality can be created as a skill that you can reuse across automations.

## Procedure

1.  On the RPA Desktop Design Studio home page, select the down arrow in the **Create Automation** button.

2.  Select **Create Skill**.

    ![Create Skill automation button.](../image/skill-automation-button.png "Create Skill automation button")

3.  In the **Name** field, enter the name of the skill automation project.

4.  In the **Description** field, enter the description of the skill automation project.

5.  In the **Location** field, enter the location of the skill automation project or retain the default location.

    **Note:** By default, the RPA Desktop Design Studio provides the latest versions of certain plugins with the automation project that you create. You can add more plugins after you create the project. In the Create new skill window, view the default plugins by selecting **View**, next to **Using latest plugins**.

6.  Select **Create**.
