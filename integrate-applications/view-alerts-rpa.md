# View alerts in RPA Hub

View notifications that are generated by the alert rules. For example, you can view alerts that pertain to robots, schedules, process jobs, and so on.

## Before you begin

Role required: sn\_rpa\_fdn.rpa\_release\_manager, sn\_rpa\_fdn.rpa\_business\_user, sn\_rpa\_fdn.rpa\_support\_user, sn\_rpa\_fdn.rpa\_developer, or sn\_rpa\_fdn.rpa\_admin

## Procedure

1.  Navigate to **All** &gt; **Robotic Process Automation** &gt; **RPA Hub Workspace**.

2.  Select the list icon \(![List icon.](../image/rpahublist-icon.png)\).

3.  On the **Lists** tab, in **Monitor**, select **Alerts**.

4.  Open an alert to view the details.
