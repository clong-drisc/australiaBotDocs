# Encryption components

The RPA Desktop Design Studio secures data through encryption, file hashes, and encoding. This feature of RPA Desktop Design Studio can be the part of a workflow.

-   **[Use the DecryptString component](../task/use-security-encryption-decryptstring.md)**  
Decrypt an encrypted string by providing the encryption key using the Decrypt String component in RPA Desktop Design Studio.
-   **[Use the EncryptString component](../task/use-security-encryption-encryptstring.md)**  
Encrypt a string by providing the encryption key with the EncryptString component in the RPA Desktop Design Studio.
-   **[Use the GenerateFileHash component](../task/use-security-encryption-generate-file-hash.md)**  
Generate the file HashCode for any file using the GenerateFileHash component in RPA Desktop Design Studio.
-   **[Use the SecureStringDecode component](../task/use-security-encryption-securestringdecode.md)**  
Decode a secure string by using the SecureStringDecode component in RPA Desktop Design Studio.
-   **[Use the SecureStringEncode component](../task/use-security-encryption-securestringencode.md)**  
Encode a string to secure it by using the Secure String Encode component in RPA Desktop Design Studio.
-   **[Properties of the Encryption components](../reference/properties-security-encryption-components.md)**  
Learn about the properties of the Encryption components in RPA Desktop Design Studio.
