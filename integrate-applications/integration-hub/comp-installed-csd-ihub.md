# Components installed with client software distribution

Several types of components are installed with client software distribution \(CSD\).

-   **[Tables installed](csd-tables.md)**  
These tables are installed with the Client Software Distribution plugin \(com.snc.orchestration.client\_sf\_distribution\).
-   **[Plugins installed](csd-installed-plugins.md)**  
These plugins are installed with the Client Software Distribution plugin, if they are not already active.
-   **[Roles installed](csd-installed-roles.md)**  
These roles are installed with the Client Software Distribution plugin.
-   **[Script includes installed](csd-script-incl-installed.md)**  
These script includes are installed with the Client Software Distribution plugin.
-   **[Properties installed](csd-prop-installed.md)**  
Properties for client software distribution \(CSD\) manage the installation status of requested software deployments and configure scheduled Discoveries.
-   **[Flows and subflows installed](csd-flows-installed.md)**  
These flows and subflows are installed with the Client Software Distribution plugin.
