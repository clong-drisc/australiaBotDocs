# Life-cycle stage statuses \(LLCS\) of a package in RPA Hub

The State transition of a package guides you through the stages of its life cycle.

<table id="table_pyx_mhd_4qb"><thead><tr><th>

Life-cycle stage statuses

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Published

</td><td>

Default value for a new package.

</td></tr><tr><td>

In-maintenance

</td><td>

Package is under maintenance.If a package is in the **In-maintenance** status, it cannot be associated with a bot process record.

</td></tr><tr><td>

Retired

</td><td>

Package is inactive. After you retire a package, you can't change the stage.

 When you retire a package, the package versions that are associated with this package are also retired.

</td></tr></tbody>
