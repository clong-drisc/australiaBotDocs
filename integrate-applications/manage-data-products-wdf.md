# Managing data products

Create, publish, and promote data products to make governed data available to consumers in your organization.

A data product groups one or more data interfaces into a governed, publishable collection. Once published, consumers can discover the data product in the Data Catalog and request access. When your data product is ready for production use, you promote it from your development instance using a ServiceNow update set.

-   **[Create a data product](../task/create-data-product-wdf.md)**  
Create a data product to group one or more data interfaces into a governed, publishable collection that consumers can discover and access through the Data Catalog.
-   **[Publish a data product](../task/publish-data-product-wdf.md)**  
Publish a data product to make it available to consumers in the Data Catalog.
