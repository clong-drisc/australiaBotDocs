# Review the bot processes

Review a list of available bot processes in Automation Center, so that you can make sure that they are on track.

## Before you begin

Roles required: sn\_ac.automation\_business\_user, sn\_ac.automation\_technical\_user, or sn\_ac.automation\_admin.

In addition to the Automation Center roles, you will also need the Robotic Process Automation \(RPA\) roles. For more information on RPA roles, see [Components installed with RPA Hub](../../rpa-hub/reference/installed-with-rpa-hub.md).

## About this task

A bot process is a configuration of the robot.

You can't create a bot process.

## Procedure

1.  Navigate to **All** &gt; **Automation Center** &gt; **Automation Center Home**.

2.  Select the list icon \(![List icon.](../images/autocenter-list-icon.png)\).

3.  On the **Lists** tab, under **Monitor**, select **All Bot Processes**.

4.  Open the bot process that you want to view.

5.  Review the information on the bot process page.
