# View action logs

View action logs to see details of actions that are executed.

## Before you begin

Role required: sn\_ac.automation\_business\_user, sn\_ac.automation\_technical\_user, catalog\_admin, or sn\_ac.automation\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Automation Center Workspace**.

2.  Select the Lists icon \(![Lists icon](../images/list-icon.png)\) on the left.

3.  Select **Monitor** &gt; **Action Logs**.

4.  The logs of all executed actions is displayed.
