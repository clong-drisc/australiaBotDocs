# Configuring data import

Configure the details of the data import, such as the kind of data, the schedule of the import, and so on.

Perform the following tasks in the same order to configure importing your automation data.

-   **[Schedule data import](../task/schedule-import.md)**  
Schedule the frequency for the data import to ensure that the automation data is updated as required.
-   **[Select a spoke](../task/select-spoke.md)**  
Choose a spoke to set up a connection. This is an optional step.
-   **[Select an action](../task/select-action.md)**  
Select an action suitable for your data import into ServiceNow instance.
-   **[Configure batch import for data](../task/config-batch-import.md)**  
Configure batch import of data to specify the pagination method for importing large data sets.
-   **[Verify and test](../task/verify-test.md)**  
Verify and test all configuration details to ensure that the flow is generated correctly.
