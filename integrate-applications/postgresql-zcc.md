# PostgreSQL

The PostgreSQL connector provides read-only access to data and metadata from a PostgreSQL instance.

A connection admin can create a connection to PostgreSQL and give data stewards access to that connection in Zero Copy Connector Hub. A data steward can select the established connection to create a data fabric table and map data from PostgreSQL. This allows users to retrieve PostgreSQL data through the table list view or GlideRecord scripts. For details on creating data fabric tables and mapping data, see [Managing data fabric tables](managing-data-fabric-tables-zcc.md).
