# Map connector to system

After creating a connector, configure connectors with an external system.

## About this task

Connectors must be mapped to an external system before they can be used. If you want to link a unmapped connector \(spoke or MCP\) to an external system, you must map it to an external system.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Workflow Data Fabric** &gt; **Connect Hub**.

2.  Click **Connectors**.

3.  Click **Map System** in the System Name column.

4.  Choose an existing system to map the connector to.

    To create an new external system, click ![Add new external system](../../../administer/flow-designer/images/create-new-record-icon.png) to [add a new external system](connecthub-create-external-systems.md#) to map an existing connector.

5.  Choose the connector type.

6.  Click **Save** to complete the mapping.
