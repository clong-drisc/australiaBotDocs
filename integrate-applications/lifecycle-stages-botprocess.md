# Life-cycle stage statuses \(LLCS\) of a bot process in RPA Hub

The State transition of a bot process guides you through the stages of its life cycle.

<table id="table_pyx_mhd_4qb"><thead><tr><th>

Life-cycle stage status

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Build

</td><td>

Default value for a new bot process.

</td></tr><tr><td>

Published

</td><td>

The bot process is ready to be active.

</td></tr><tr><td>

In Maintenance

</td><td>

When changes are to be done in the bot process.If a bot process is in **In Maintenance** status, you cannot start or stop a bot process or launch a bot process in RPA Desktop Design Studio.

</td></tr><tr><td>

Retired

</td><td>

Bot process is inactive. After you retire a bot process, you cannot change the stage.

</td></tr></tbody>
