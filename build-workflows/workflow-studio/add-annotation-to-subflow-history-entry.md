# Add an annotation to a subflow history entry

Add a text description to a subflow's flow history entry. Summarize the purpose or contents of a particular flow history entry.

## Before you begin

Role required: flow\_designer or admin

## About this task

Annotations are only visible from the sidebar History view.

## Procedure

1.  Navigate to **All** &gt; **Process Automation** &gt; **Workflow Studio**.

2.  On the homepage, select **Subflows**.

3.  Open the subflow whose flow history entries you want to annotate.

4.  Select **Edit subflow**.

5.  From the sidebar view, select the **History** icon.

6.  From the list of flow history entries, select the entry that you want to annotate.

7.  From More actions menu, select **Add annotation**.

    ![Add annotation menu item](../images/subflow-history-add-annotation-01.png)

8.  In the text box, enter the text of the annotation.

    ![Sample annotation reading Wait 3 days then close incident](../images/subflow-history-add-annotation-02.png)

9.  Press Enter to save the annotation.

    ![Subflow's flow history entry with annotation](../images/subflow-history-add-annotation-03.png)


## What to do next

You can change the annotation using the **Modify annotation** option.
