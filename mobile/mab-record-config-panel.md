# Configuration panel within a record screen

The configuration panel is the main work area of the Mobile App Builder where you perform the editing of your records.

When you select a record, field types that are specific to the selected record open in the configuration panel. For a list of all the different field types and their usage, see [Record screen input field types](mab-record-screen-input-types.md). Not all fields require input or editing and only the fields marked with an asterisk \(![Asterisk to indicate a mandatory field.](../image/mab-mandatory-star.png)\) are required.
