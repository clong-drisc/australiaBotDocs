# Accessing and posting to the mobile activity stream

Use the activity stream to submit and list work notes, files, images, and videos to a record within your mobile app.

## Before you begin

Role required: none

## About this task

An activity stream enables you to communicate with other users about the work done on a record.

![Activity stream tab and records.](../image/activity-stream-widescreen.png)

## Procedure

1.  Access the activity stream screen segment for a record by opening a record and tapping the **Activity Stream** tab.

    **Note:** The default tab name is activity stream, but your administrator might have changed it to a name like Updates, History, or Notes. The example in the above image uses the name **Updates** for the activity stream.

2.  Tap an icon displayed within the ribbon in the record title bar to post items to the activity stream.
