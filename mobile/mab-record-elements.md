# UI elements of the record screen

Familiarize yourself with the various elements contained within a record screen.

<table id="table_ebd_ktg_5qb"><thead><tr><th>

UI element

</th><th>

Function

</th></tr></thead><tbody><tr><td>

![Mobile App Builder home icon](../image/mab-button-home.png)

</td><td>

The home icon that you can use at any time to return to the categories screen.

</td></tr><tr><td>

![Mobile App Builder record indicator.](../image/mab-indicator-record.png)

</td><td>

A record indicator that displays the top-level record you are working in.

</td></tr><tr><td>

![Mobile App Builder application scope indicator.](../image/mab-application-scope-indicator.png)

</td><td>

An application scope indicator that displays the application scope you are working in.

</td></tr><tr><td>

![Mobile App Builder icon to access documentation](../image/mab-button-access-doc.png)

</td><td>

A quick-reference icon that opens a menu of help options:-   **Record configuration tour** reopens available self help tour modules
-   **Mobile App Builder overview** reopens an overview tour module
-   **App preview instructions** opens a modal to help find and download the ServiceNow mobile app for testing purposes
-   **Get started guide** opens, in a separate tab, the ServiceNow get started website
-   **Go to product documentation** opens, in a separate tab, the ServiceNow documentation website
-   **Version** lists the current installed version of Mobile App Builder

</td></tr><tr><td>

![Mobile App Builder save button highlighted.](../image/mab-button-save-bold.png)

</td><td>

The **Save** button is highlighted to show if there are any unsaved record changes anywhere in the record hierarchy. If no record changes exist, the button is inactive \(![Mobile App Builder save button not highlighted.](../image/mab-button-save-not-bold.png)\).

</td></tr><tr><td>

![Mobile App Builder more icon.](../image/mab-button-more.png)

</td><td>

The more icon provides the following options:-   The table name of the selected record, for example, Record screen \[sys\_sg\_form\_screen\].
-   **Delete record**: Select this option to delete the displayed record.
-   **Open in platform**: Opens the record you are working on in the web-based UI.

 **Note:** All screens in the Mobile App Builder are exact reflections of screens in the web-based UI.

</td></tr></tbody>
</table>
