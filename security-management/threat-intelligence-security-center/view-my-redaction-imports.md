# View my Redaction Imports

Use this section to view import records created by the currently logged-in user.

## Before you begin

Role required: sn\_sec\_tisc.admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Threat Intelligence Security Center** &gt; **Imports**.

2.  Select **Redaction Imports** &gt; **My Imports**

    This page displays all the records those import records which are categorised for further process for the import job.

3.  Open any record to view the status of the import process.

4.  View the summary of the import records that are imported by the logged in user.
