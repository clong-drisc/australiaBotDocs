# Linking Canvas from a Case

Use this section to link a canvas from a case.

## Before you begin

Role required: sn\_sec\_tisc.analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Threat Intelligence Security Center**.

2.  Click the **Threat Analyst Workbench** icon.

3.  Go to **Investigation Canvases** &gt; **All Canvases**.

4.  From the **All Cases** list view, select a case which is not linked to the canvas.

5.  Go to **Investigation Canvas** section.

6.  Select **Link Case** from the drop down list on the empty canvas page \(the option is shown in the following screenshot\).

    ![Link Canvas from a Case](../image/tisc-link-canvas-from-case.png)

7.  Select the canvas.

8.  Select the **Link** button to link the canvas to a case.

    An information message is displayed indicating that the canvas data is refreshed successfully.
