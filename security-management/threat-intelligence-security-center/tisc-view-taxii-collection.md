# Viewing TAXII Collection Records

View the records that are added to the TAXII collections.

## Before you begin

Role required:

-   sn\_sec\_tisc.admin
-   sn\_sec\_tisc.analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Threat Intelligence Security Center** &gt; **Administration** &gt; **TAXII Outbound Server**.

2.  Select **TAXII Collections**.

3.  Go to **TAXII Collection Records** section after you save the new TAXII collection.

    All the records added to the TAXII collection are displayed under this section.

4.  Select **Remove** button to remove any records from TAXII collection.

    A user with sn\_sec\_tisc.analyst role can only be able to remove records from TAXII collection.

5.  Select the records to be removed and select **Remove**.
