# Viewing Outbound Intelligence

Use this section to view all outbound intelligence sharing records. Review the intelligence data and take necessary actions to approve or reject them.

## Before you begin

Role required: sn\_sec\_tisc.analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Threat Intelligence Security Center** &gt; **Threat Intelligence Sharing**.

    The Threat Intelligence Sharing page is displayed with the following intelligence sharing sections that requires the appropriate action.

    -   Outbound intelligence
    -   Inbound intelligence
    -   TAXII Outbound Server
2.  Select **Outbound Intelligence** &gt; **All**

    The outbound intelligence page displays the records that are in the process for sharing intelligence. It displays the following sub sections:

    -   **All**: Displays all the outbound intelligence sharing records.
    -   **Created By Me**: Displays the outbound intelligence sharing records that are created by you.
    -   **Pending Approvals**: Displays the outbound intelligence records that are pending for approvals.
    -   **My Approvals**: Displays the outbound intelligence records that are awaiting your approval.
3.  Open any record to view the status of the sharing record.

    Verify the sharing intelligence records that are awaiting for approval or rejection.

4.  Select **Review** section to review the outbound intelligence data.

5.  Verify the published jobs associated to the outbound intelligence data.

6.  Select **Approve** to approve the sharing record.

    The Approve outbound Intelligence is displayed.

7.  Add any comments, if necessary.

8.  Select **Approve** button to approve the sharing intelligence record.

    A confirmation message is displayed indicating that the outbound intelligence record is approved.

9.  Additionally, select **Reject** button if you need to reject the outbound sharing intelligence.
