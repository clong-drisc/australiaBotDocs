# Email logs

This section provides a clear visibility to the TISC administrators on the emails that are sent out using the configured email notification rules.

## Email Notifications &amp; Logs purview

TISC provides different purview to access, view, or modify both email notifications and logs based on different persona.

**Note:**

<table id="table_rv1_chz_dbc"><thead><tr><th>

Persona

</th><th>

Access levels

</th></tr></thead><tbody><tr><td>

TISC admin \(sn\_sec\_tisc.admin\)

</td><td>

As an sn\_sec\_tisc.admin, you can only view the email notifications that are configured for TISC.​**Note:** Click on the email logs hyperlinks to navigate to the classic UI to view these notifications.

</td></tr><tr><td>

System administrator \(admin\)

</td><td>

As an admin, you can create, view, and edit the notifications in the classic UI.**Note:** Click on the email logs hyperlinks to navigate to the classic UI to view and create these notifications.

</td></tr></tbody>
</table>**Note:** Both TISC admin and system admin can use the filters and search for the email logs.
