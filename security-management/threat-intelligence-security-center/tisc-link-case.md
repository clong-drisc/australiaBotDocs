# Creating a Case and Linking from Investigation Canvas

Use this section to create and link a case\(s\) from an investigation canvas.

## Before you begin

Role required: sn\_sec\_tisc.analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Threat Intelligence Security Center**.

2.  Select **Threat Analyst Workbench** icon.

3.  Go to **Case Management** &gt; **All Cases**.

    This displays all the cases.

4.  Open any case record from the list.

5.  Select **Link Case** from the **Details** section.

    The Create and Link Case dialogue box appears.

6.  Select a case from the list to associate the case to an investigation canvas.

7.  Select **Create New Canvas**.

    The Create New Case dialogue box appears.

8.  Fill in the form fields, as appropriate.
