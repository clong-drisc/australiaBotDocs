# View Intelligence Reports

View intelligence reports.

## Before you begin

Role required: sn\_sec\_tisc\_analyst

## Procedure

1.  Navigate to **Workspaces** &gt; **Threat Intelligence Security Center** &gt; **Threat Intel Library**.

2.  Select **Reports** &gt; **Intelligence Reports**

    This page displays the list of intelligence reports.

3.  Select any report.

4.  Select **Download** to download the report.

5.  Select **Share** to share the report in email format.

6.  Enter the recipient and select **Send**.

    **Note:** You can also preview the report before sharing.
