# Security Operations - Isolate Host Flow

The **Security Operations - Isolate Host** flow is a high-level flow independent of integrations. It uses the configured queries to search for a set of configuration items. Use it to fulfill an integration, such as Carbon Black.

## Before you begin

Role required: sn\_si.analyst

## About this task

This flow is triggered from the **Configuration Items** tab on a security incident related list.

![Flow Designer for Security Operations - Isolate Host](../image/isolate-host-flow.png "Isolate Host")

Activities specific to this flow are described here. For more information on other activities, see [Common Security Operations integration flows and orchestration activities](../../security-operations-common/concept/common-wf-activities.md).
