# Home page in the IT Remediation Workspace

The Home page in the IT Remediation Workspace provides a consolidated list of host, application, container and test result remediation tasks that are assigned to you and your assignment groups.

-   sn\_vul.remediation\_owner for host vulnerable items \(VITs\)
-   sn\_vul.app\_security\_champion for application vulnerable items \(AVITs\)
-   sn\_vul\_container.remediation\_owner for container vulnerable items \(CVITs\)
-   sn\_vulc.remediation\_owner for configuration test results \(CTRs\)



The Home page consists of the following cards and list of remediation tasks assigned to you and your assignment groups, records assigned to you have solutions.

-   **Assigned remediation tasks card**

    Number of active remediation tasks assigned to you. Select the tile to view the list of remediation tasks.

-   **Preferred solutions on VIs card**

    Number of preferred solutions available to remediate the host vulnerable items. The Solutions list is displayed if the Vulnerability Solution Management application is installed. This card shows you solution data, potential and preferred solutions, from third-party vendors about your vulnerable items.

-   **Impacted CIs card**

    The number of configuration items that have VIs that are assigned to you or your groups. For more information, see [View configuration items with vulnerabilities in the IT Remediation Workspace](../../vulnerability-response/task/vr-ws-remed-task-v16.md).

-   **Preferred patches on VIs card**

    Number of preferred patches available that can be used to remediate the host vulnerable items.

-   **Remediation tasks list**

    The list of remediation tasks assigned to you and your assignment groups. You can filter the records by selecting the **Assigned to me** and **Assigned to my groups** options.

    You can click a remediation task to view the details and start an investigation. In the Lists view, you can view the Remediation Tasks, Vulnerable Items, Configurations Test Results and so on. You can open a remediation task and click **Assign to me** to assign the task to yourself and start investigation.

    **Note:** If you do not assign the task to yourself in the remediation task, you cannot view any UI actions at the record level.

    Click the More Actions ellipses and click **Start Investigation** to resolve a remediation task.

    For more information about what you can do on the home page, see [Use remediation task records in the IT Remediation Workspace](../../vulnerability-response/task/vr-ws-remed-task.md).
