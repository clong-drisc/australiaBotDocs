# Identify observable sources

If an observable has no sources defined, it uses all types of sources. However, if you add one or more threat sources to an observable, it limits the sources used.

## Before you begin

Role required: sn\_ti.admin

## Procedure

1.  Navigate to **All** &gt; **Threat Intelligence** &gt; **IoC Repository** &gt; **Observables**.

2.  Click the observable to which you want to add observable sources.

3.  Click the **Observable Sources** related list.

4.  Click **Edit**.

5.  As needed, use the filters to locate the source you want to associate with the observable.

6.  Using the slushbucket, add the observable source to the **Observable Sources** list.

7.  Click **Save**.
