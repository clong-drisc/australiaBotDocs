# Search for security artifacts

You can perform a keyword search on any security artifact list.

## Before you begin

The Threat Intelligence plugin must be activated to use Security Case Management.

Role required: sn\_ti.case\_user\_write

## Procedure

1.  Open a case that contains security artifacts.

2.  Click the tab for the security artifacts you want to perform a keyword search on.

3.  In the **Keyword search** box, begin to type the name of the artifact you want to locate.

    When artifacts that match your entry are found, they are displayed.
