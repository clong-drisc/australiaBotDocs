# Mute participants in a conference call

As an MSI manager or a conference call host, you have the option to mute a particular participant or all the participants on the conference call.

## Before you begin

Role required: sn\_msi.workspace\_manager

Before starting this procedure, verify that there’s an active conference call with one or more participants.

## Procedure

1.  Navigate to **Workspaces** &gt; **Major Security Incident Management**.

2.  From the Major Security Incident list, select the major security incident that you're trying to resolve on the conference call.

3.  Select the ![Conference call](../image/msim-conf-call-icon.png) icon on the right-context menu to view the details of the conference calls.

4.  Select **Join call** on the active conference bridge.

5.  After joining the conference call, select a participant that you want to mute and select **Mute** on the conference bridge.

6.  To mute all participants on the conference call, select **Participants** in the meeting controls and select **Mute All**.
