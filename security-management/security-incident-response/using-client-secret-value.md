# Using Client Secret value

Use client secret value to configure the Graph application registry record.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **App registrations** in Azure AD portal.

2.  Click **Certificates &amp; secrets** from the **Manage** section.

3.  Click **New client secret**.

4.  Copy and make a note of the created client secret from the **Value** field.
