# Create shifts through Security Incident Response Workspace

Create shifts through the Security Incident Response Workspace to use when configuring shift handover records.

## Before you begin

Role required:

-   sn\_escm\_sh.shift\_admin
-   sn\_si.admin, which includes the sn\_escm\_sh.shift\_admin role

## Procedure

1.  Navigate to **Workspaces** &gt; **Security Incident Response Workspace**.

2.  Select **Administration**.

3.  Select **Shift Handover** and then select **Shift Time Configurations**.

4.  Select **New**.

5.  In the **Shift** field, enter a name for the shift.

6.  In the **Start Time** and **End Time** fields, provide the starting and ending times for the shift.

    **Note:** The time values are in your configured timezone.

7.  Select **Save**.


## Result

The new shift is added on the Shift Time Configurations page.
