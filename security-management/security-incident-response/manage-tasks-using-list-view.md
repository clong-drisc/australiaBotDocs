# Manage tasks using the List view

Track and manage all the tasks associated with a major security incident using the List view.

The List view displays the details of all the tasks associated with the major security incident like Task number, priority, current state, assigned to, assigned group. Using these list categories and filters, you can quickly find and track the required tasks that you need to work on in a major security incident.

## Create a task using the List view

Role required: sn\_msi.workspace\_manager.

MSI Managers can create a task in the List view using the following steps:

1.  Navigate to **Workspaces** &gt; **Major Security Incident Management** &gt; **Major Security Incidents**.
2.  Select the **Lists** view.
3.  Choose the required major security incident record.
4.  Select the **Tasks** tab.

    ![Create a task using the List view](../image/msim-tasks-create-list-view.png "Create a task using the List view")

5.  Select **List view**, and choose **New task**.
6.  On the form, fill in the fields.

MSI Managers can view the details of a task and activity information from the MSI Responders, and modify a task in the List view using the following steps

1.  Navigate to **Workspaces** &gt; **Major Security Incident Management** &gt; **Major Security Incidents**.
2.  Select the **Lists** view.
3.  Choose the required major security incident record.
4.  Select the **Tasks** tab, and choose **List view**.
5.  Choose the desired Task on the list view to view the details of the task.

    2.  Select the **Settings** icon, and use the **Edit columns** option to modify the list of columns that appear on the List view. Use the **Reset widths** option to reset the widths of the columns.

## Delete a task using the List view

Role required: sn\_msi.workspace\_manager.

MSI Manager can delete a task in the List view using following steps:

1.  Navigate to **Workspaces** &gt; **Major Security Incident Management** &gt; **Major Security Incidents**.
2.  Select the **Lists** view.
3.  Choose the required major security incident record.
4.  Select the **Tasks** tab.
5.  Select **List view**.
6.  Choose a Task that you want to delete from the list.
7.  In the Major security incident task pop-up, select the **Delete** option.

    The Delete task pop-up appears requesting you to confirm the deletion of the task.

8.  Select **Delete**.
