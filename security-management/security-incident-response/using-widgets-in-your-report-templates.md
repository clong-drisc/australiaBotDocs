# Use Visualizations in Report Templates

Use report widgets to define and include User Interface Builder \(UIB\) elements such as data visualizations in the MSIM status reports.

## Before you begin

Role required: sn\_msi.workspace\_manager

## Procedure

1.  Navigate to **All** &gt; **Major Security Incident Management** &gt; **MSI Administration** &gt; **Status Report Setup**.

    The MSI Status Report Setup page displays.

2.  Drill down to **Report Template** &gt; **Visualizations**.

3.  Select **Configure**.

    The following are the predefined UIB Visualizations as a part of the base system, which you can use to configure and modify your status reports.

    |Widget name|Description|
    |-----------|-----------|
    |**Active Team**|Displays the list of active teams involved in resolving the MSI.|
    |**Affected Assets**|Displays the list of affected assets.|
    |**Affected Locations**|Displays the list of affected locations.|
    |**Affected Users**|Displays the list of affected users.|
    |**Collaboration Pie Graph**|Includes the collaboration activities in the Pie graph.|
    |**Collaboration Trend Graph**|Displays the collaboration activities as a trend graph.|
    |**Estimated resolution date**|Displays the estimated resolution date by when the MSI is estimated to be resolved.|
    |**Linked Incidents Bar Graph**|Displays the linked incidents bar graph.|
    |**Linked Incidents Trend Graph**|Displays the linked incidents trend graph.|
    |**MSI Response Tasks Bar Graph**|Displays the MSI Response tasks graph.|
    |**Response Tasks Bar Graph**|Displays the Response tasks bar graph.|
    |**Response Tasks in Progress by Labels**|Displays the response tasks in progress by labels.|
    |**Response Tasks Overdue**|Displays the total response tasks that are overdue for closure.|
    |**Response Tasks Trend Graph**|Displays the response tasks trend graph.|
    |**Time elapsed since incident detection**|Displays the time elapsed from the time the incident is detected.|
    |**Timeline**|Displays the timeline of key events and milestones that have occurred as part of the major security incident.|
