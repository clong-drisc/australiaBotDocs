# Submit workplace contract for review in Core UI

You can send a contract that is in Draft state for approval.

## Before you begin

Role required: sn\_wsd\_la.manager or sn\_wsd\_la.admin

## Procedure

1.  Navigate to **All** &gt; **Workplace Lease Administration** &gt; **Workplace Contracts**.

2.  Select a contract in Draft state.

3.  In the **Approver** field, select an approver for the contract.

4.  Select **Submit For Review**.


## Result

An email message is sent to the selected approver and the contract substate changes to Under Review.
