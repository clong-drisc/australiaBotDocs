# Cancel workplace contract in Core UI

Cancel an active contract if it is no longer relevant.

## Before you begin

Role required: sn\_wsd\_la.manager or sn\_wsd\_la.admin

## Procedure

1.  Navigate to **All** &gt; **Workplace Lease Administration** &gt; **Workplace Contracts**.

2.  Select a contract.

3.  Select **Cancel Contract**.

4.  Click **Yes** to confirm contract cancellation.


## Result

The contract state changes to Canceled.
