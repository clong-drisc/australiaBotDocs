# Host portal

Use the host portal to manage visits and visitors.

The Host Portal enables employees to manage visits. As a host, you can create, view, and manage visits from the portal. The portal displays a list of visits and a detailed view with visitor information, response status, and pre-check task completion.

As a host, you can use the portal for the following:

-   Create single-occurrence or recurring visits
-   Specify the visit purpose, location, date and time, and an optional co-host
-   Save a visit as a draft and resume editing later
-   Add individual visitors or upload a CSV or an Excel file with details of multiple visitors
-   Remind visitors to respond, edit visitor details, or remove a visitor from a visit
-   Edit a single occurrence or an entire series of recurring visits
