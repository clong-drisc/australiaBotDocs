# New visit form

Use the New visit form to select a purpose, location, date and time for your visit. Additionally, add visitors and optional co-hosts

<table id="table_wky_r4w_n3c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Visit description

</td><td>

Description with details about the purpose of the visit.

</td></tr><tr><td>

Location

</td><td>

Location of the visit.

</td></tr><tr><td>

Host

</td><td>

Name of the host of the visitor. By default, the field is set to your name.

</td></tr><tr><td>

Co-hosts

</td><td>

Co-hosts for the visit. Co-hosts have the visit displayed on their host portal and can edit the visit or manage visitors.

</td></tr><tr><td>

From

</td><td>

Start date and time for the visit.

</td></tr><tr><td>

To

</td><td>

End time and date for the visit.

</td></tr><tr><td>

All-day invitation

</td><td>

Option for visits that last for an entire day.

</td></tr><tr><td>

Recurring invitation

</td><td>

Option to make a recurring invitation for the visitor.For recurring invitations, you can configure the following fields:

-   Repeat: Frequency of the recurring invitation. It can be **Daily**, **Weekly**, or **Monthly**.
-   Lasts: How long the recurring invitation lasts. You can the following values.
    -   For: Specifies how long the invitation lasts based on the **Number of days** field.
    -   Until: Specifies when the invitation expires based on the **Ends on this date** field.
-   Select days of the week: Days the visitors will arrive on. This field is only applicable for the **Weekly** repeat.

</td></tr><tr><td>

Private note to receptionist

</td><td>

Additional requirements or a private note to the receptionist.
