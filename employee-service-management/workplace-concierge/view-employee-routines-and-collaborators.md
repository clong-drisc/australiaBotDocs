# View employee presence modules

Track the routines and exceptionscreated by employees and the collaborators added by employees.

## Before you begin

Role required: sn\_wsd\_concierge.admin

## About this task

Using the Employee Presence module, you can perform the following:

-   Add or view routine details such as the workplace profile associated with the routine, routine start and end time, and the in-office days set in the routine.
-   Add or view the presence exceptions for employee in-office days or remote days.
-   Add or view collaborators added by employees.

## Procedure

1.  Navigate to **All** &gt; **Employee Presence**.

2.  View employee routines by selecting the Routines module.

    A list of routines created in the application is displayed.

    **Note:** The list contains a filter that hides inactive routines.

3.  View the routine exceptions by selecting the Exceptions module.

    A list of exceptions created by employees is displayed.

    **Note:** Records that are older than six months are removed from the list.

4.  View the collaborators of employees by selecting the Collaborators module.

    A list of your collaborators is displayed.
