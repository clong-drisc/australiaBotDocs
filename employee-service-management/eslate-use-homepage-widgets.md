# Use homepage widgets

Access tasks, announcements, and popular content through homepage widgets that provide quick navigation to detailed views and actions.

## Before you begin

Role required: none

## About this task

Homepage widgets provide immediate access to your most important work items. Each widget offers different interaction options for accessing detailed information and completing actions.

## Procedure

1.  Access Employee Slate homepage as your starting point.

    The homepage displays AI-powered chat and search at the top, followed by five default widgets.

    You see Inbox, Announcements, Popular Content, Upcoming Holiday, and Quick Links widgets.

2.  Review items in the Inbox widget to handle approvals and tasks.

    1.  Select an approval item to view detailed information.

        The system redirects you to the detailed view where you can review and take necessary action.

    2.  Select the widget button to view all inbox items in the full-page listing view.

3.  Browse the Announcements widget carousel to view company communications.

    1.  Select an announcement to view specific content details.

    2.  Navigate through the carousel to see all promoted content.

4.  Explore the Popular Content widget to find popular resources.

    This widget shows the most viewed knowledge articles and most requested catalog items within a set timeframe.

    1.  Select a knowledge article to access the full content.

    2.  Select a catalog item to start a request process.

5.  Use the Quick Links widget to access frequently used resources.

    1.  Select a quick link to navigate directly to the destination.

        The system takes you to the linked knowledge article, catalog item, or external resource.

    2.  Complete any required action at the destination.

6.  Use the navigation controls to return to the homepage from any detailed view.

    Each detailed view provides navigation options to return to the homepage.


## Result

You can efficiently access and complete daily work tasks through the homepage widgets without navigating through multiple menus or pages. The widgets provide proactive notifications and quick access to the most relevant information for your role.
