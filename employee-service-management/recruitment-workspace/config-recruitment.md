# Configuring Recruitment Workspace

Set up the Recruitment workspace to provide recruiters with an integrated workspace and a collaborative communication channel.

-   **[Install Recruitment workspace](../task/install-recruitment-workspace.md)**  
You can install the Recruitment workspace application \(sn\_ta\_recruiter\) if you have the admin role. The application includes demo data and installs related ServiceNow® Store applications and plugins if they are not already installed.
-   **[Activate the performance analytics jobs in Hiring Experiences](../task/activate-pa-jobs.md)**  
Activate the performance analytics \(PA\) jobs to create an uninterrupted flow of job requisition updates by their states in the Recruitment workspace.
-   **[Create assignment groups](create-assignment-groups-ta.md)**  
Create assignment groups to allocate job requisitions efficiently among the recruiters.
-   **[Configure the fields to be duplicated in a job requisition](config-dplcte-fields.md)**  
Configure which fields are duplicated when a recruiter duplicates a job requisitions.
-   **[Enable recruiters to view email details in the activity stream](../task/enable-emails-recruiters.md)**  
Add the recruiter role to a system property. The recruiter can view the details of the email in the activity stream of the Recruitment workspace.
-   **[Configure hiring task types](../task/config-task-type.md)**  
Configure different types of hiring tasks in the Recruitment workspace for efficient task management.
-   **[Map task types in Recruitment workspace](../task/config-task-mapping.md)**  
Map tasks with its corresponding task types in the Universal Task to select and assign in the Recruitment workspace.
-   **[Create job description templates](create-jd-template.md)**  
Create job description templates to make the new requisition requesting process more streamlined.
-   **[Configure approval workflows](../task/set-up-approval-ta.md)**  
Customize an approval workflow according to your organizational requirements.
-   **[Customize job requisition state management workflow](../task/customize-job-req-state-mgmnt.md)**  
Add new job requisition states or customize the job requisition state management workflow according to your organizational needs.
-   **[Configure application state](../task/configure-app-state.md)**  
Add new job application states according to your organizational needs.
-   **[Modify the re-approval trigger](../task/modify-approval-trigger.md)**  
Modify which fields can trigger an approval request if it is updated.
