# Close a job requisition

Close a job requisition if it’s no longer required.

## Before you begin

Role required: sn\_ta\_hiring\_core.recruiter

## Procedure

1.  Navigate to **Workspaces** &gt; **Recruitment workspace**.

2.  Select a requisition that you want to close.

3.  From the ellipses at the top-right corner, select **Close requisition**.

4.  In the pop-up window, select **Cancel requisition** and select **Close**.

    If a requisition is closed while it was on hold, the on hold state won't be retained. For more information, see [Job requisition state management](../reference/job-req-state-mngmnt.md).
