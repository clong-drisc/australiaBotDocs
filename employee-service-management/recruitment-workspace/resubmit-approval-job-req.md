# Resend a job requisition for approval

Resend a job requisition for approval if last-minute changes or technical faults.

## Before you begin

Role required: sn\_ta\_hiring\_core.recruiter

## Procedure

1.  Navigate to **Workspaces** &gt; **Recruiter workspace**.

2.  Open a job requisition that you want to resubmit for approval.

3.  Select **Resend for approval**.

    -   The requisition which was in **Ready** state before goes back to **Pending approval** state.
    -   For more information on the state management system, see [Job requisition state management](../reference/job-req-state-mngmnt.md).

## Result

The approval request is resend to the assigned approver.

On the **Approvers** tab, the state of the approval request changes to Requested.
