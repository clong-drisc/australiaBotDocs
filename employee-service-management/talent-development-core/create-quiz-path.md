# Add exams to a learning path from Career Assessment

Create internal learning content to add exams as content items within a learning path from Career Assessment.

## Before you begin

Role required: sn\_lc.catalog\_manager

## Procedure

1.  Navigate to **All** &gt; **Learning** &gt; **Content** &gt; **Internal Content**.

2.  Select **New**.

3.  On the form, fill in the fields.

    For a description of the field values, see [Learning internal content form](learning-internal-content-form-ca.md).

4.  Select **Submit**.
