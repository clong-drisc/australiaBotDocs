# Create exam catalogs

Create exam catalogs to be able to segregate exams based on category.

## Before you begin

Role required: sn\_exam.catalog\_admin

## Procedure

1.  Navigate to **All** &gt; **Career Assessment** &gt; **Exam Catalogs** &gt; **Create New**.

2.  On the form, fill in the fields.

    |Field|Description|
    |-----|-----------|
    |Name|Name of the exam catalog.|
    |Admins|Option to select administrators for the exam catalog.|
    |Owner|Person who owns the exam catalog.|
    |Description|Description of the contents of the exam catalog.|

3.  Select **Submit**.


-   **[Define user access for an exam catalog](define-exam-catalog-user-access.md)**  
After creating an exam catalog, define user access criteria at the exam catalog level.
-   **[Add exams to a catalog](assign-catalog-exams.md)**  
Add exams to an exam catalog in Career Assessment.
