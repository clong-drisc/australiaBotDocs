# Talent Development Core application reference

Administrators use reference information to configure and maintain applications and features.

-   **[Talent Growth Plan reference](../reference/egd-growth-plan-ref.md)**  
Employee growth plans are created by administrators and used by managers and employees to help facilitate and track growth plans.
-   **[Talent Outcome Types reference](../reference/egd-outcome-types-form.md)**  
Outcome types are created by administrators.
-   **[Talent Properties reference](../reference/egd-properties-reference.md)**  
Properties control how skills approvals are managed and whether Skills Foundation is used.
-   **[Talent Activity Types reference](../reference/egd-activity-type-form.md)**  
Administrators create activity types that are used by managers and employees.
-   **[Talent Activity Category reference](../reference/egd-activity-category-form.md)**  
Administrators create activity categories that are used by managers and employees.
-   **[Talent Aspirations reference](../reference/egd-aspirations-form.md)**  
Administrators manage aspirations used by managers and employees.
-   **[Talent Feature growth item reference](../reference/egd-featured-growth-item.md)**  
Administrators create featured growth items that are used by managers and employees.
-   **[Talent Activity template reference](../reference/egd-activity-template-form.md)**  
Activity templates are used by managers and individuals to create activities in a growth plan. Administrators create activity templates.
-   **[Talent Sample texts reference](../reference/egd-sample-text-form.md)**  
Sample text is content that is used in different types of content in a selected document. Administrators create sample text templates.
-   **[Talent Aspiration types form](../reference/egd-aspiration-type-form.md)**  
Administrators create aspiration types that are use by employees and managers.
-   **[Talent Activity Template Recommendation form](../reference/egd-activity-recommendation-form.md)**  
Administrators create activity template recommendation templates that employees and managers use to create career growth plans.
-   **[Talent Interest form](../reference/egd-interests-form.md)**  
Administrators create interests that employees and managers use to create career growth plans.
-   **[Components installed with Talent Development Core](../reference/egd-components-roles-tables.md)**  
Several types of components are installed with Talent Development Core, including roles and tables.
-   **[Employee Application Access Criteria reference](../reference/egd-empl-profile-app-access-criteria-form.md)**  
Talent Development uses the **Employee licensing** feature in the Employee Profile app to license users.
