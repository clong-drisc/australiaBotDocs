# Edit skills and proficiencies in the Skills Matrix

Edit skills data and their proficiency levels to manage the progression at each role level in a role group using the Skills Matrix in the Skills Workspace.

## Before you begin

Role required: sn\_skills\_int.admin

## Procedure

1.  Navigate to **All** &gt; **Skills Workspace**.

2.  Select the Lists icon \(![Lists icon](../../../common/image/icon-list.png)\).

3.  Go to **Job architecture** &gt; **Role groups**.

4.  Select a role group for which you want to edit the skills data.

5.  Select the**Skills Matrix** tab.

6.  Select the check box next to the skills to be edited.

7.  Select the **Edit** button.

8.  Insert proficiency levels for the skill from the Skills \[cmn\_skills\] table by selecting **Autofill proficiency levels**.

    -   To update only missing proficiency levels, select **Update only empty fields**
    -   To update all proficiency levels and overwrite any existing information, select **Update and replace all the existing fields**
9.  Modify the mandatory or optional setting of a skill by selecting **Update required status** and selecting the appropriate value.

10. Remove the selected skill or skills by selecting **Remove skills**.

11. Select **Save**.
