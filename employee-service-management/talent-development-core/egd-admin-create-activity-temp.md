# Create Talent Development Activity Templates

As a Talent Development Core admin you create **Activity Templates** that are used in growth plans.

## Before you begin

Role required: sn\_egd\_core.admin

## About this task

An **Activity Template** logically groups activities by name and requires an **Activity type**.

## Procedure

1.  Navigate to **All** &gt; **Talent Development Core** &gt; **Activity Templates**.

2.  Select **New**.

3.  Complete the **Activity Templates** form.

    Read details about the different fields in [Talent Activity template reference](../reference/egd-activity-template-form.md).

4.  Select **Submit**
