# Career Assessment

The Career Assessment application works as an exam engine that provides the ability to create exams that help with career advancement and help test a learner's knowledge of a subject.

Career Assessment is a test taking application that enhances the learning experience of users in Career Assessment. Career Assessment offers a unique user experience. In the back-end, learning administrators create exams and learning paths using Career Assessment to create customized learning journeys for users.
