# Create an exam

Create an exam on Career Assessment.

## Before you begin

Role required: sn\_exam.catalog\_admin

Users configured as part of the admin list, can contribute user criteria or owner of the associated catalog can create exams.

## Procedure

1.  Navigate to **All** &gt; **Career Assessment** &gt; **Exams** &gt; **Create New**.

2.  On the form, fill in the fields.

    For a description of the field values, see [Exam form](../reference/exam-form.md).

3.  Select **Submit**.


## Result

The exam is created in Draft state.

**Note:** The exam may only be published when it contains at least one question and two options with the required number of options marked as correct.
