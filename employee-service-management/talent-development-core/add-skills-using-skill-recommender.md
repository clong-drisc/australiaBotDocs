# Add skills by using the skill recommender

Search for the skills that you're interested in developing or improving and then add them to your skills profile by using the skills recommender in the Skills Foundation application.

## Before you begin

Role required: sn\_skills\_int.emp

## About this task

By adding skills to your profile, you can track all these details in one place.

## Procedure

1.  Navigate to **All** &gt; **Employee Center**.

2.  In the Employee Center, navigate to **Profile** &gt; **Career**.

3.  On the skills widget, select the **Add skill** button.

4.  In the search bar, search for the name of the skill that you want to add as shown in the following example.

    Search results are sorted alphabetically, with the top results displaying first.

    ![Skill search results](../image/skills-recommender.png)

    **Note:** If you don't see the skill recommendations, it indicates that the AI Search setup is broken. Clear your instance cache by entering `server_url/cache.do` in your browser. If the problem persists, then contact your system admin.

5.  From the list of recommendations, select the skill that you want to add to your profile.


## Result

The skill is added to your additional skills section.
