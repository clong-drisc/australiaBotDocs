# Create a learning library

Create a learning library that includes learning content from internal and external content repositories.

## Before you begin

Role required: sn\_lc.catalog\_manager, sn\_lc.catalog\_group\_manager

## Procedure

1.  Navigate to **All** &gt; **Learning** &gt; **Content** &gt; **Learning Libraries**.

2.  Select **New**.

3.  On the Learning library form, fill in the fields.

    For a description of the field values, see [Learning library form](../reference/learning-library-form_0.md).

4.  Select **Submit**.


-   **[Add course items to learning library](add-learning-library-course-items.md)**  
To add course items to the catalog
-   **[Add users to learning library](add-learning-library-users.md)**  
You can add applicable users to a learning library.
