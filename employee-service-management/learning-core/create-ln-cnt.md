# Create internal learning content

Create internal learning courses so that employees can learn new content and enhance their skills set.

## Before you begin

Role required: sn\_lc.catalog\_manager

## Procedure

1.  Navigate to **Learning** &gt; **Content** &gt; **Learning Content**.

2.  Click **New**.

3.  On the form, fill in the fields.

    For a description of the field values, see [Learning internal content form](../reference/learning-internal-content-form.md).

4.  Click **Submit**.
