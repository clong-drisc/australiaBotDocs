# Learning task form

Assign a learning task to an employee that can be completed from the To-dos page in Employee Center.

|Field|Description|
|-----|-----------|
|Number|Unique identifier of the learning task.|
|Assigned to|User to whom you want to assign the learning task.|
|Learning Library|Name of the learning library that is assigned via the learning task.|
|Course Item|Name of the course item that is assigned via the learning task.|
|Optional|Option to make the learning task non-mandatory.|
|State|Current state of the task.|
|Due date|Date by which the assignee must complete the learning task.|
|Opened by|Name of the user who has created the learning task.|
|Short description|Additional details of the learning task.|
