# Create a learning task

Assign a learning task to an employee that can be completed from the To-dos page in Employee Center.

## Before you begin

Role required: sn\_lc.task\_creator

## Procedure

1.  Navigate to **All** &gt; **Learning** &gt; **Tasks** &gt; **Learning Tasks**.

2.  Select **New**.

3.  On the Learning task form, fill in the fields.

    For a description of the field values, see [Learning task form](../reference/learning-task-form_0.md).

4.  Select **Submit**.
