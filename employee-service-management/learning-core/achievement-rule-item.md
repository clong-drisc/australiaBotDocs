# Create an achievement rule item

Create an achievement rule item to define if all or a count of items need to be completed to receive an achievement.

## Before you begin

Role required: sn\_lep.achievement\_manager

Create an achievement rule. For more information see, [Create an achievement rule](create-achievement-rule.md).

## Procedure

1.  Navigate to **All** &gt; **Achievements** &gt; **Achievement Rules**.

2.  Open the achievement rule that you want to add an achievement rule item to.

3.  In the achievement rule item section, select **New**.

4.  On the form, fill in the fields.
