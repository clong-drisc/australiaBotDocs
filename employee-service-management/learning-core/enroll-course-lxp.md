# Enroll in a course collection on My Learning

Enroll in a course collection to pursue courses that can enhance your skills.

## Before you begin

Role required: learning.user

## Procedure

1.  Navigate to **All** &gt; **Employee Center** &gt; **My Learning** &gt; **Discover**.

2.  Select a course collection or path that you want to pursue.

    **Note:** The option to enroll is available only within a course collection or a path.

3.  Select **Enroll**.

4.  Select **Start** to start learning.

5.  Select **Skip** within a course collection and continue to the next course.

6.  Toggle the **Show completed** switch to view the courses that you've completed and ignore the ones that you've skipped.

    **Note:**

    -   Although certain courses can be skipped in a course collection or a path, they must be completed or skipped in the order that they appear.
    -   The Show completed toggle switch is available only for content that is skipped within a course collection or path.
