# View external Learning content

View the details of all course items that are pulled from external sources into the Learning application.

## Before you begin

Role required: sb\_lc.learning\_admin

## Procedure

1.  Navigate to **All** &gt; **Learning** &gt; **Content** &gt; **External Content**.

2.  Select the course item that you want to view.

3.  On the Learning External Content form, view the fields.

    For a description of the field values, see [Learning External Content form](../reference/learning-external-content-form.md).


-   **[Define user access to course within external learning content](define-external-content-user-access.md)**  
Define user access to configure who has read access to external learning content.
-   **[Assign topics to external content](assign-external-content-topics.md)**  
Assign topics to external learning content in Learning.
-   **[Add associated skills](add-associated-skills.md)**  
Add associated skills to external learning content in Learning.
