# Access an anonymous report

You can retrieve your anonymous report to submit or view comments and updates from the agents working on your anonymous report. You must have the report key and report number in order to access your anonymous report.

## Before you begin

Role required: none

## Procedure

1.  Navigate to **All** &gt; **Self Service** &gt; **Employee Center**.

2.  From the **Human resources** tab, select **Employee Conduct**.

3.  Select **Visit Anonymous Report Center**.

    You are logged out of the Employee Center and logged into the **Anonymous Report Center** as an anonymous user.

4.  Select **Follow up on a report**.

5.  Enter the **Report Key** and **Report Number**.

    **Note:** This information was provided when you submitted your anonymous report. Without this information, you cannot access it. If you no longer have this information, create a new anonymous report with details on the misconduct or discrimination and indicate that you lost the report key and report number from your first anonymous report.

6.  Enter the text verification code or check the **I'm not a robot** box.

7.  Select **Next**.

8.  Enter questions or comments related to the anonymous report.

9.  Select **Post**.
