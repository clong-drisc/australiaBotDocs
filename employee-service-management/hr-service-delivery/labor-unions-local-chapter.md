# Create a local chapter

Use Local Chapters to set up and track the chapters of your union. Local chapters are children to labor unions.

## Before you begin

Role required: admin

## Procedure

1.  **HR Administration** &gt; **Labor Unions** &gt; **Local Chapters**,

2.  Select **New**.

3.  Fill in the form.

<table id="simpletable_ill_mnk_npb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

The name of the local union chapter.

</td></tr><tr><td>

Labor union

</td><td>

The name of the labor union that the local chapter belongs to.**Note:** For more information, see [Create a labor union](labor-unions-create.md).

</td></tr><tr><td>

Active

</td><td>

Indicates that the union local chapter record is active and available for use.

</td></tr><tr><td>

Union chapter description

</td><td>

A description of the local union chapter.

</td></tr><tr><td>

Local chapter URL

</td><td>

The website URL of the local union chapter.

</td></tr></tbody>
</table>4.  Select **Save** or **Submit**.

    If you select **Save**, the **Labor Union Contacts** and **Employee Union Memberships** tabs appear.

    **Note:** For more information on **Labor Union Contacts**, see [Create union contacts](labor-union-contacts.md). For more information on **Employee Union Memberships**, see [Create employee union memberships](labor-union-employee-membership.md).
