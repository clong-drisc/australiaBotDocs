# Create a conversation form

Use the Create a conversation form to schedule a conversation with an employee when the Microsoft Outlook integration isn’t enabled for Manager Hub.

<table id="table_a3r_m54_4cc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Organizer

</td><td>

Name of the manager scheduling the conversation.

</td></tr><tr><td>

Have this conversation with

</td><td>

Name of the employee with whom you want to converse.

</td></tr><tr><td>

Conversation title

</td><td>

Name of the conversation. You can update this value.

</td></tr><tr><td>

Date

</td><td>

Date of the conversation. The format is &lt;YYYY-MM-DD&gt;. You can use the Show Calendar for Date icon to select a date.

</td></tr><tr><td>

Reoccurs

</td><td>

The frequency at which the conversation occurs.

</td></tr><tr><td>

Send invite

</td><td>

Select **Send invite** to schedule the conversation.The scheduled conversation appears in a new tab. You can update talking points, action items, or any other notes required for the conversation.

</td></tr><tr><td>

Cancel

</td><td>

Close this conversation.
