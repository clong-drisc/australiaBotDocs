# Workday learning todos staging form

Use this form to create Workday learning todos.

|Field|Description|
|-----|-----------|
|Domain|Name of the domain.|
|Description|Brief description of the todo activity.|
|Old assignee|Person who the todo was assigned to previously.|
|Completed by admin|Activities completed by the admin.|
|Integration user ID|Unique identification number of the integration user.|
|URL|URL of the todo.|
|Due date|Date by which the todo is due.|
|Last functionally updated|Date on which the todo was last updated.|
|Previous state|State of the todo prior to the current state.|
|Completed by|Date on which the todo is completed.|
|Created moment|Time at which the todo was created.|
|Integration todo ID|Unique identification number of the todo.|
|State|State of the todo.|
