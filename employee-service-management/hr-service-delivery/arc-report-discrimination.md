# Anonymously report discrimination

Create discrimination cases that can be sent to your Human Resources department without your identity being revealed.

## Before you begin

Role required: none

**Note:** Ensure that the Anonymous Report Center \(com.sn\_anonymous\_report\_center\) and Human Resources Scoped App: Employee Relations \(com.sn\_hr\_employee\_relations\) plugins are activated.

## Procedure

1.  Navigate to **All** &gt; **Self Service** &gt; **Employee Center**.

2.  From the **Human resources** tab, select **Employee Conduct**.

3.  Select **Visit Anonymous Report Center**.

    You are logged out of the Employee Center and logged in to the **Anonymous Report Center** as a guest user.

4.  Select **Submit an Anonymous Report**.

5.  Select **Report discrimination**.

6.  On the form, fill in the fields.

    All information you provide is confidential.

8.  Enter the text verification code or select the **I'm not a robot** check box.

9.  Select **Submit**.

10. Save a PDF copy of your report, which contains the report key and report number necessary to access report updates, progress, or comments, by selecting the **Download report copy** link.

    **Note:** Only HR agents and admins that have access to the Anonymous Report Center and can access reports. For more details, see [Access an anonymous report](arc-access-anon-report.md).
