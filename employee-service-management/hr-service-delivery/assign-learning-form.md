# Assign learning form

Use the Assign learning form to assign a learning course to an employee.

|Field|Description|
|-----|-----------|
|Assign this learning to|Employee to whom you want to assign the learning course.|
|Assigned material|Learning course that you want to assign to employee.|
|Assignment description|Additional information about the learning course.|
|This task is optional|Option to indicate that the learning assignment is optional.|
|Due by|Date by which you want the employee to complete the course.|
