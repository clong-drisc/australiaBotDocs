# Edit a record in HR Service Delivery Integration with Workday Learning

Use this form to create a new record in HR Service Delivery Integration with Workday Learning for multiple scenarios like Sync User Activity, Self Enrollment Activity, and Track Approval.

|Field|Description|
|-----|-----------|
|Name|Name of the activity.|
|Import set table label|Label used for the import set table.|
|Import set table name|Name of the import set table.|
|Type|Defines the type of item being uploaded. If it is a file, JDBC, LDAP, OIDC, Rest, Data stream, or custom|
|Format|The format of the file being uploaded. Formats supported are CSV, custom \(by script\), excel, JSON or XML.|
|Application|The application the data is being uploaded to.|
|File retrieval method|The method in which the date uploaded must be retrieved.|
|CSV delimiter|Labels assigned to delimit the data being uploaded.|
