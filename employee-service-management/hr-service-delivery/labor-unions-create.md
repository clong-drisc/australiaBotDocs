# Create a labor union

Create a union with details that identify it.

## Before you begin

Role required: admin

**Note:** You must create a union before you can create local chapters and union contacts.

## Procedure

1.  Navigate to **All** &gt; **HR Administration** &gt; **Labor Unions** &gt; **Unions**.

2.  Fill in the form.

    |Field|Description|
    |-----|-----------|
    |Union name|The name that identifies the union.|
    |Industry|The industry the union represents.|
    |Active|Indicates that the union record is active and available for use.|
    |Union description|Text that describes the union.|
    |Union URL|The website URL of the union.|

3.  Select **Submit** or **Save**.

    If you select **Save**, the **Local Union Chapters** and **Labor Union Contacts** tabs appear.

    **Note:** For more information on **Local Union Chapters**, see [Create a local chapter](labor-unions-local-chapter.md). For more information on **Labor Union Contacts**, see [Create union contacts](labor-union-contacts.md).
