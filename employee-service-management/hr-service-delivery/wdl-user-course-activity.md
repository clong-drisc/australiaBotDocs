# HR Service Delivery Integration with Workday Learning user course activity staging

Import workday learning user course activity staging information.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Workday Learning** &gt; **System import sets** &gt; **Import set tables** &gt; **Workday learning user course activity staging**.

2.  Select **New**.

3.  On the form, edit the fields.

    For a description of the field values, see [Workday learning user course activity staging form](../reference/wdl-user-course-activity-form.md).

4.  Select **Submit**.
