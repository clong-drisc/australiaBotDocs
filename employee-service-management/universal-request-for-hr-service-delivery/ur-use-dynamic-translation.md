# Translate a Universal Request field

Use dynamic translation to translate Universal Request fields to another language.

## Before you begin

Role required: admin

## About this task

As of now, you can only translate the short description, description and work notes fields on a Universal Request form.

## Procedure

1.  Navigate to **All** &gt; **Settings** &gt; **General** &gt; **Language**.

2.  Select the language.

    **Note:** Only the languages that have been configured will be displayed. To activate a language, see [Activate a Language](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/platform-administration/system-localization/t_ActivateALanguage.md)

3.  Navigate to **Universal Requests** &gt; **All**.

4.  Open the request to be worked on.

    A translate icon ![Dynamic translation icon](../images/icon-translation.png) appears next to the fields. The logged-in user can click on ![Dynamic translation icon](../images/icon-translation.png) to translate the text to their preferred language as configured. ![Dynamic translation in action](../images/ur-dynamic-translation.gif)
