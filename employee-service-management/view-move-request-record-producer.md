# Update a move-related workplace service record producer

Update a move-related workplace service record producer. Link a workplace service to one or more record producers.

## Before you begin

Ensure that you have the following applications installed:

-   Workplace Core
-   Workplace Case Management

Role required: sn\_wsd\_case.manager or workplace manager

## About this task

After a workplace service is created, the service is linked to a record producer. The record producer is displayed to employees when the employee tries to select the workplace service.

## Procedure

1.  Navigate to **All** &gt; **Workplace Case Management** &gt; **Workplace Case Management - Setup** &gt; **Record Producers**.

2.  Select any of the following record producers which you want to edit:

    -   **Request a desk change**
    -   **Request an employee's desk change**
3.  On the form, edit the fields that you want change.

    For more information about each field on the form, see [Configure a Record producer](../../workplace-case-mgmt/task/wsd-create-record-producer.md).

4.  Click **Update**.


## Result

The record producer is updated.
