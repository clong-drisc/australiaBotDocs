# Bulk case request form

The bulk case request form provides information to create a bulk case request to enable the creation of the same type of HR cases.

<table id="table_abc_p2z_bbc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Number

</td><td>

Automatically generated bulk case request number.

</td></tr><tr><td>

Name

</td><td>

Name describing the bulk case request.

</td></tr><tr><td>

HR service

</td><td>

HR service associated with the bulk case request.

</td></tr><tr><td>

Status

</td><td>

Status of the bulk case request.The bulk case request statuses are:

-   Draft - The default status.
-   Ready - A user segment group has been created for the bulk case request.
-   In progress - The request is in the bulk case request creation process.

</td></tr><tr><td>

Create parent case

</td><td>

Option to create a parent case that is associated with the bulk cases.

</td></tr><tr><td>

Opened by

</td><td>

Name of the user creating the bulk case request.

</td></tr><tr><td>

Selected user count

</td><td>

Number of users selected for the bulk case request. The value of this field is taken from the user selection field while creating a user group segment.

</td></tr><tr><td>

Requested time

</td><td>

Date and time your bulk case request was submitted to create cases. The value of this field is auto-populated upon creation of the user segment group.

</td></tr></tbody>
