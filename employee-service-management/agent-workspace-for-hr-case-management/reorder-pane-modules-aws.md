# Reorder the navigation pane modules in HR Agent Workspace

Quickly access the navigation pane modules in HR Agent Workspace. For example, Home, List, Inbox, and so on.

## Before you begin

Role required: ui\_builder\_admin

## Procedure

1.  Navigate to **All** &gt; **Now Experience Framework** &gt; **UI Builder**.

2.  From the **Experiences** list, select **HR Agent Workspace**.

3.  Select **Open in UI Builder**.

4.  On the HR Agent Workspace Experience page, navigate to the Side navigation section.

    This section displays all modules in the left navigation pane in Agent Workspace for HR Case Management.

5.  Drag the modules in the required order.
