# Create a landing page variant

Create a variant of your landing page to build a custom portal experience for your users.

## Before you begin

Role required: sn\_hr\_agent\_ws.admin and sn\_hr\_core.basic

## Procedure

1.  Navigate to **All** &gt; **HR Case Management** &gt; **UI Builder for Agent Workspace for HR Case Management**.

2.  Select **Landing page default**.

3.  Select the **Settings** tab.

    ![Settings tab](../image/settings_tab_uibuilder.png)

4.  Select **Duplicate**.

    If the **Duplicate** action in the **Actions** column is not selectable and a message about a different application scope is displayed, select the **Edit in original scope** button.

5.  Enter the name of your variant.

6.  Define who can view your page by selecting **Add an audience** and selecting or defining the audience.

    For more information, see [Learn about audiences](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/application-development/ui-builder/add-audiences.md).

7.  Select **Create**.

8.  In the **Order** field, set the order of the variant to a lower value than the Landing page default.

    The order of the variant defines the copy to be used.
