# Import messages from Agent Workspace

You can import the conversation with a user from the agent workspace and append them in the **Comments** to store all the important messages in the ServiceNow.

## Before you begin

Role required: user

## Procedure

1.  Navigate to an open ticket in your ServiceNow instance from agent work space.

    |Record|Description|
    |------|-----------|
    |Incident|Navigate to **Incident** &gt; **Open**. Open the record that you want to work on.|
    |Change Request|Navigate to **Change** &gt; **Open**. Open the record that you want to work on.|
    |Request Item|Navigate to **Requests** &gt; **Open Requests**. Open the record that you want to work on.|

2.  From the **Start Microsoft Teams Chat** menu, select **Import messages**.

    ![Option to import messages.](../images/import-messages-agent-workspace.png)

    **Note:** If you aren't logged in, you receive a prompt to log into Microsoft Teams.

3.  Select the messages that you would like to import, then select **Import messages**.
