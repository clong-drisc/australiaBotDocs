# Default Employee Profile Header Configuration record

The default Employee Profile header configuration record contains the following field values.

|Field|Value|
|-----|-----|
|Name|Default|
|Show Pronoun|Selected|
|Edit Images|Cleared|
|Disable Links|Selected|
|Default|Selected|
|Background Image|Selected|
|Actions|Employee Profile Group|

|Field|Description|
|-----|-----------|
|Choose Username Format|FirstName, LastName|

|Field|Description|
|-----|-----------|
|Fields|user.title,user.department,user.email|

|---|---|
|Channel|Microsoft Teams|
|Show Presence Status|Selected|
