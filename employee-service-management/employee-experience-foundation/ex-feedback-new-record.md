# Feedback configuration form

Description of the feedback configuration form fields to target a portal for feedback responses.

<table id="table_i5m_f4g_tzb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the table record

</td></tr><tr><td>

Feedback portal

</td><td>

Targeted portal for feedback

</td></tr><tr><td>

Anonymous response

</td><td>

-   **User opt-in**: Admin receives the user details from the feedback.
-   **Anonymous**: User details remain anonymous.

</td></tr><tr><td>

Application

</td><td>

Application scope where the record resides.

</td></tr><tr><td>

Active

</td><td>

SelectedOption to enable the application scope.

</td></tr><tr><td>

Order

</td><td>

Order of priority for the record.

</td></tr><tr><td>

Overlay session limit

</td><td>

Number of times the feedback widget reloads.

</td></tr><tr><td>

Feedback title

</td><td>

Introductory text for the feedback request.

</td></tr><tr><td>

Feedback button label

</td><td>

Text visible on the button of the feedback widget.

</td></tr><tr><td>

Privacy Information

</td><td>

-   **User privacy link**: Link that contains terms and conditions of company privacy policy.
-   **User privacy text**: Prompt text to notify the user about the privacy policy.
-   **User privacy help text**: Help text with a short description of the privacy policy.

</td></tr><tr><td>

Success Message Preferences

</td><td>

-   **Success icon**: Icon to display with the success message.
-   **Success text**: Success message that is displayed to the user after submitting feedback.
-   **Success description**: Acknowledgment message that is displayed after you submit your feedback.

</td></tr><tr><td>

Drawer Options

</td><td>

-   **Drawer text**: Welcome text on the feedback drawer widget.
-   **Page placement**: Placement of the feedback drawer on the portal page.
-   **Orientation**: Orientation of the feedback drawer on the portal page.
-   **Position from top**: Position where the Drawer shows up on the page.
