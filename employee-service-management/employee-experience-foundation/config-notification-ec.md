# Configure notifications for portal

Configure the notifications functionality on any portal of your choice to get a unified notification system for all your tasks and reminders.

## Before you begin

Role required: notification\_provider\_admin

## Procedure

1.  Navigate to **Employee Center** &gt; **Portal Notifications** &gt; **Notification Configurations**.

    If you have upgraded to the Employee Center plugin \(sn\_ex\_sp\) version 36.0 along with the Notifications for Employee Center plugin \(sn\_ex\_sp\_notifs\), and the Australia release version, you see the **Employee center configuration** by default in your Notification Configurations.

2.  Select **New** to set up notifications for any other preferred portal.

3.  On the Portal notification configuration form, fill in the fields.

    For a description of the field values, see [Portal notification configuration form](../reference/portal-notif-config-form.md).

    You can also configure the notification content and the trigger conditions for each notification type. For more information, see [Configure notification content for portal](config-notif-content.md) and [Configure notification triggers](setup-notif-content-triggers.md).

4.  Select **Submit**.


## Result

You have configured the notifications for Employee Center functionality on your portal.
