# Link content form

Link content form.

<table id="table_bx5_4pz_vfb"><thead><tr><th>

Field

</th><th>

Descriptions

</th></tr></thead><tbody><tr><td>

Name

</td><td>

The name that describes the link to content.

</td></tr><tr><td>

URL

</td><td>

The URL link to content to appear on the service portal.This field only appears when you select **Video** or **Link** from the **Type** list.

</td></tr><tr><td>

To-do content

</td><td>

The to-do content you want to reference to your link content.This field only appears when you select **To-do** from the **Type** field.

**Note:** Use this field to reference your to-do content to a link that can be used in a banner or styled content. For more information, see [Creating to-do content](../concept/ecpro-content-to-dos.md).

</td></tr><tr><td>

Type

</td><td>

The type of link for the content.-   Video: Link is a video.

**Note:** Must be an embedded link.

-   Link: Link is to content, information, image, or documentation.
-   To-do: Link is a to-do.

**Note:** Use this when you want to place a to-do link in a banner or styled content.

-   Forums Link: Link to a Communities forum post. Only appears when Communities plugin \[com.sn\_communities\] is installed.

 **Note:** After link content is associated with Portal Content, the **Type** cannot be modified. See [Portal content](../concept/ec-streamline-content.md).

</td></tr><tr><td>

Load link in

</td><td>

You can override how your links open within your portal. Previously, all link content loaded in a new tab. Use this field to load your link into the current portal page.

New tab: Default value and loads your link content in a new tab on your portal.

Current page: Select to load your link content in your portal and not as a separate tab.

</td></tr><tr><td>

Forum

</td><td>

The forum in Communities you want the link to appear and share with members of the forum. Only appears when **Forums Link** is selected from **Type** and the Communities plugin \[com.sn\_communities\] is installed.

 For information on Communities, see [Communities](https://raw.githubusercontent.com/ServiceNow/ServiceNowDocs/australia/markdown/customer-service-management/communities/servicenow-communities.md).

</td></tr><tr><td>

Topic

</td><td>

Topic you want the link to appear in. Topics are subcategories to a forum.For example, HR is the forum and Benefits is a subcategory to HR.

 Only appears when **Forums Link** is selected from **Type** and the Communities plugin \[com.sn\_communities\] is installed.

</td></tr><tr><td>

Forum post

</td><td>

The list of posts from the topic or forum the link appears. For example:

-   Forum: HR
-   Open Enrollment: Benefits
-   Forum post:

 Only appears when **Forums Link** is selected from **Type** and the Communities plugin \[com.sn\_communities\] is installed.

</td></tr></tbody>
