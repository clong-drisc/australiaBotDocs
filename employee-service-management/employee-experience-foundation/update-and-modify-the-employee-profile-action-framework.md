# Modify the Employee Profile action group

The Employee Profile actions include some frequently used action items in a framework for quick access.

## Before you begin

Role required: sp\_admin or sn\_hr\_sp.esc\_admin

## About this task

The Employee Profile action framework provides quick access to frequently used action items, like, viewing org chart, using email directly from the Employee Profile page or calls. You can edit the actions beyond the default behaviors to match your requirements.

## Procedure

1.  Navigate to **All** &gt; **Employee Center** &gt; **Action Groups**.

    The default Action Group items available are:

    -   View Org Chart
    -   Email
    -   Call
2.  You can reorder and make any of the items **Primary** as **True** or **False** to access that item as a button or have all the options in a list.

3.  You can edit the **Order** as priority suits and remove the primary button altogether.

4.  You can use the available default **Action Visibility** namely:

    -   Visibility to Self
    -   Visibility to Manager
    -   Visibility to all
    -   Visible to all except Self
5.  You can also create a specific **User Criteria** who can modify the action, as required, or create a custom visibility with the **Advanced** option.


## Result

Use and customize the action framework according to your priority to provide better employee experience with quick accessibility to work email, calls, and other organizational details through the Employee Profile page.
