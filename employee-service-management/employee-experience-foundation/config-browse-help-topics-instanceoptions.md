# Modify the Browse help content widget display

The widget displays the help content for the Employee Center Pro Kiosk user. You can configure the instance options based on your display preferences.

## Before you begin

Role required: sn\_hr\_sp.esc\_admin

## Procedure

1.  Navigate to Employee Center Pro Kiosk home page.

    The kiosk home page is displayed.

2.  Click the **Browse help content** widget.

3.  Press and hold the control key on the keyboard, right-click the widget, and then select **Instance Options**.

4.  Configure the instance options.

<table id="table_cq4_vdk_crb"><thead><tr><th>

Instance option

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Title

</td><td>

</td></tr><tr><td>

Time frame for upcoming shift information

</td><td>

Number of shifts to show. The choices are 7, 14, and 30 days. Default value: 7**Note:** Earliest shift details are displayed.

</td></tr></tbody>
</table>5.  Click **Save**.

    **Note:** The latest shift information is available only after relogging.
