# Quick Link form

Use the Quick Link form to create a new quick link and specify the content type for the quick link.

<table id="table_n3h_cg2_mqb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Content Type

</td><td>

The type of content that you want the quick link to display. The choices are:-   **Knowledge**
-   **Catalog item**
-   **Page**
-   **External link**
-   **Guided Self-service**

</td></tr><tr><td>

Name

</td><td>

The name of the quick link.

</td></tr><tr><td>

Knowledge

</td><td>

Knowledge article you want to open when you click the quick link. This field appears only when **Knowledge** is selected from **Content Type** field.

</td></tr><tr><td>

Catalog item

</td><td>

Catalog item that you want to open when you click the quick link. This field appears only when **Catalog item** is selected from **Content Type** field.

</td></tr><tr><td>

Page

</td><td>

Page you want to open when you click the quick link. This field appears only when **Page** is selected from **Content Type** field.

</td></tr><tr><td>

Additional query parameters

</td><td>

For the page that you selected in the **Page** field, you can specify additional query parameters here. This field appears only when **Page** is selected from **Content Type** field.

</td></tr><tr><td>

External link

</td><td>

External link you want to open when you click the quick link. This field appears only when **External** is selected from **Content Type** field.The background image for quick link is now visible to external users.

</td></tr><tr><td>

Guided Self-service

</td><td>

Guided Self-service that you want to open when you click the quick link. This field appears only when **Guided Self-service** is selected from **Content Type** field.

</td></tr><tr><td>

Active

</td><td>

Option to activate the quick link for use.

</td></tr><tr><td>

Title

</td><td>

The title that you enter in this field overrides the default title of the quick link.

</td></tr><tr><td>

Short description

</td><td>

The short description that you enter in this field overrides the default short description of the quick link.

</td></tr><tr><td>

Background image

</td><td>

The background image for the quick link.

</td></tr><tr><td>

Icon

</td><td>

The icon for the quick link.

</td></tr><tr><td>

Domain

</td><td>

The application scope on which the functionality runs.

</td></tr></tbody>
