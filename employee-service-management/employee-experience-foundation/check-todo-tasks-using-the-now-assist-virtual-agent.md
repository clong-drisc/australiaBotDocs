# Check your to-do tasks by using the Now Assist in Virtual Agent

Check your to-do and pending tasks by using the Now Assist in Virtual Agent AI chat window. By checking these tasks, you can stay organized, prioritize the most important tasks on your list, and monitor how you're doing against your deadlines.

## Before you begin

You can also perform the following steps in Now Assist in Virtual Agent for Microsoft Teams. You can check your to-dos and tasks similarly.

Role required: None

## Procedure

1.  Navigate to **All** &gt; **Self-Service** &gt; **Employee Center**.

2.  Select **Open chat**.

    Now Assist in Virtual Agent appears.

3.  Enter your query.

    To check your to-do tasks, you can enter instructions like `Show my tasks`, `Show my pending tasks`, `Show my open tasks`, or `Show me my tasks due today/due this week/due later/due soon`.

    ![Now Assist for Employee Experience message.](../images/uex-nowassist-startmsg.png)

4.  View the list of tasks, under the To-do Tasks, by selecting **Start**.

    ![Now Assist in Virtual Agent To-do tasks.](../images/uex-nowassist-showmytask.png)

5.  View the list of pending tasks in more detail by selecting **Task Center** in the response.

6.  From the list of tasks displayed, select the task that you want to work on.

    The task details are displayed in the card.

7.  Select the task that you want to view or update.

    Save the changes.

8.  Check other tasks by selecting **Continue**.

9.  End the chat by selecting **End chat**.
