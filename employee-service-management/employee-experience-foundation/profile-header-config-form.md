# Employee Profile Header Configuration form

Use the Employee Profile Header Configuration form to create a header for a profile.

|Field|Description|
|-----|-----------|
|Name|Default name or any other name.|
|Show Pronouns|Option to display user pronouns.|
|Edit Images|Option to enable employees to edit the avatar and background image.|
|Disable Links|Option to disable links in the header.|
|Background Image|Option to enable the background image.|
|Action Groups|Record of the action group to be added in the header.|
