# Notification content form

Notification content form

<table id="table_tqr_vhn_vlb"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Title

</td><td>

Title that displays for the push notification.

</td></tr><tr><td>

From

</td><td>

The email address of the person or organization you want the notification to come from.**Note:** This field only appears when **Content type** is **Email**.

</td></tr><tr><td>

Reply to

</td><td>

Email address you want to appear in the **Reply to** field of the email.

</td></tr><tr><td>

Subject

</td><td>

The subject of the email.

</td></tr><tr><td>

Use block content

</td><td>

Indicates you want to use block content as a notification.

</td></tr><tr><td>

Block content

</td><td>

Only displays when you check **Use block content**. Select the block content you want to appear as a notification or select **New** to create block content.**Note:** For more information, see [Add or modify block content](../task/ecpro-manage-block-content.md).

</td></tr><tr><td>

Body

</td><td>

Only displays when you check **Use block content**. Enter message text for your notification.**Note:** You can use the rich text edits to enhance your notification.

</td></tr><tr><td>

Description

</td><td>

Only appears when you select **Push** from the **Content type** list.A description for the push notification.

</td></tr><tr><td>

Content type

</td><td>

The type of notification you want to send. Set the value to `Push` or `Email`.

</td></tr><tr><td>

Active

</td><td>

Option to activate the push notification for use.**Note:** Your content must be active in order to publish it.

</td></tr></tbody>
