# Enhanced Requests Experience on Now Mobile

You can get the Enhanced Requests Experience on the Now Mobile app, bringing the full request management capabilities from the Employee Center portal to mobile. Employees can view, search, filter, and manage their requests on the go.

You can directly view your Enhanced requests experience from the Now Mobile homepage. Under **My items**, select **Requests**.

**Important:** To get the Enhanced Requests Experience on the Now Mobile app, you must opt-in for the setup. For more information, refer to [Opt in to the enhanced Enhanced Requests Experience](../task/opt-in-emre.md).

## Requests

The Enhanced requests experience page gives you a consolidated view of all your requests on your mobile device. You can switch between tabs to see requests relevant to you, and use the search bar and filter options to quickly find what you need. Each request shows key details like its status, title, number, and when it was last updated, making it easy to stay on top of your requests at a glance.

![Enhanced Requests page on Now Mobile app.](../images/enh-request-page.png)

-   **Tabs**

    The Enhanced requests experience page supports multiple tabs to help you view requests by type. Each tab shows a relevant set of requests, making it easy to switch between views. You can view any additional tabs configured based on your organization requirements by selecting the Show more icon.

    ![Additional tabs display on mobile.](../images/enh-request-tabsptions.png)

    The following default tabs are available:

    -   **For you**: Displays requests created for you or by you.
    -   **For others**: Displays requests you have created on behalf of others.
    -   **Drafts**: Displays requests saved as drafts that have not yet been submitted. You can open a draft request and submit it anytime.
-   **Filters**

    You can apply filters to narrow down the relevant request. Select the Filters button \(![Filter icon](../images/enh-request-mob-filter-icon.png)\) to see requests of certain type. Apply filters based on their Request type, Status, Updated, Department, and Urgency.

    ![Status filter options.](../images/enh-request-filteroptions.png)

    ![Filters for filtering the requests.](../images/enh-request-status.png)

-   **Search for keyword**

    Use the search bar to find requests by keyword.

-   **Request details**

    Each request card shows key details at a glance, so you can track and manage requests on the go. Tap a card to view the full request details.

    ![Request details page on the Now Mobile app.](../images/enh-request-details.png)

    Each request card shows the following details:

    -   Request number
    -   Short description
    -   Status
    -   Last updated time
    -   Priority or urgency
    -   Department and Requested for fields
