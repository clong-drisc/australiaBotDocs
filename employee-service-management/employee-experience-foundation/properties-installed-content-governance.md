# Properties installed with Content Governance

The Content Governance \[sn\_cg\] plugin adds properties that determine how many records to show in the Content Operations dashboard.

<table id="table_btz_j1r_gnb"><thead><tr><th>

Property

</th><th>

Usage

</th></tr></thead><tbody><tr><td>

Set amount of days from when campaign content will be expiring

</td><td>

Sets the number of days for the "Campaign content expiring within x days" card in the Content library and campaigns sectionDefault value: 7 days

</td></tr><tr><td>

Set amount of days from when content needs approval

</td><td>

Sets the number of days for the "Content requests due within x days" card in the Content requests sectionDefault value: 7 days

</td></tr><tr><td>

Set amount of days from when schedule content will be expiring

</td><td>

Sets the number of days for the "Content expiring within x days" card in the Content library and campaigns sectionDefault value: 7 days

</td></tr></tbody>
