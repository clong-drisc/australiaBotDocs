# Professional ID form

Access, update, and validate the Professional ID details of a healthcare professional in the Professional ID form.

<table id="id_ggf_35j_jbc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Professional profile

</td><td>

Professional profile associated to the healthcare professional.

</td></tr><tr><td>

ID number

</td><td>

Professional ID number of the healthcare professional.

</td></tr><tr><td>

Issue date

</td><td>

Date on which the Professional ID was issued to the healthcare professional.

</td></tr><tr><td>

Expiration date

</td><td>

Date when the Professional ID will be expired.

</td></tr><tr><td>

Active

</td><td>

Option to make the Professional ID record available to use.

</td></tr><tr><td>

Status

</td><td>

Status of the Professional ID record: -   New: When profile intake form is sent to the healthcare professional for filling.
-   Pending validation: When data is filled in by healthcare professional and sent for credentialing and validation to HR agents.
-   Invalid: When HR agents review the data and reject due to insufficient or invalid details.
-   Valid: When HR agents validate and approve the information provided by the healthcare professional.

</td></tr><tr><td>

State of registration

</td><td>

State in which the Professional ID has been issued.

</td></tr><tr><td>

Country

</td><td>

Country in which the Professional ID has been issued.
