# Languages Spoken form

Access, update, and validate the language details of a healthcare professional in the Languages form.

<table id="table_l3d_s5j_jbc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Employee profile

</td><td>

Employee profile associated with the healthcare professional.

</td></tr><tr><td>

Professional profile

</td><td>

Professional profile associated with the healthcare professional.

</td></tr><tr><td>

Language code

</td><td>

Option to choose a language name.

</td></tr><tr><td>

Active

</td><td>

Option to indicate that the language spoken record is active.

</td></tr><tr><td>

Primary

</td><td>

Option to indicate that the language specified is primary language of the healthcare professional.

</td></tr><tr><td>

Status

</td><td>

Status of the languages spoken record: -   New: When profile intake form is sent to the healthcare professional for filling.
-   Pending validation: When data is filled in by healthcare professional and sent for credentialing and validation to HR agents.
-   Invalid: When HR agents review the profile data and reject due to insufficient date or invalid details.
-   Valid: When HR agents validate and approve the information provided by the healthcare professional.
