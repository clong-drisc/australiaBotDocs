# Professional speciality form

Review and update the Professional speciality details of a healthcare professional in the Professional speciality form.

<table id="table_l3d_s5j_jbc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Professional profile

</td><td>

Professional profile associated to the healthcare professional.

</td></tr><tr><td>

Professional specialty

</td><td>

Healthcare domain in which the healthcare professional specialises, or has taken the required training or certification.

</td></tr><tr><td>

Initial certification date

</td><td>

Date on which the healthcare professional received the specialization certificate.

</td></tr><tr><td>

Expiration date

</td><td>

Date on which the certification expires.

</td></tr><tr><td>

Recertification date

</td><td>

Date on which recertification needs to be taken by the healthcare professional.

</td></tr><tr><td>

Active

</td><td>

Option to make the professional speciality details record available to use by other workflow or teams.

</td></tr><tr><td>

Type

</td><td>

Type of certification: Primary or Secondary

</td></tr><tr><td>

Are you board certified

</td><td>

Option to indicate that the healthcare professional is certified by a medical board.

</td></tr><tr><td>

Certifying board code

</td><td>

Name of the certifying board code.**Note:** This field appears only when the **Are you board certified** field is selected.

</td></tr><tr><td>

If not board certified

</td><td>

Option to select when the user is not certified by a medical board.**Note:** This field does not appear when the **Are you board certified** is selected.
