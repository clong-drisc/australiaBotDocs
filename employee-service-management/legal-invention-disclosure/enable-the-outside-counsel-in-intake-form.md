# Enable Outside Counsel to be added to Legal Invention Disclosure matters

Enable outside counsels to be added to matters related to Legal Invention Disclosure request.

## Before you begin

Role required: sn\_lg\_ip.patent\_config, sn\_lg\_matter.matter\_config

## Procedure

1.  Navigate to **All** &gt; **Legal Administration** &gt; **Practice Areas**.

2.  Select **Intellectual Property** from the list.

3.  Select **Invention Disclosure** in the **Intake Forms** tab.

    If a message appears about the application scope, select **here** to be able to edit the record.

4.  Select the lock icon ![Lock icon.](../../legal-request-management/image/lock.png) in the **Options** field in the Record Producer section.

5.  Enter `Enable` in the Select target record field and select **Enable Outside Counsel on Matter** from the list.

6.  Select **Update**.
