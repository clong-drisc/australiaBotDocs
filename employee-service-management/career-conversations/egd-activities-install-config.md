# Install and configure Career Conversations

Install and configure the features and components for the Career Conversations application.

-   **[Install Career Conversations](../task/egd-activities-install.md)**  
Install the Career Conversations app to enable employees and managers to create, manage, and track conversations.
-   **[Add and manage Talking Points](../task/egd-add-manage-talking-point.md)**  
As a Career Conversations admin, you can add or update talking points that are used by managers to help create a conversation with an employee.
-   **[Add Conversations](../task/egd-admin-create-conversation.md)**  
As an Career Conversations admin you can add or manage conversations that are assigned to employees on behalf of managers.
-   **[Configure automatic closure of past conversations](../task/egd-admin-auto-close-conversation.md)**  
Use a system property to define whether a batch job can automatically close conversations that have transpired.
-   **[Set up Microsoft Outlook integration in Career Conversations](outlook-integration-cc.md)**  
Perform the following steps to set up an integration with Microsoft Outlook
