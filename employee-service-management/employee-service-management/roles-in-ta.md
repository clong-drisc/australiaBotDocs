# Roles in Hiring Experiences

An overview of the different roles and user criteria in Hiring Experiences.

|Name|Description|
|----|-----------|
|Hiring Core admin \[sn\_ta\_hiring\_core.admin\]|This role is the super admin in the Hiring Experiences environment.|
|Hiring Core - Recruiter \[sn\_ta\_hiring\_core.recruiter\]|This role is for the recruiter who creates and manages the job requisitions.|
|Hiring Core - Interviewer \[sn\_ta\_hiring\_core.interviewer\]|This role is for Interviewers who evaluate the applicants on the Job requisition.|
|Hiring Core - Recruitment Coordinator\[sn\_ta\_hiring\_core.recruitment\_coordinator\]|This role is for the recruiter coordinator who creates interviews and manages the application workflow.|
|Hiring Core - Hiring manager \[sn\_ta\_hiring\_core.hiring\_manager\]|This role is for the hiring manager for a job requisition.|
|Hiring Core - Applicant \[sn\_ta\_hiring\_core.applicant\]|This role is to provide common information to both the internal and external applicants. This role is a parent role of both the internal and the external applicant roles.|
|External applicant \[sn\_ta\_hiring\_core.external\_applicant\]|This role is for the external applicants and contains the snc\_external role. It applies to any external applicant who isn’t an employee of the company.|
|Internal applicant \[sn\_ta\_hiring\_core.internal\_applicant\]|This role is for internal applicants and contains the snc\_internal role. This only applies to existing employees in the company.|
|approver\_user|This role is required for approver users.|
|Talent Profile admin \[sn\_ta\_tp.talent\_profile\_admin\]|This role is required to set up Talent Profile.|
|Talent Profile user \[sn\_ta\_tp.talent\_profile\_user\]|This role is required to use the Talent Profile functionalities.|
|admin|This is the administrator role. This role has access to all the system features, functions, and data, regardless of the security constraints.|
