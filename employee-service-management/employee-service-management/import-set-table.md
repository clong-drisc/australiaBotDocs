# Load data form

Use the import sets to dump data to your local instance from a third-party integration.

|Field|Description|
|-----|-----------|
|Import set table|Type of import set table.|
|Label|Label of the table being populated.|
|Name|Name of the table being populated.|
|Source of the import|Type of data source.|
|File|Option to upload a file.|
|Sheet number|Number of allowed sheets in the uploaded file.|
|Header row|Number of header row allowed in the uploaded file.|
