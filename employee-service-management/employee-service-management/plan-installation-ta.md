# Plan your installation

Identify the requirements and plan your installation of the Hiring Experiences suite.

**Parent Topic:**[Configuring Hiring Experiences](config-frmwrk-ta.md)

## Dependencies of Hiring Experiences

The following plugins are required to use all the features in the Hiring Experiences suite.

Hiring Experiences has the following major components:

-   Hiring tab \(sn\_ta\_hiring\_mgr\)
-   Recruitment workspace \(sn\_ta\_recruiter\)
-   Applicant Center \(sn\_ta\_applicant\)
-   Hiring Connector \(sn\_ta\_connector\)
-   Talent Profile \(sn\_ta\_tp\)
-   Interview Management \(sn\_ta\_int\_mgmt\)

When you install all the plugins within the Hiring Experiences suite, the following plugins are installed by default.

-   Employee Center Pro \(sn\_ex\_sp\_pro\)
-   Manager Hub \(sn\_mh\)
-   Employee Profile \(com.sn\_employee\)
-   Content Experiences \(sn\_ca\)
-   Skills Foundation \(sn\_skills\_int\)
-   Finance Common Architecture \(sn\_fin\)
-   Hiring Core \(sn\_ta\_hiring\_core\)
-   Time Limited Authentication \(com.snc.authenticate.time\_limited\_authentication\)
-   Explicit roles \(com.glide.explicit\_roles\)
