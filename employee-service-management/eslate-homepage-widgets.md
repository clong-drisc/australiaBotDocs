# Employee Slate home page widgets

Home page widgets provide quick access to tasks, announcements, and popular content in a personalized dashboard that aggregates relevant information for daily work.

Employee Slate home page includes five default widgets. These widgets surface actionable information and give employees a consolidated view of what requires attention. The widgets adapt based on role, department, and usage patterns to display the most relevant information and services.

## AI-powered chat and search

The home page features a prominent AI-powered chat and search experience at the top. Employees can find anything across the platform without navigating elsewhere, including knowledge articles, catalog requests, or quick answers.

## Default home page widgets

Employee Slate includes the following default widgets:

-   **Inbox**

    Displays approval items and tasks assigned to the employee. Selecting an approval item opens the detailed view where employees can review details and take the necessary action. Provides proactive nudging on items requiring action instead of requiring employees to discover them by navigating to inbox details.

-   **Announcements \(Employee Communications\)**

    Shows promoted content as a carousel. Selecting an announcement opens the announcement details.

-   **Popular Content**

    Highlights popular knowledge articles and catalog items that other employees are engaging with. Shows the most viewed knowledge and most requested catalog items within a set timeframe.

-   **Upcoming Holiday**

    Displays the next holiday based on the employee work location.

-   **Quick Links**

    Enables administrators to configure knowledge, catalog, or external links. Selecting a quick link takes employees to the destination to complete any required action.


## Widget navigation

Each widget provides navigation options to access more detailed views:

-   Widget buttons provide navigation to full-page listing views where employees can view complete information.
-   Individual items within widgets link to detailed views for specific actions.
-   Employees can return to the home page from any detailed view using navigation controls.
