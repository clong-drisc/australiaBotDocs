# Configure the menu for Contractor Service Center

As an admin you can configure the menu for the contractor service center. You can update the **Quick links to get you started** or the **Explore our categories** menu items, or create a menu item.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Contractor service center** &gt; **Menu configuration**.

2.  Select the existing menu titles to edit them or select **New** to add a new menu.

3.  To edit the existing menu,

    1.  Select an existing menu.

    2.  Select an existing menu item to edit it or select **New** to create a menu item.

    3.  For a new menu item, fill in the form.

        For form field description, see [Contractor service portal menu item form](../concept/hs-contractor-service-portal-menu-item-form.md)

    4.  Select **Update**.

    The menu item appears on the portal.
