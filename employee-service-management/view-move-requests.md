# View move requests

Monitor the move requests that are assigned to you, the requests that are currently open, and all the other requests that are created in the application.

## Before you begin

Role required: sn\_wsd\_move.manager

## Procedure

1.  Navigate to **All** &gt; **Workplace Move Management** &gt; **Move requests**.

2.  To view move requests, select any of the following options:

    -   **Assigned to me**: Select this option to view the move requests that are assigned to you.
    -   **Open requests:** Select this option to view open move requests.
    -   **All move requests**: Select this option to view all the move requests.
3.  Scroll down to the Related list section to check the status of the following types of items:

    -   Child cases
    -   Child tasks
    -   Approvers
    -   Case SLAs
    -   Associated knowledge base articles
    -   Related cases
    -   Conflicts review
