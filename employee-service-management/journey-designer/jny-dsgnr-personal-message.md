# Send a personal message to journey users

Journey owners and mentors with the **Can add personalized message** permission can send a personal message to the employee assigned to a journey.

## Before you begin

Role required: Managers and mentors with **Can add personalized message** permission

## About this task

The personalized message feature requires that the **Can add personalized message** is enabled for journey owners or mentors. This permission is set for each journey configuration. See, [Update users' permissions for journeys](jny-dsgnr-update-user-permissions.md).

The **Send a note** button is only visible in published journeys.

## Procedure

1.  From the Employee Center, select **Journeys**.

2.  Create and publish a journey, or open an existing journey that has been published.

    Journey configurations with the **Can add personalized message** enable a **Send a note** button.

3.  Select **Send a note**.

    You can modify the note and add attachments.

4.  Select **Send** to send the note to the user.

    An email is sent to the email address that is in the HR profile.
