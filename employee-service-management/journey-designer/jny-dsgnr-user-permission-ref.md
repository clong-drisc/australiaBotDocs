# Journey designer user permissions reference

Administrators can configure the user permissions for different Journey designer journey configurations.

## Journey owner and mentor permissions

Some permissions are only available when other applications are integrated with Journey designer.

<table><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

**Can add mentors**

</td><td>

Allows a user to add mentors to a journey. Adding mentors is helpful if an employee needs guidance from a user with a special skill set.

</td></tr><tr><td>

**Can edit journey**

</td><td>

Allows a user to modify a journey. Managers can enable mentors to modify an employee's journey.

</td></tr><tr><td>

**Can view satisfaction score** **Note:** Available for only journey owners, and only available if the Listening Posts app is installed.

</td><td>

Allows a journey owner to view statistical data related to an organizations journey.

</td></tr><tr><td>

**Can add recommended learning**

</td><td>

When the Learning Posts app is installed and configured, journey owners and mentors can add recommended learning to an employee's journey.

</td></tr><tr><td>

**Can add quick links**

</td><td>

Allows the user to add quick links to a journey. Quick links can link to different content types. There are two types of quick links.-   Administrator can add quick links to a journey configuration.
-   Journey owners and mentors can add URL links to individual journeys.

</td></tr><tr><td>

**Can add personalized message**

</td><td>

Enables journey owners and mentors to include personalized messages from a journey to employees.

 Permission can be set for either journey owners or mentors.

**Note:** The **Send a note** button is visible in only published journeys.

</td></tr></tbody>
