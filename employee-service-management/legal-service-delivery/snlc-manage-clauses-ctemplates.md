# Manage clauses and contract templates

As administrator manage clauses, clause variations, and contract templates.

-   **[Edit a contract clause variation](../../contract-mgmt-pro/task/cncore-edit-cv.md)**

    Edit a clause variation to modify the content that will be used in a contract template.

-   **[Edit contract templates by manually adding content controls](../../contract-mgmt-pro/task/cncore-edit-ct.md)**

    Edit a contract template to reflect any modifications that must be included in the template to generate a standard contract when an employee submits a contract request.

-   **[Edit a contract template using Microsoft Word add-in for ServiceNow Contracts](../../contract-mgmt-pro/task/cncore-edit-ct-addin.md)**

    Edit the metadata, signatories, or clauses in a contract template that must be included in the contract template to generate a standard contract when an employee submits a contract request using the Microsoft Word add-in for ServiceNow Contracts.

-   **[Create a contract template from an existing template](../../contract-mgmt-pro/task/cncore-create-template-revision.md)**

    Create a new contract template from an existing template if you need only a small variation in the content of an existing template.

-   **[Generate and preview a contract document from a template](../../contract-mgmt-pro/task/cncore-preview-template.md)**

    View the final form of a contract document with the content controls replaced with the values of the variables or return values of scripts.

-   **[Delete a contract template](../../contract-mgmt-pro/task/cncore-delete-template.md)**

    Delete contract template when you no longer need it.
