# Isolate a layer in the CAD editor

CAD editor provides the ability to isolate a layer from multiple layers while editing your CAD file.

## Before you begin

Role required: admin

## Procedure

1.  Select the **CAD Editor** tab.

2.  Navigate to **Autocad Layers**.

    ![Autocad layers and filter layers by layer type.](../images/wsd-autocad-editor-view.png)

3.  Use the **Filter layers** option to filter and search for layers.

    When you have numerous layers in your CAD file, the filter option helps in sorting and working on a few selected layers at a time.

4.  Select a layer and select the Isolated icon \( ![Isolate a layer icon.](../images/wsd-autocad-isolate-icon.png) \).

5.  The preview shows the layer that was isolated.

    For example: polylines or polygons.

6.  Use the Select tool icon \( ![Select tool icon.](../images/wsd-autocad-select-icon.png)\) to select the isolated layer in the editor.

7.  Edit and make changes as required.

    **Note:** Blocks can’t be edited but attributes within a block can be edited.
