# Drawing and editing tools in the CAD editor

The CAD Editor tab provides editing and drawing tools to configure, optimize, and edit your CAD files before importing it to the Map Studio.

It provides the following editing and drawing capabilities:

-   Draw line: The Draw line tool \(![Draw line icon](../images/wsd-draw-line-icon.png)\) is used to draw a line or polyline. Use the Draw line tool to close unclosed polylines in a polygon. For more information, see [Draw a polyline or a polygon](../task/draw-a-line-autocadv2.md).
-   Text: The Draw text tool \(![Draw Text icon.](../images/wsd-text-icon.png)\) is used to add text or mtext \(multiline\) text objects. For more information, see [Update or add text attributes](../task/update-text-autocadv2.md).
-   Select tool: The select tool \(![Select icon](../images/wsd-autocad-select-icon.png)\) is used to select regions, objects, or elements within an active layer.
-   Undo: The undo tool \(![Undo icon.](../images/wsd-autocad-undo-icon.png)\) is used to revert the most recent editing action or actions to their previous state. The undo tool restores the CAD edits to its previous state before the action was performed.
-   Redo: The Redo tool ![Redo con.](../images/wsd-autocad-redo-icon.png) is used to reapply or cancel modifications to the CAD source file in the CAD editor. The CAD editor reapplies the last action that was modified.

-   **[Draw a polyline or a polygon](../task/draw-a-line-autocadv2.md)**  
Draw a polyline, polygon, or a geometrical shape using the Draw line tool available in the CAD Editor. Join lines to form a polygon or a shape.
-   **[Update or add text attributes](../task/update-text-autocadv2.md)**  
Add, edit, update, or remove text attributes in the CAD Editor.
-   **[Editing block attributes](../task/editing-block-attributes.md)**  
Edit block attributes in CAD Editor.
-   **[Closing a geometry or a polyline in a layer](../task/editing-a-shape-autocadv2.md)**  
Close an unclosed line in a layer on the CAD editor.
