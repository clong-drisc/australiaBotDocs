# Editing block attributes

Edit block attributes in CAD Editor.

## Before you begin

**Note:** Blocks can’t be edited. Only attributes within a block can be edited. Polylines linked to a block can’t be edited.

Role required: admin

## Procedure

1.  Select the **CAD Editor** tab.

2.  On the preview layout, select a block reference.

3.  The editing panel appears.

    ![Edit block attributes.](../images/wsd-block-edit-panel.png)

4.  Edit the attributes as required.

    For example, edit the text attributes in a block.

5.  Select **Remove** to remove the element from the CAD drawing.
