# Closing a geometry or a polyline in a layer

Close an unclosed line in a layer on the CAD editor.

## Before you begin

A shape is closed when all the nodes are connected by lines. If a line is missing or if the vertices aren’t joined properly, the properties for a selected geometry show the shape as unclosed.

Role required: admin

## Procedure

1.  Select the **CAD Editor** tab.

2.  Select a polyline, a shape, or a geometry with unclosed lines or vertices in the CAD file.

3.  Select to open the editing panel.

    The editing panel displays a "The shape isn’t closed" validation message.

    ![elected entity showing polyline as unclosed.](../images/wsd-autocad-polygon-shape-unclosed.png)

4.  Select **Close shape** to close the polylines automatically.

    The shape is closed.

    ![Selected entity showing polyline as closed.](../images/wsd-autocad-polygon-shape-closed.png)

5.  Select **Remove** to remove the element from the CAD drawing.
