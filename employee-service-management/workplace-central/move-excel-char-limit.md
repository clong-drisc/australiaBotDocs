# Excel column lengths for move projects

The Excel columns must be within the specified length to create a move project.

|Excel Column|Column length|
|------------|-------------|
|Employee name|151|
|Employee email|100|
|Urgency|40|
|From building|100|
|From floor|
|From space|
|To building|
|To floor|
|To space|
|Additional details|4000|
