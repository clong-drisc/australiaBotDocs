# User Workplace Profile form

The User Workplace Profile form specifies details about an employee like their name, primary location, and allocation.

<table id="table_ugg_krv_h3c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Employee

</td><td>

Employee who the workplace profile must be created for.

</td></tr><tr><td>

Active

</td><td>

Option to set the workplace profile to active.

</td></tr><tr><td>

Primary workplace location

</td><td>

Location assignment of the user that is set as primary.This field is automatically filled can't be edited.

</td></tr><tr><td>

Reason for anomaly

</td><td>

Anomaly type for the location assignment of the profile.This field is automatically set to **no** and can't be edited.

</td></tr><tr><td>

Workplace entity

</td><td>

Workplace entity that the user must be allocated to.For more information about workplace entity, see [Configure Workplace entity and entity types](../../workplace-service-delivery/concept/workplace-entity.md).

</td></tr><tr><td>

Enable employee location privacy

</td><td>

Location privacy setting for the workplace profile. This field is automatically set to **None** and can't be edited.

</td></tr></tbody>
