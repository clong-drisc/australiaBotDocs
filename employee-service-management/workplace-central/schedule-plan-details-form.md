# Schedule Plan details form

The details form helps you select the project and schedule for your schedule plan.

<table id="table_s51_tq4_vcc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the schedule plan.

</td></tr><tr><td>

DepartmentCost Center

Workplace Entity

</td><td>

Department, Cost Center, or Workplace Entity that the schedule plan is created for.This field name is based on the value of the **sn\_wsd\_core.ALLOCATION\_TYPE** property.

</td></tr><tr><td>

Description

</td><td>

Description for the schedule plan.

</td></tr><tr><td>

Active

</td><td>

Option to activate the schedule plan record.This field can't be edited and is automatically updated based on the schedule plan state. This field is selected by default and is cleared if the schedule plan is cancelled or complete.

This field appears only after the schedule plan record is created and is visible while editing the schedule plan.

</td></tr><tr><td>

Assigned to

</td><td>

User that the schedule plan is assigned to.This field is automatically set to the user who creates the schedule plan.

This field appears only after the schedule plan record is created and is available while editing the schedule plan.

</td></tr><tr><td>

Domain

</td><td>

Domain that the schedule plan is created for.This field appears only after the schedule plan is created and is available while editing the schedule plan.

</td></tr></tbody>
</table><table id="table_dkv_j1c_wcc"><thead><tr><th>

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Schedule

</td><td>

Schedule that is followed to assign workspaces.

</td></tr><tr><td>

Start date

</td><td>

Start date of the workspace assignments.The start date must be in the future.

</td></tr><tr><td>

End date

</td><td>

End date of the workspace assignments.

</td></tr></tbody>
</table>**Note:** You can edit a schedule plan only if it is in the Draft state. If the schedule plan is not in the Draft state, you can only change the **Name**, **Assigned to**, and **Description** fields.
