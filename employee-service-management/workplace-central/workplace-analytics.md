# Workplace Analytics

The ServiceNow Workplace Analytics dashboard displays key metrics that enable you to optimize the use of workplace resources and maximize your return on investment.

## Key features

-   Data-driven insights, trends, and key performance indicators for the following applications:
    -   Space Optimization
    -   Lease Administration
    -   Maintenance Management
    -   Workplace Connectors
-   Filters to narrow down the data selection.

## Benefits

-   Key metrics and trends in a single space
-   Data range selection using filters
-   Anticipate trends
-   Prioritize resources by analyzing trends
-   Reduced cost through resource optimization
-   Customizable widgets to see relevant metrics

## Accessing Workplace Analytics

To access Workplace Analytics, Workplace Central must be installed. For more information, see [Install Workplace Central](../task/install-workplace-central.md).

You can access Workplace Analytics by using one of the following methods:

-   Navigate to **All** &gt; **Workplace Central** &gt; **Workplace Central**.
-   Navigate to **Workspaces** &gt; **Workplace Central**.

For more information about how to use Workplace Analytics, see [Working with Workplace Analytics](working-with-workplace-analytics.md).
