# Create a workplace case on behalf of an employee through Workplace Agent for mobile

Create workplace cases on behalf of an employee in the Workplace Agent for mobile work view page.

## Before you begin

Verify that you have installed the required plugins.

Role required: sn\_wsd\_case.workplace\_agent

## Procedure

1.  Log in to Workplace Agent for mobile application.

2.  On the **Work view** page, click the quick action option on the top corner of the header.

3.  Tap **Create a case**.

4.  On the **Create a case** page, select any option.

5.  On the form, fill in all the required details.

6.  Tap **Submit**.
