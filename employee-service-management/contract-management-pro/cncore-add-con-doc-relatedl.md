# Add Contract documents tab to the contract repository record

Enable display of contract documents in a contract repository record by adding the Contract documents tab to it.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Definition** &gt; **Tables**

2.  Select the table configured as a contract repository.

    For more information, see [Create a contract configuration](cncore-contract-config.md).

3.  Right-click the context menu icon ![Context menu icon](../../universal-task/images/context-menu.png) and select **Configure** &gt; **Related Lists**.

4.  In the **View** list, select your workspace.

5.  Add the Contract documents tab in contract repository records by moving **Contract documents** from the Available list to the Selected list.

    ![Moving contract documents from Available list to Selected list.](../image/cmpro-enable-contract-doc.png "Add contract documents related list")

6.  Select **Save**.

7.  Select **Update**.


## Result

The Contract document tab is available your workspace to view the contract documents and it's versions.

![The contract document related list is available in your record](../image/cmpro-bu-add-doc-rl-result.png "Contract document related list")
