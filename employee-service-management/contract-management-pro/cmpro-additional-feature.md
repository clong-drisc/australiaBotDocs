# Configure additional features in Contract Management Pro

Set up these configurations to unlock additional capabilities in Contract Management Pro, such as enabling email notifications, displaying related lists in contract requests, generating audit certificates, activating contract workspace, and more.

-   **[Configuring Contract Workspace](cncore-conf-cntrct-wrkspc.md)**  
Configure and customize Contract Workspace by using the UI Builder web user interface builder.
-   **[Configure signature pause duration when modifying signatories](../task/cncore-pause-sig-property.md)**  
Configure the duration for which a signature workflow is paused when signatories are modified for an ongoing signature workflow.
-   **[Auto-populate the start date and end date for contract requests](../task/cncore-conf-start-end-date-for-cntrcts.md)**  
Configure an extension point implementation to automatically add the start date and end date while creating a contract request.
-   **[Activate a system property to generate a certificate of completion](../task/cncore-config-system-prop-COC.md)**  
As a contract configurator, activate a system property to generate a certificate of completion for electronically signed contracts.
-   **[Enable users to view email details in activity stream](../task/cncore-enbl-user-email-astream.md)**  
As a contract configurator, specify the user roles to enable users to view email details in the activity stream of contract requests.
-   **[Enable keyword search for contract templates](../task/cncore-enable-search-cont-tmplts.md)**  
Enable keyword search in contract templates.
-   **[Configuring contract summarization for Contract Management Pro](../task/cmpro-conf-cntr-summary.md)**  
Configure the smart documents skill in Now Assist in Document Management to generate a contract summary in Contract Management Pro.
-   **[Configure conditions to send reminder notifications for expiring contracts](../task/cncore-config-rem-exp-cont.md)**  
Configure contract repository rules to determine the expiration conditions under which reminder notifications for expiring contracts are sent.
-   **[Copy fields from parent request to amendment request](../task/cncore-cpy-fld-parent-amedreq.md)**  
Configure the ContractManagementExt extension point to automatically copy fields to contract request from the linked parent request and parent contract.
-   **[Manage notifications in Contract Management Pro](cmpro-manage-notifications.md)**  
Configure the notifications in Contract Management Pro.
