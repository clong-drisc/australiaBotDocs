# Submit a maintenance request

Raise a maintenance request for an equipment for a location using either the Workplace Service Portal or the Employee Center.

## Before you begin

Role required: sn\_wsd\_core.workplace\_user

## Procedure

1.  Navigate to the portal from one of the following locations.

<table id="choicetable_presence_dashboard"><thead><tr><th align="left" id="d461772e57">

Location

</th><th align="left" id="d461772e60">

Navigation

</th></tr></thead><tbody><tr><td id="d461772e66">

**Workplace Service Portal**

</td><td>

1.  Navigate to **All** &gt; **Workplace Core** &gt; **Workplace service portal**.

The portal homepage opens.

2.  Open the Site safety category.

    1.  Select **Catalog**.
    2.  Select **Browse by Categories**.
    3.  In the Workplace Services Catalog, select **General**.
**Note:** If there are multiple catalogs configured on the portal, go to the Catalogs list and select **Workplace Services Catalog** &gt; **General**.

3.  Select **Submit a maintenance request**.


</td></tr><tr><td id="d461772e136">

**Employee Center**

</td><td>

1.  Navigate to **All** &gt; **Self-Service** &gt; **Employee Center**.
2.  On the Employee Center landing homepage, select the **Workplace Services** &gt; **Maintenance**.
3.  Select **Submit a maintenance request**


</td></tr></tbody>
</table>2.  On the form, fill in the fields.

<table id="table_ogt_gql_w1c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

What would you like to report?

</td><td>

Select the type of your maintenance request from the following options:-   **Equipment issue**: Select this option if the maintenance request is related to an asset or a physical item.
-   **Location**: Select this option is the maintenance request is related to a location.


</td></tr><tr><td>

Item

</td><td>

Select the item that you want raise the request for.**Note:** The option appears only if you selected **Equipment issue** as the type.

</td></tr><tr><td>

Find the space on a floor map

</td><td>

Select the location on the floor map.**Note:** The option appears only if you selected **Location** as the type.

</td></tr><tr><td>

Building

</td><td>

Select the building where the location is located.**Note:** The option appears only if you selected **Location** as the type.

</td></tr><tr><td>

Floor

</td><td>

Select the floor where the location is located.**Note:** The option appears only if you selected **Location** as the type.

</td></tr><tr><td>

Space

</td><td>

Select the space that you want to report.**Note:** The option appears only if you selected **Location** as the type.

</td></tr><tr><td>

Describe the issue

</td><td>

Enter a detailed description of the issue.

</td></tr><tr><td>

Add attachments

</td><td>

Select the option if you want to add any attachments.

</td></tr></tbody>
</table>3.  Select **Submit**.

    The maintenance request is submitted. A workplace maintenance case is created and the state is set to Work in Progress.
