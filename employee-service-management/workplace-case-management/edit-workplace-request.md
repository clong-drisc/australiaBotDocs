# Edit a workplace request

Edit the details of a workplace request or change workplace service items included in a workplace request.

## Before you begin

Editing a request is supported for the following request types:

-   Badge requests
-   Catering
-   Cleaning
-   Employee parking
-   Equipment
-   Furniture request
-   General request
-   Heating and cooling request
-   Space arrangement request
-   Tech support

If you have created a request from a workplace reservation, you can only edit the request from the reservation summary page. For more information, see [Share, modify, or cancel a reservation](../../wsd-reservation-management/task/modify-cancel-res-request.md).

**Note:**

-   Based on the configuration set by your admin, you might not be able to edit a request.
-   Editing a request is only supported for open requests. If a request is closed or complete, it can't be edited.
-   Editing a request is supported only if the request contains workplace service items.
-   If all the workplace services of a request are fulfilled, the request can't be edited, even if it's an open request.
-   Service items that are within the lead time can't be modified.

Role required: sn\_wsd\_core.workplace\_user

## Procedure

1.  Navigate to **All** &gt; **Self-Service** &gt; **Employee Center**.

    1.  On the Employee Center landing homepage, select the **Workplace Services** &gt; **Maintenance**.
    2.  Select **Submit a maintenance request**.
2.  On the Employee Center landing page, select **My Requests**.

3.  From the My Requests list, select the workplace request that you want to edit.

4.  On the request details page, select **Actions**, then select **Edit request**.

5.  Edit the details of the request based on your requirement.

6.  Edit workplace service items that are included in your request.

    You can add or remove workplace service items from the request. You can also change the quantities of service items that are already added to the request.

    **Note:** A workplace service item can't be edited if it has already been fulfilled.

7.  Select **Save**.

    The Workplace Request is updated.
