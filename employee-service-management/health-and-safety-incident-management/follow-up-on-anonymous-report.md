# Follow up on an anonymous report

You can retrieve your anonymous report to submit or view comments and updates from the agents working on your anonymous report. You must have the report key and report number to access your anonymous report.

## Before you begin

Role required: none

## About this task

After submitting the anonymous report, you’ll see a modal dialogue requesting you to download a report copy. This copy contains a unique key that gives you access to the anonymous report you created.

**Note:** The anonymous report can’t be accessed without the report key. If you no longer have this information, submit a new an anonymous report. Add details on the safety incident and indicate that you lost the report key and report number from your previous anonymous report.

## Procedure

1.  Navigate to **All** &gt; **Self Service** &gt; **Employee Center**

2.  In the Health and Safety tab,

    -   Select the **Anonymously report a safety concern** quick link.
    -   Select **Health and safety observations** and then select the **Anonymously report a safety concern** card.
    -   Select the **Browse health and safety** button and then select the **Anonymously report a safety concern** card.
    -   Select the **Anonymous report center**URL on the report copy.
3.  Select **Continue** on the modal dialogue.

    You are signed out of the Employee Center and signed into the **Anonymous Report Center** as a guest user.

4.  Select the **Follow up on a Report** card.

5.  Enter the report key and the report number from the report copy.

6.  Enter the text verification code.

7.  Select **Next**.

8.  Enter questions or comments related to the anonymous report.

9.  Select **Post**.
