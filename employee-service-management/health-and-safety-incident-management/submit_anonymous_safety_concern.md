# Submit an anonymous safety concern

Report a safety concern or observation without revealing your identity.

## Before you begin

Role required:

-   For **Employee Center**: sn\_ohs\_im.requestor
-   For **Anonymous Report Center**: None/Anonymous reporter

## About this task

The **Anonymous report center** \[com.sn\_anonymous\_report\_center\] plugin must be installed to enable reporting a safety concern anonymously. For more information, see [Activate the Anonymous Report Center](../../human-resources/task/arc-activate.md).

## Procedure

1.  Navigate to **All** &gt; **Self service** &gt; **Employee Center**.

2.  In the Health and Safety tab

    -   Select the **Anonymously report a safety concern** quick link.
    -   Select **Health and safety observations** and then select the **Anonymously report a safety concern** card.
    -   Select the **Browse health and safety** button and then select the **Anonymously report a safety concern** card.
3.  Select **Continue** on the modal dialogue.

    You are signed out of the Employee Center and signed into the **Anonymous Report Center** as a guest user.

4.  Select the **Submit an Anonymous Report** card and then select the **Health and safety concern** card.

5.  On the form, fill in the details.

    The report key and report number are required to:

    -   Track, ask questions, and view or add details about the anonymous case.
    -   Respond to any follow-up questions.
    **Note:** The anonymous report can’t be accessed without the report key.


## Result

The anonymous report is submitted and you can use the report key and report number to check the progress of the report.
