# Managing interviews as an interviewer

As an interviewer, view and manage interviews that you’re part of.

Interviewers are notified by email about any interviews that they’re part of, after the interview is proposed or scheduled.

After an interview is scheduled, an email invite with a link to the application is received. You can select the Applicant profile link on the email to view the application details.

The application page displays the stages that the application has gone through and the upcoming stages.

The **Overview** tab displays the following information:

-   Upcoming activities section that displays the upcoming proposed and scheduled interviews in a chronological order. You can request the recruiter for a new time slot for a scheduled interview by selecting the corresponding **Request new time** button. For a description of the field values, see [Request reschedule form](../reference/request-reschedule-applicant-interviewer-form.md).
-   Applicant details that include the basic information, professional experience, education information, and certifications, if any.

The **Interview** tab lists all the interviews that you're part of, with details such as interview name, type, state, start time, and duration. You can select an interview from the list to navigate to the interview record page and view the interview details, attendee list, and feedback provided by you.

The **Feedback** tab enables the interviewer to perform the following:

-   Provide interview feedback. For more information, see [Provide interview feedback](../task/provide-interview-feedback-hm-interviewer.md).
-   Edit the submitted feedback.
