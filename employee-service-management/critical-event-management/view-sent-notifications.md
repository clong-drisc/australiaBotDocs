# View sent notifications for a critical event

View all notifications sent for a critical event in the **Related records** Related Item in Critical Event Management.

## Before you begin

Role required: sn\_critical.agent

## Procedure

1.  Navigate to **Workspaces** &gt; **Health and Safety Workspace** and select the **Critical Event Management** tab.

2.  Open the critical event.

3.  Select the **Related records** tab.

4.  Select the **Sent notification** tab.
