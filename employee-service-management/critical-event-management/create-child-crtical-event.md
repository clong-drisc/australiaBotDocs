# Relate a child critical event

Mark duplicate or related critical events as child events and link them under the main, parent event.

## Before you begin

Role required: sn\_critical.agent

## Procedure

1.  Navigate to **Workspaces** &gt; **Health and Safety Workspace** and select the **Critical Event Management** tab.

2.  Open the critical event.

3.  Select the **Related records** Related Item.

4.  Select **Critical child event**.

5.  Select the **Add** button.

    A list of open critical events appears.

6.  Select one or more critical events that were created for the same emergency situation as the parent critical event.

7.  Mark the critical events as duplicate by selecting **Add**.

    Alternatively, on a child event record, you can link it to a parent record by entering a record number in the **Parent** field.

    The status of the child event is set to **Consolidated**. When you open this child event, the **Details** Related Item displays the parent critical event in the record view.
