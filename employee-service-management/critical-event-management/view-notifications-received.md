# View notifications received

View the list of notifications received for an impacted person in Critical Event Management.

## Before you begin

Role required: sn\_critical.agent

## Procedure

1.  Navigate to **Workspaces** &gt; **Health and Safety Workspace** and select the **Critical Event Management** tab.

2.  Open the critical event.

3.  Select the **Impacted people** tab.

4.  Select an individual from the **Total Impacted** list.

5.  Select the **Notifications** tab.

    This tab shows the communication sent to and received from the impacted person.
