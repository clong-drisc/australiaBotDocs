# Export an impacted people list

Export an impacted people list as a PDF, CSV, JSON, or Microsoft Excel file in Critical Event Management and either download or email the file.

## Before you begin

Role required: sn\_critical.agent

## Procedure

1.  Navigate to **Workspaces** &gt; **Health and Safety Workspace** and select the **Critical Event Management** tab.

2.  Open the critical event.

3.  Select the **Impacted people** tab.

4.  Select the **Export** button.

5.  In the File Type drop-down list, choose your desired file format for the exported list.

    The file format options are the following:

    -   Excel
    -   CSV
    -   JSON
    -   PDF
6.  In the Delivery Type drop-down list, choose where to send the exported file.

    -   To download the file to your device, select **Download**.
    -   To send the file via email, select **Email**.
7.  If you chose to send the file to an email address, then in the **Email** field, enter the email address.

    The **Email** field only appears after selecting **Email** from the drop-down list.

8.  Select **Export**.
