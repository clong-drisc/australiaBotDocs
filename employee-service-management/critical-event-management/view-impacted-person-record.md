# View an impacted person record

View and track the status of an impacted person's record as an agent in Critical Event Management.

## Before you begin

Role required: sn\_critical.agent

## Procedure

1.  Navigate to **Workspaces** &gt; **Health and Safety Workspace** and select the **Critical Event Management** tab.

2.  Open the critical event.

3.  Select the **Impacted people** tab.

4.  Select the individual from the **Total Impacted** list to view that person's record.

    The record shows the following information:

    -   Phone number
    -   Email
    -   Critical event number
    -   Name
    -   State of the emergency
    -   Job title
    -   Last date and time the person responded
    -   Location of the critical event
    -   Status
    -   Manager
    -   Present on Site
    -   Notes
    The record also shows the activity stream and notifications.
