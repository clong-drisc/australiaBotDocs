# Map on-site responder group with location

Map the on-site responder groups with the location, enabling these groups to be automatically populated when the location is selected in a new critical event.

## Before you begin

Role required: admin or sn\_critical.manager

## Procedure

1.  Navigate to **All** &gt; **Health and Safety** &gt; **Critical Event Management** &gt; **Responder Group mapping**, to view the list of the on-site responder groups.

2.  Select **New** to add a new on-site responder group.

3.  Select the group and location and select **Submit** to configure the group and map it with the entered location.

4.  Alternatively, select the Critical Event Management list view.

5.  Navigate to **On-site Responder Mapping** &gt; **All** to view the list of on-site groups mapped with their location.

6.  Select **New** to add on-site responder groups and their location.

7.  Select **Save**.
