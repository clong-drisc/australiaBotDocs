# Add roles to the group

Add roles to groups that are mapped to a location.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **System Security** &gt; **User and groups** &gt; **Groups**.

2.  Open the relevant group.

3.  Select **Edit** in the **Roles** tab.

4.  Select the appropriate role from the list of roles.

5.  Select ![Chevron right](../../../reuse/icons/product-icons/chevron-right-fill-24.svg) to add the role to the group.
