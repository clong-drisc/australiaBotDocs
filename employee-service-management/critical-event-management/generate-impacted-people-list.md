# Generate impacted people list

Generate an impacted people list for a critical event in Critical Event Management. The impacted people list is used to account for every individual in the critical event location.

## Before you begin

Role required: sn\_critical.agent

## About this task

Employees are mapped to their workspace location. The employee list is automatically generated according to the location added in the critical event ticket.

## Procedure

1.  Navigate to **Workspaces** &gt; **Health and Safety Workspace** and select **New Critical Event**.

    You can also select an existing critical event in the **Critical Event Management** tab.

2.  In the Critical Event form, fill out the fields.

    For a description of the field values, see [Critical Event form](../reference/new-critical-event-form.md).

3.  Select **Save**.

4.  Select the **Impacted people** tab.

5.  Select the **Load Impacted People** button.

    The list of impacted people is extracted based on the location. To add people manually, see [Add a person manually to the impacted people list](add-person-manually-to-impacted-people.md).
