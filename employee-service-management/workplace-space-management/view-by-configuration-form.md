# View by Configuration form

The view by configuration form helps you select the table and field used to filter the view on the scenario planner or building overview.

<table id="table_rcv_hhy_vcc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Title

</td><td>

Title of the view by configuration that is displayed on the scenario planner or building overview.

</td></tr><tr><td>

View by table

</td><td>

Table that contains the field you want to use for the view by configuration.The following tables are supported for view-by configurations:

-   Scenario Plan Space Detail \[sn\_wsd\_spcmgmt\_scenario\_plan\_space\_detail\]
-   Space \[sn\_wsd\_core\_space\]
-   Space Detail \[sn\_wsd\_spcmgmt\_space\_detail\]

</td></tr><tr><td>

View by field

</td><td>

Field that is used to filter the view of the scenario planner or building overview.You can dot-walk to a maximum of two fields. You can only select fields that are of the reference, choice, or true/false type.

</td></tr><tr><td>

Active

</td><td>

Option to activate the record.

</td></tr><tr><td>

Application

</td><td>

Application that the record is created in.This field is automatically set to Workplace Space Management and can't be edited.

</td></tr><tr><td>

Mode

</td><td>

Mode where the view by configuration is enabled.**Note:**

-   Building overview is not supported for the Scenario Plan Space Detail \[sn\_wsd\_spcmgmt\_scenario\_plan\_space\_detail\] table.

-   Scenario planning is not supported for the Space Detail \[sn\_wsd\_spcmgmt\_space\_detail\] table.


</td></tr><tr><td>

Order

</td><td>

Order of the view by configuration in the drop-down list of the scenario planner or building overview.

</td></tr><tr><td>

Domain

</td><td>

Domain for the view by configuration.This field is automatically set to global.

</td></tr></tbody>
