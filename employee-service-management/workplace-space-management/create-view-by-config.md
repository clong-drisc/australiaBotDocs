# Create a view-by configuration

Create a configuration to filter the view of the scenario planner or building overview.

## Before you begin

Role required: sn\_wsd\_spcmgmt.admin

## Procedure

1.  Navigate to **All** &gt; **Workplace Core** &gt; **Administration** &gt; **View-by Settings Configuration**

2.  On the View by Configurations list, select **New**.

3.  On the View by Configuration form, fill in the fields.

    For a description of the field values, see [View by Configuration form](../reference/view-by-configuration-form.md).

4.  Select **Submit**.
