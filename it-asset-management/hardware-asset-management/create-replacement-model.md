# Configure replacement models for a refresh model

Configure replacement models for a hardware asset model that is refreshed. When configured, employees can select replacement models for their hardware assets on the Zero Touch Refresh form.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **All** &gt; **Hardware Asset Workspace** &gt; **Asset operations**.

2.  In the **Zero touch refresh** list, select **Refresh models**.

3.  Create the refresh model.

    1.  On the Refresh models page, select **New**.

    2.  On the form, fill in the fields.

<table id="table_qqn_mhc_pxb"><thead><tr><th>

Field

</th><th>

</th></tr></thead><tbody><tr><td>

Model

</td><td>

Model that is being refreshed.

Location for which the replacement models are valid.**Note:** Configuring replacement models for a country is the most efficient setup. However, you can configure replacement models for more specific locations such as state or city.

</td></tr><tr><td>

Active

</td><td>

Option that indicates whether the replacement rule is active.

</td></tr></tbody>
</table>    3.  Select **Save**.

4.  Add replacement models for the refresh model.

    1.  In the **Replacement models** tab, select **Add**.

    2.  In the **Replacement Model** dialog box, select the models that you want to add as replacement for the refresh asset model and select **Add**.

    3.  Select **Save**.


## Result

The refresh model with the assigned replacement models is included in the **Refresh models** list.
