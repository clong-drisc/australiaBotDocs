# View the open hardware asset pick tasks for your stockroom

View the open hardware asset pick tasks for your stockroom to get the details of the tasks that need action.

## Before you begin

Role required: inventory\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **Hardware Asset Workspace** &gt; **Inventory**.

2.  Select the **All stockrooms** tab.

3.  Select the stockroom.

    In the Open stockroom tasks section, the **Open pick tasks** card displays the number of open pick tasks.

4.  Select the **Open pick tasks** card.


## Result

The list of open asset pick tasks for the stockroom is displayed.

## What to do next

Manage all the hardware asset pick tasks assigned to you by using the Mobile Agent application. For details, see [Manage hardware asset pick tasks using the Mobile Agent application](../concept/manage-asset-pick-task-ham-mobile-app.md).
