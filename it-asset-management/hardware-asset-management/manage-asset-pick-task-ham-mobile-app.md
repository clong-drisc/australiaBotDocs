# Manage hardware asset pick tasks using the Mobile Agent application

Work on the hardware asset pick tasks assigned to you by using the Mobile Agent application.

-   **[Start working on a hardware asset pick task using the Mobile Agent application](../task/start-work-pick-task-mobile-ham.md)**  
Indicate in the Mobile Agent application when you start work on an asset pick task assigned to you.
-   **[Record time worked on a hardware asset pick task using the Mobile Agent application](record-time-pick-task-mobile-agent-ham.md)**  
Manage and record time worked on hardware asset pick tasks using the Mobile Agent application. After you start working on a task, you can pause and resume work.
-   **[Locate and pick the hardware assets in your stockroom using the Mobile Agent application](../task/locate-and-pick-hardware-asset-using-mobile-app.md)**  
Locate a hardware asset placed in a specific aisle and space in the stockroom by using indoor maps. After locating the asset, pick and drop off the asset to another location within the stockroom using the Mobile Agent application.
