# View the open enterprise asset pick tasks for your stockroom

View the open asset pick tasks for your stockroom to get the details of the tasks that need action.

## Before you begin

Role required: inventory\_user

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Asset Workspace** &gt; **Inventory**.

2.  Select the **All stockrooms** tab.

3.  Select the stockroom.

    In the Open stockroom tasks section, the **Open pick tasks** card displays the number of open pick tasks.

4.  Select the **Open pick tasks** card.


## Result

The list of open asset pick tasks for the stockroom is displayed.

## What to do next

Manage all the enterprise asset pick tasks assigned to you by using the Mobile Agent application. For details, see [Manage enterprise asset pick tasks using the Mobile Agent application](../concept/manage-asset-pick-task-eam-mobile-app.md).
