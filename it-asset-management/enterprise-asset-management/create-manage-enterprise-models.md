# Create and manage enterprise models

Create and manage your enterprise models to keep track of your enterprise assets.

-   **[Create a multi-component model](../task/create-multicomponent-model.md)**  
Create a multi-component model to define more than one model component based on the same product model.
-   **[Create enterprise models](../task/create-model-eam.md)**  
Create a model based on a top tier Enterprise Asset Management model category. Use models to manage and keep track of your assets.
-   **[Create enterprise model components](../task/create-model-components-eam.md)**  
Create model components and add them to a model. A model comprising of one or more model components is a multi-component model.
-   **[Create model categories](../task/create-custom-model-category.md)**  
You can create model categories for the Enterprise Asset Management application.
-   **[Add a substitute model for an enterprise model](../task/add-enterprise-model-substitute.md)**  
On an enterprise model record, specify the related models that you can substitute the enterprise model with.
-   **[Create classification codes](../task/create-classification-codes.md)**  
Create classification codes and assign them to enterprise models and model categories to increase the efficiency of enterprise models.
-   **[Create sources for classification codes](../task/create-class-source-eam.md)**  
Create sources that you can then assign to classification codes.
-   **[Import classification codes](../task/import-class-codes-eam.md)**  
Import multiple classification codes at one go to your ServiceNow instance.
-   **[Add calibration attributes to an enterprise model](../task/add-calibration-attributes-enterprise-model.md)**  
Add calibration attributes to an enterprise model to specify and manage the calibrations that are required for the associated enterprise assets throughout their life cycles.
-   **[Clone calibration attributes to an enterprise model](../task/clone-calibration-attributes-enterprise-model.md)**  
Clone an existing calibration attribute to an enterprise model to add a calibration attribute with similar settings.
-   **[Enterprise Asset Management normalization](normalization-eam.md)**  
The Enterprise Asset Management normalization process enables you to normalize your manufacturer, model name, model number, and model type data of your enterprise models.
-   **[View the enterprise model hierarchy in the Enterprise Asset Workspace](../task/view-model-hierarchy-eam.md)**  
View the hierarchy details of pre-assembled and user-assembled enterprise models in a tree-like format in the Enterprise Asset Workspace.
-   **[Manage the lifecycle of enterprise models with calculated lifecycle templates](manage-eam-lifecycle-temp.md)**  
Manage the complete lifecycle of enterprise models by creating calculated lifecycle templates and associating these templates with the models.
