# Create knowledge articles for firmware models

Attach helpful and relevant knowledge articles to the firmware model in the OT Asset Workspace.

## Before you begin

**Important:** The OT Asset Management application must be activated to access the OT Asset Workspace. For details, see [Install OT Asset Management](install-otam.md).

Role required: sn\_otam.ot\_asset\_manager

## Procedure

1.  Navigate to **Workspaces** &gt; **OT Asset Workspace** &gt; **OT Model Management**.

2.  Select the **Firmware** tab.

3.  Select the firmware model for which you want to attach the knowledge article.

4.  Select the **Knowledge** tab.

5.  Select **New**.

6.  On the Create New Knowledge form, select the knowledge article that you want to add in the **Knowledge** field.

7.  Select **Save**.
