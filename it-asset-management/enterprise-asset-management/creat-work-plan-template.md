# Create a work order plan template

Create a reusable work order plan template to use when creating work order plans.

## Before you begin

Role required: sn\_eam.asset\_manager

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Asset Workspace** &gt; **Work management**.

2.  Select the **Work order plan templates** tab.

3.  Select **New**.

4.  On the form, fill in the fields.

<table id="table_sjb_xy5_j3c"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Name

</td><td>

Name of the work order plan template.

</td></tr><tr><td>

Description

</td><td>

Description of the work order plan.

</td></tr><tr><td>

Primary target

</td><td>

Asset or asset group for which the work order plan template is created. When you select an asset group, all assets in the asset group are automatically added to the Affected assets list.

**Note:** This field is required.

</td></tr><tr><td>

Location

</td><td>

Location of the primary target.This field is automatically populated when you select a primary target.

**Note:** If you select a location first, the **Primary target** and **Additional assets** fields are filtered to show only assets in that location.

</td></tr><tr><td>

Additional assets

</td><td>

Assets to include in the work order plan template along with the primary target. Additional assets must be in the same location as the primary target.

</td></tr><tr><td>

Work order template

</td><td>

Work order template that defines the collection of tasks to be performed on the primary target and any additional assets.

</td></tr></tbody>
</table>5.  Select **Save**.


## Result

The work order plan template is created and appears in the **Work order plan templates** tab.

## What to do next

Use this work order plan template when you [create a work order plan](create-work-order-plan-eam.md).
