# Create work notes about the work order tasks using the Mobile Agent application

Add notes with details about work order task to the task record using the Mobile Agent application.

## Before you begin

Role required: wm\_agent

## Procedure

1.  From your mobile device, launch the Mobile Agent application.

2.  On the navigation bar at the bottom of the screen, tap the **My Tasks** tab.

    The home screen of the My Tasks application opens with the first few tasks in the list of tasks assigned to you displayed.

3.  If the task to which you want to add a work note isn't displayed, tap **See all**.

4.  Filter or sort the task view.

    -   To filter your tasks, tap the Filter icon ![](../image/filter-mobile-task-eam.png) and enter the values to use as a filter in the **Due Date**, **Asset**, **Location**, or **Priority** fields.
    -   To sort your tasks, tap the Filter icon ![](../image/filter-mobile-task-eam.png), tap **Sort by**, and select the fields to sort your tasks.
5.  Tap the work order task for which you want to add a work note.

6.  Tap the **Activity** tab.

    All the work notes related to the work order tasks are listed.

7.  Tap **Compose**.

8.  On the Compose screen, type your comments related to work order task.

9.  Tap **Post**.


## Result

The posted work note is shown in the **Activity** tab of the work order task record.
