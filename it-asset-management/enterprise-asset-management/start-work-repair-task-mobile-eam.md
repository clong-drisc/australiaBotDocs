# Start working on a enterprise asset repair task using the Mobile Agent application

Indicate in the Mobile Agent application when you start work on a repair task assigned to you.

## Before you begin

Role required: sn\_eam.enterprise\_asset\_technician

## About this task

You can work on the repair tasks that belong to the following task types:

-   Asset troubleshoot task
-   Asset repair task
-   Asset evaluate task

## Procedure

1.  From your mobile device, launch the Mobile Agent application.

2.  On the navigation bar at the bottom of the screen, tap the **My Tasks** tab.

    The home screen of the My Tasks application opens displaying only the first few tasks in the list of tasks assigned to you.

3.  If the task you want to start isn't displayed, tap **See all**.

4.  Filter or sort the task view.

    -   To filter your tasks, tap the Filter icon ![](../image/filter-mobile-task-eam.png) and enter the values to use as a filter in the **Due Date**, **Asset**, **Location**, or **Priority** fields.
    -   To sort your tasks, tap the Filter icon ![](../image/filter-mobile-task-eam.png), tap **Sort by**, and select the fields by which to sort your tasks.
5.  Tap the repair task that you want to start.

6.  On the **Details** tab, tap **Start work**.


## Result

On the **Details** tab, the **Start work** link changes to **Start timer**, **Record time**, and **Close** automatically.
