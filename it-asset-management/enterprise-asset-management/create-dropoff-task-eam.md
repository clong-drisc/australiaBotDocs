# Create a Drop off task for enterprise assets

Create a Drop off task to return the enterprise assets in your personal stockroom to any other warehouse stockroom.

## Before you begin

Role required: sn\_eam.enterprise\_asset\_technician

## Procedure

1.  Navigate to **Workspaces** &gt; **Enterprise Asset Workspace** &gt; **Work management**.

2.  Select the **My stockroom assets** tab.

    The list of enterprise assets in your personal stockroom is displayed.

3.  Select **Schedule drop off**.

4.  On the form, fill in the fields.

<table id="table_snq_rb4_4bc"><thead><tr><th>

Field

</th><th>

Description

</th></tr></thead><tbody><tr><td>

Stockroom

</td><td>

Stockroom at which you want to drop off the assets.This field is required.

</td></tr><tr><td>

Delivery by date

</td><td>

Date by when you want to deliver the assets to the destination stockroom.This field is required.

</td></tr><tr><td>

Assignment group

</td><td>

Assignment group to which you want to assign the drop off task.

</td></tr><tr><td>

Assigned to

</td><td>

Person from the assignment group who is primarily responsible for the drop off task.

</td></tr></tbody>
</table>5.  Select **Save**.


## Result

A Drop off task in the Open state is created and listed in the **Asset tasks** tab.
