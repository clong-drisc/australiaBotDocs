# Configure display properties for a Data snapshots indicator

Change display properties for viewing a specific native Data snapshots indicator in KPI Details.

## Before you begin

Role required: None. However, upgraded instances may require pa\_viewer.

## Procedure

1.  Open KPI Details for a native Data snapshots indicator, as described in [View KPI Details](../../performance-analytics/task/view-kpi-details.md).

2.  Open the **Properties** panel.

    ![Properties panel and open/close button in KPI Details.](../image/kpid-native-ds-display-props.png)

3.  Set the following options.

    |Option|Description|
    |------|-----------|
    |Show continuous lines|Fills in the lines where there are missing values|
    |Decimal precision|Number of digits after the decimal point to round numbers off to|
    |Enable abbreviation|Option to use an alphabetical abbreviation. For example, M to represent million.|
