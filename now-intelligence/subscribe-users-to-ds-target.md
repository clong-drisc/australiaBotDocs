# Subscribe users to Data snapshots indicator target

If you have targets set on a native Data snapshots indicator, you can specify users to receive emails about that target. Your instance must be configured to send emails.

## Before you begin

Role required: pa\_target\_admin

## About this task

Because responsible users cannot be configured for targets on Data snapshots indicators, you have the option to subscribe users to emails instead.

## Procedure

1.  Open KPI Details for a native Data snapshots indicator, as described in [View KPI Details](../../performance-analytics/task/view-kpi-details.md).

2.  Select the target icon \(![Target icon](../image/target-icon.png)\) in the sidebar.

    The Targets panel opens.

3.  Select the Envelope button in the Targets panel ![Envelope button](../image/kpid-envelope.png).

4.  Select users from the Users \[sys\_user\] table to receive email notifications when a target is met or missed.
