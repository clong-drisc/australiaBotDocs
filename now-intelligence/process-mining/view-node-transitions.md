# Viewing activity transitions

View activity transitions you've defined from the Bottleneck Analysis feature.

Expand the Bottleneck Analysis box to see the top five overall and specified activity state transitions across these specific metrics.

-   Unique Occurrences
-   Maximum Repeat Occurrences
-   Average Duration
-   Total Duration
-   Total Occurrences

View selected transitions and metrics from the lists.

![View activity transitions](../image/transitions-box-process-map2.png)

Select a transition link from the list to edit activities for the selected filter.

![Transition link to edit a transition filter](../image/transition-link2.png)
