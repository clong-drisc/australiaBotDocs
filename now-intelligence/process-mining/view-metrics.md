# Viewing metrics and activity transitions

View metrics and activity transitions you have defined.

## View metrics

View your selected metrics, or the full list of metrics for an activity or connection. The metrics box \(![metrics-boxes](../image/metrics-boxes.png)\) on a route shows the primary and secondary metrics selected to show on the process map.

Select an activity or connection to see a full list of metrics.

![Metrics list box](../image/metrics-list-box2.png)

-   **[Viewing activity transitions](view-node-transitions.md)**  
View activity transitions you've defined from the Bottleneck Analysis feature.
-   **[Viewing records for an activity or connection](view-records-node-arc.md)**  
See the list of records which passed through an activity or connection to analyze data in more detail.
