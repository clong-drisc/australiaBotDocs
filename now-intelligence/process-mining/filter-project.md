# Filtering project data

Apply filters to refine and drill into specific aspects of your process map.

Process Mining has these filtering tools for refining a process map, accessible from the **Filters** tab and the process map.

## Filters tab

-   Breakdown filters - Show records grouped by category.
-   Conditions - Use the condition builder to filter by table field conditions.
-   Transitions filter - Filter by activity conditions to see routes that transitioned through an activity.

## Process map

-   Metrics filter - Show your selected metrics on activities and connections.
-   Activities and Connections filters - Use sliders to view more or less activities or connections.
