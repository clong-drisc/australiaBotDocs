# Add the Now Assist Context Menu

Use the NACM to enable viewers of a dashboard to generate a summary of the contents of a dashboard or dashboard tab.

## Before you begin

Role required: Any user with an internal role can create add the context menu to an inline dashboard.

## Procedure

1.  Navigate to **All** &gt; **Platform Analytics** &gt; **Library** &gt; **Dashboards**.

2.  Either [Create a dashboard with the in-line editor](create-db-in-ac.md) or open the dashboard you want to edit.

3.  Expand **Add New Element**.

4.  Select the **Now Assist Context Menu** element.

5.  And so forth
