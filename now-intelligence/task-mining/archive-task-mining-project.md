# Archive or reopen a Task Mining project

Prepare to delete an unused project according to your data retention policy.

## Before you begin

Role required: admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Task Mining Workspace**.

2.  Navigate to the project you want to archive.

3.  Select the More actions icon ![](../../integrationhub/images/more-actions-menu-icon.png).

    ![Screenshot showing the More actions menu and the Archive option highlighted.](../image/tm-archive.png)

4.  Select **Archive** and confirm.

    ![Screenshot showing the Archive confirmation panel.](../image/tm-archive-confirm.png)

    Your project is scheduled to close after the end of your Project retention period. For more information, see [Modify the Task Mining data retention period](define-data-retention.md).

5.  Navigate to the project you archived that you want to reopen.

6.  Select the More actions icon ![](../../integrationhub/images/more-actions-menu-icon.png).

    ![Screenshot showing the More actions menu and the Reopen option highlighted.](../image/tm-reopen.png)

7.  Select **Reopen** and confirm.

    ![Screenshot showing the Reopen confirmation panel.](../image/tm-reopen-confirm.png)
