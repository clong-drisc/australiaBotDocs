# Data visualization library recommendations

Analytics managers can view information about potentially problematic data visualizations.

## Data visualization recommendation panel

![Data visualization library showing recommendation panel.](../image/dv-recommendation-panel.png)

Visible to roles: viz\_admin and higher, including pa\_admin.

The book icon ![](../image/book-icon.png) expands and hides the data visualization recommendation panel. It also shows how many recommendation tiles there are. Selecting a tile filters the library on the matching data visualizations.

The following types of recommendations are currently available:

|Tile|Description|
|----|-----------|
|n Unused data visualizations|Number of data visualizations that are not used in any dashboards|
|n No active owner|Number of data visualizations without an active owner|
