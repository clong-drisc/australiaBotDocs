# Certify a data visualization in the Visualization Designer

Certify a data visualization to indicate that it is company-approved for reuse.

## Before you begin

Role required: admin, viz\_admin

## Procedure

1.  Navigate to **Workspaces** &gt; **Platform Analytics Workspace**, or the workspace that has the Analytics Center \(![Analytics center icon](../../par-for-workspace/image/analytics-center-icon.png)\) page and the dashboard you want to certify.

2.  On the **Data visualizations** tab, select the dashboard you want to certify.

3.  Select the information button \(![Info icon](../../par-for-workspace/image/icon-info.png)\) to open the Details panel.

    ![Info button to open visualization details panel.](../../par-for-workspace/image/paw-vd-info-button.png)

4.  Select the toggle labeled **Mark as certified**.

5.  Select **Save**.


## Result

The Certified badge appears next to the visualization's name in the Visualization Designer.

On the Data Visualizations tab, the visualization is listed under Certified data visualizations. The visualization is also marked True in the Certified column. This column may not be visible. Select the Edit columns icon \(![Edit columns icon](../image/icon-cogwheel-ac.png)\) to add it.
