# Configure a data visualization to follow filters or not

Data visualizations follow filters by default. A data visualization follows filters in the same dashboard tab as itself or above the tabs. Data visualizations either follow all such tabs that target their data sources, or none.

## Before you begin

Role required: You can edit any data visualization you own unless it is a shared visualization, in the library. If the data visualization is in the library and you own it, you need the itil, report\_user, or viz\_creator role to edit it. You can edit any data visualization with the viz\_admin role or higher.

## Procedure

1.  Locate the data visualization either in its dashboard or in the Library.

2.  Open the visualization for editing.

    -   If in the data visualization library, select the visualization to open it.
    -   If in a dashboard, select **Edit** on the dashboard, focus on the desired data visualization, and select **Configure** in the visualization header menu.
3.  In the Data update section, select or deselect **Follow filters**.

4.  To show or not show a filter icon \(![Filter icon](../../dashboards/image/InteractiveFilterFilteringIcon.png)\) on the data visualization when it follows an interactive filter, toggle **Show filter icon** \(on by default\).
