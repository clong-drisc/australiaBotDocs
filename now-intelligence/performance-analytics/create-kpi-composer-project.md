# Create a KPI Composer project

As the first step in using KPI Composer, create a project.

## Before you begin

Role required: sn\_kpi\_composer.user, sn\_kpi\_composer.admin, admin

## Procedure

1.  Navigate to **All** &gt; **KPI Composer** &gt; **Projects** and click **New**.

2.  In the Create new project dialog, give the project a meaningful name and click **Save**.


## Result

Your project is created and is visible in the **Projects** page.

## What to do next

As you work on the project, keep track collaboratively in the Project Journal.
