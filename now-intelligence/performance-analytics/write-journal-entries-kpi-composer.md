# Write journal entries for a project

Keep track of your KPI Composer project with journal entries

Prerequisites
:   [Create a KPI Composer project](create-kpi-composer-project.md)

## Before you begin

Role required: Anyone with access to a KPI Composer project, including view access, can write a journal entry.

## Procedure

1.  Click the journal icon \(![](../image/kpi-composer-journal-icon.png)\).

2.  Write a journal entry.

3.  Click **Post**.


## Result

All posted journal entries appear in descending chronological order.
