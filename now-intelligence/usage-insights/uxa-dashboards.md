# Viewing dashboards

Create dashboards for Platform Analytics directly in Usage Insights. View dashboards that you create in Usage Insights directly in Platform Analytics.

## Before you begin

Role required:

For viewing dashboards: any analytics\_viewer role- chat\_analytics\_viewer, mobile\_analytics\_viewer, portal\_analytics\_viewer, core\_ui\_analytics\_viewer, now\_experience\_analytics\_viewer.

For creating dashboards: analytics\_admin or analytics\_viewer.

## Procedure

1.  Navigate to **All** &gt; **Platform Analytics** &gt; **Usage Insights**.

2.  Select the **Dashboards** icon in the **main sidebar menu**.

    A card view of available custom Platform Analytics dashboards opens.

3.  Create a dashboard according to your specifications.

    You can view your created dashboards in **Platform Analytics**.

    Admin roles can grant viewing permissions to viewer roles, but not creation permissions.

    Any type of admin role can grant viewing permissions to any type of viewer role.

    Currently, if you only have a channel-specific child role \(for example, portal\_analytics\_viewer\) but not analytics\_admin or analytics\_viewer, the Usage Insights data source doesn’t appear.
