# Export destinations

When you export a dashboard or data visualization in the Now Assist panel, you have to specify the destination.

You can select from the following file destinations:

-   Download
-   Email
-   Scheduled email

Specify recipients for an email. If you select scheduled email, also specify the frequency of the email. You are asked for more details depending on the frequency you have selected.

**Note:** If you select **Download** and the export takes longer than 10 seconds, you receive the export as an email attachment instead of a download. Thus the export completes asynchronously, freeing up the Now Assist panel.
